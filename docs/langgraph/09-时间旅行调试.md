# 第九章 时间旅行调试

> 来源: [LangGraph 官方文档](https://langchain-ai.github.io/langgraph/)

---

## 什么是时间旅行

时间旅行（Time Travel）是 LangGraph 提供的强大调试功能，允许你查看图执行的历史状态、回放到任意时间点、以及从特定状态重新执行。这对于调试复杂的 Agent 工作流非常有用。

### 时间旅行的核心能力

| 功能 | 说明 |
|------|------|
| **状态历史** | 查看图执行过程中的所有状态快照 |
| **状态回放** | 回到任意历史状态查看当时的数据 |
| **分支执行** | 从某个历史状态开始，用不同的输入重新执行 |
| **对比分析** | 比较不同执行路径的结果 |

## 获取状态历史

### 基本用法

```typescript
import { StateGraph, Annotation, MemorySaver } from "@langchain/langgraph";

// 定义状态
const State = Annotation.Root({
  messages: Annotation<string[]>({
    reducer: (prev, curr) => [...prev, ...curr],
    default: () => [],
  }),
  step: Annotation<number>({
    default: () => 0,
    reducer: (_, curr) => curr,
  }),
});

// 创建图
const graph = new StateGraph(State)
  .addNode("step1", async (state) => {
    return {
      messages: ["步骤 1 完成"],
      step: 1,
    };
  })
  .addNode("step2", async (state) => {
    return {
      messages: ["步骤 2 完成"],
      step: 2,
    };
  })
  .addNode("step3", async (state) => {
    return {
      messages: ["步骤 3 完成"],
      step: 3,
    };
  })
  .addEdge("__start__", "step1")
  .addEdge("step1", "step2")
  .addEdge("step2", "step3")
  .addEdge("step3", "__end__")
  .compile({ checkpointer: new MemorySaver() });

// 执行图
const config = { configurable: { thread_id: "thread-1" } };
await graph.invoke({ messages: ["开始"] }, config);

// 获取状态历史
const history = await graph.getStateHistory(config);

for (const state of history) {
  console.log(`步骤 ${state.values.step}:`, state.values.messages);
}
```

### 详细历史信息

```typescript
import { StateGraph, Annotation, MemorySaver } from "@langchain/langgraph";

const State = Annotation.Root({
  data: Annotation<string>,
  timestamp: Annotation<number>({
    default: () => Date.now(),
  }),
});

const graph = new StateGraph(State)
  .addNode("process", async (state) => {
    return {
      data: `处理: ${state.data}`,
      timestamp: Date.now(),
    };
  })
  .addEdge("__start__", "process")
  .addEdge("process", "__end__")
  .compile({ checkpointer: new MemorySaver() });

const config = { configurable: { thread_id: "thread-2" } };
await graph.invoke({ data: "输入数据" }, config);

// 获取详细历史
const history = await graph.getStateHistory(config);

for (const snapshot of history) {
  console.log("状态快照:", {
    values: snapshot.values,           // 当前状态值
    next: snapshot.next,               // 下一个要执行的节点
    config: snapshot.config,           // 该状态的配置
    metadata: snapshot.metadata,       // 元数据
    createdAt: snapshot.createdAt,     // 创建时间
    parentConfig: snapshot.parentConfig, // 父配置
  });
}
```

## 回放状态

### 回放到特定状态

```typescript
import { StateGraph, Annotation, MemorySaver } from "@langchain/langgraph";

const State = Annotation.Root({
  counter: Annotation<number>({
    default: () => 0,
    reducer: (_, curr) => curr,
  }),
  history: Annotation<string[]>({
    reducer: (prev, curr) => [...prev, ...curr],
    default: () => [],
  }),
});

const graph = new StateGraph(State)
  .addNode("increment", async (state) => {
    return {
      counter: state.counter + 1,
      history: [`计数器变为 ${state.counter + 1}`],
    };
  })
  .addEdge("__start__", "increment")
  .addEdge("increment", "__end__")
  .compile({ checkpointer: new MemorySaver() });

const config = { configurable: { thread_id: "thread-3" } };

// 第一次执行
await graph.invoke({ counter: 0 }, config);

// 第二次执行
await graph.invoke({ counter: 10 }, config);

// 第三次执行
await graph.invoke({ counter: 20 }, config);

// 获取历史并回放
const history = await graph.getStateHistory(config);
const historyArray = [];

for await (const state of history) {
  historyArray.push(state);
}

// 回放到第二个状态
const targetState = historyArray[1];
console.log("回放到状态:", targetState.values);

// 从该状态重新执行
const replayResult = await graph.invoke(null, targetState.config);
console.log("回放结果:", replayResult);
```

## 分支执行

### 从历史状态分支

```typescript
import { StateGraph, Annotation, MemorySaver } from "@langchain/langgraph";

const State = Annotation.Root({
  path: Annotation<string[]>({
    reducer: (prev, curr) => [...prev, ...curr],
    default: () => [],
  }),
  decision: Annotation<string>({
    default: () => "",
    reducer: (_, curr) => curr,
  }),
});

const graph = new StateGraph(State)
  .addNode("start", async (state) => {
    return { path: ["开始"] };
  })
  .addNode("decisionPoint", async (state) => {
    return { path: ["决策点"] };
  })
  .addNode("pathA", async (state) => {
    return { path: ["路径 A"], decision: "A" };
  })
  .addNode("pathB", async (state) => {
    return { path: ["路径 B"], decision: "B" };
  })
  .addNode("end", async (state) => {
    return { path: [`结束于 ${state.decision}`] };
  })
  .addEdge("__start__", "start")
  .addEdge("start", "decisionPoint")
  .addConditionalEdge("decisionPoint", async (state) => {
    return state.decision === "A" ? "pathA" : "pathB";
  })
  .addEdge("pathA", "end")
  .addEdge("pathB", "end")
  .addEdge("end", "__end__")
  .compile({ checkpointer: new MemorySaver() });

const config = { configurable: { thread_id: "thread-4" } };

// 执行路径 A
await graph.invoke({ decision: "A" }, config);
console.log("路径 A 完成");

// 获取到决策点的状态
const history = await graph.getStateHistory(config);
let decisionPointState;

for await (const state of history) {
  if (state.values.path.includes("决策点")) {
    decisionPointState = state;
    break;
  }
}

// 从决策点分支，执行路径 B
if (decisionPointState) {
  const branchConfig = {
    configurable: {
      thread_id: "thread-4-branch",
      checkpoint_id: decisionPointState.config.configurable.checkpoint_id,
    },
  };

  // 修改决策并重新执行
  const branchResult = await graph.invoke(
    { decision: "B" },
    branchConfig
  );
  console.log("分支结果:", branchResult);
}
```

## 调试技巧

### 1. 使用断点调试

```typescript
import { StateGraph, Annotation, MemorySaver } from "@langchain/langgraph";

const State = Annotation.Root({
  data: Annotation<string>,
  debug: Annotation<any[]>({
    reducer: (prev, curr) => [...prev, ...curr],
    default: () => [],
  }),
});

const graph = new StateGraph(State)
  .addNode("step1", async (state) => {
    // 添加调试信息
    const debugInfo = {
      node: "step1",
      timestamp: Date.now(),
      input: state.data,
    };
    return {
      data: `步骤1: ${state.data}`,
      debug: [debugInfo],
    };
  })
  .addNode("step2", async (state) => {
    const debugInfo = {
      node: "step2",
      timestamp: Date.now(),
      input: state.data,
    };
    return {
      data: `步骤2: ${state.data}`,
      debug: [debugInfo],
    };
  })
  .addEdge("__start__", "step1")
  .addEdge("step1", "step2")
  .addEdge("step2", "__end__")
  .compile({ checkpointer: new MemorySaver() });

const config = { configurable: { thread_id: "debug-thread" } };
const result = await graph.invoke({ data: "测试" }, config);

// 查看调试信息
console.log("调试日志:", result.debug);
```

### 2. 状态对比

```typescript
import { StateGraph, Annotation, MemorySaver } from "@langchain/langgraph";

const State = Annotation.Root({
  value: Annotation<number>({
    default: () => 0,
    reducer: (_, curr) => curr,
  }),
});

const graph = new StateGraph(State)
  .addNode("process", async (state) => {
    return { value: state.value * 2 };
  })
  .addEdge("__start__", "process")
  .addEdge("process", "__end__")
  .compile({ checkpointer: new MemorySaver() });

// 执行两次，不同输入
const config1 = { configurable: { thread_id: "compare-1" } };
const config2 = { configurable: { thread_id: "compare-2" } };

await graph.invoke({ value: 5 }, config1);
await graph.invoke({ value: 10 }, config2);

// 获取两个执行的历史
const history1 = await graph.getStateHistory(config1);
const history2 = await graph.getStateHistory(config2);

// 对比结果
const states1 = [];
const states2 = [];

for await (const state of history1) {
  states1.push(state.values.value);
}

for await (const state of history2) {
  states2.push(state.values.value);
}

console.log("执行 1 的状态变化:", states1);
console.log("执行 2 的状态变化:", states2);
```

### 3. 可视化执行流程

```typescript
import { StateGraph, Annotation, MemorySaver } from "@langchain/langgraph";

interface ExecutionStep {
  nodeId: string;
  timestamp: number;
  input: any;
  output: any;
  duration: number;
}

const State = Annotation.Root({
  data: Annotation<string>,
  executionLog: Annotation<ExecutionStep[]>({
    reducer: (prev, curr) => [...prev, ...curr],
    default: () => [],
  }),
});

// 创建带执行日志的节点包装器
function createLoggedNode(
  name: string,
  fn: (state: any) => Promise<any>
) {
  return async (state: any) => {
    const startTime = Date.now();
    const result = await fn(state);
    const endTime = Date.now();

    return {
      ...result,
      executionLog: [
        {
          nodeId: name,
          timestamp: startTime,
          input: state.data,
          output: result.data || state.data,
          duration: endTime - startTime,
        },
      ],
    };
  };
}

const graph = new StateGraph(State)
  .addNode(
    "processA",
    createLoggedNode("processA", async (state) => {
      return { data: `A: ${state.data}` };
    })
  )
  .addNode(
    "processB",
    createLoggedNode("processB", async (state) => {
      return { data: `B: ${state.data}` };
    })
  )
  .addEdge("__start__", "processA")
  .addEdge("processA", "processB")
  .addEdge("processB", "__end__")
  .compile({ checkpointer: new MemorySaver() });

const config = { configurable: { thread_id: "visual-thread" } };
const result = await graph.invoke({ data: "输入" }, config);

// 打印执行流程
console.log("\n=== 执行流程 ===");
for (const step of result.executionLog) {
  console.log(`[${step.nodeId}] ${step.timestamp} - 耗时: ${step.duration}ms`);
  console.log(`  输入: ${step.input}`);
  console.log(`  输出: ${step.output}`);
}
```

> **注意**：时间旅行功能需要检查点（Checkpointer）支持，确保图编译时配置了检查点。

> **建议**：在开发阶段积极使用时间旅行调试，可以大大加快问题定位和修复的速度。

> **提示**：使用 `getStateHistory` 时，返回的是一个异步迭代器，需要使用 `for await...of` 语法遍历。

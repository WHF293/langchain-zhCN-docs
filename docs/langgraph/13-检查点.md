> 来源: [LangGraph 官方文档](https://langchain-ai.github.io/langgraph/)

## 什么是检查点

检查点（Checkpointer）是 LangGraph 的状态持久化机制，它允许你保存图执行过程中的状态快照，支持暂停/恢复执行、时间旅行调试和错误恢复。

### 检查点的核心功能

| 功能 | 说明 |
|------|------|
| **状态持久化** | 将图的状态保存到外部存储 |
| **执行恢复** | 从任意检查点恢复执行 |
| **时间旅行** | 查看和回放历史状态 |
| **错误恢复** | 从失败点重新执行 |
| **并发控制** | 支持多线程并发访问 |

## MemorySaver

### 基本用法

```typescript
import { StateGraph, Annotation, MemorySaver } from "@langchain/langgraph";

const State = Annotation.Root({
  counter: Annotation<number>({
    default: () => 0,
    reducer: (_, curr) => curr,
  }),
  history: Annotation<string[]>({
    reducer: (prev, curr) => [...prev, ...curr],
    default: () => [],
  }),
});

// 创建内存检查点
const checkpointer = new MemorySaver();

const graph = new StateGraph(State)
  .addNode("increment", async (state) => {
    return {
      counter: state.counter + 1,
      history: [`计数器变为 ${state.counter + 1}`],
    };
  })
  .addEdge("__start__", "increment")
  .addEdge("increment", "__end__")
  .compile({ checkpointer });

// 执行图
const config = { configurable: { thread_id: "thread-1" } };
const result = await graph.invoke({ counter: 0 }, config);

console.log("结果:", result);
// 结果: { counter: 1, history: ["计数器变为 1"] }
```

### 多线程隔离

```typescript
import { StateGraph, Annotation, MemorySaver } from "@langchain/langgraph";

const State = Annotation.Root({
  value: Annotation<string>({
    default: () => "",
    reducer: (_, curr) => curr,
  }),
});

const checkpointer = new MemorySaver();

const graph = new StateGraph(State)
  .addNode("process", async (state) => {
    return { value: `处理: ${state.value}` };
  })
  .addEdge("__start__", "process")
  .addEdge("process", "__end__")
  .compile({ checkpointer });

// 不同的 thread_id 会创建独立的状态空间
const config1 = { configurable: { thread_id: "thread-A" } };
const config2 = { configurable: { thread_id: "thread-B" } };

await graph.invoke({ value: "输入 A" }, config1);
await graph.invoke({ value: "输入 B" }, config2);

// 每个线程的状态是独立的
const state1 = await graph.getState(config1);
const state2 = await graph.getState(config2);

console.log("线程 A:", state1.values);  // { value: "处理: 输入 A" }
console.log("线程 B:", state2.values);  // { value: "处理: 输入 B" }
```

### 获取检查点列表

```typescript
import { StateGraph, Annotation, MemorySaver } from "@langchain/langgraph";

const State = Annotation.Root({
  step: Annotation<number>({
    default: () => 0,
    reducer: (_, curr) => curr,
  }),
});

const checkpointer = new MemorySaver();

const graph = new StateGraph(State)
  .addNode("step", async (state) => {
    return { step: state.step + 1 };
  })
  .addEdge("__start__", "step")
  .addEdge("step", "__end__")
  .compile({ checkpointer });

const config = { configurable: { thread_id: "checkpoint-list" } };

// 多次执行，创建多个检查点
await graph.invoke({ step: 0 }, config);
await graph.invoke({ step: 10 }, config);
await graph.invoke({ step: 20 }, config);

// 获取所有检查点
const checkpoints = [];
for await (const checkpoint of checkpointer.list(config)) {
  checkpoints.push(checkpoint);
}

console.log("检查点数量:", checkpoints.length);
for (const cp of checkpoints) {
  console.log("检查点:", {
    id: cp.config.configurable?.checkpoint_id,
    step: cp.values.step,
  });
}
```

## SqliteSaver

### 安装和配置

```bash
npm install @langchain/langgraph-checkpoint-sqlite
```

### 基本使用

```typescript
import { StateGraph, Annotation } from "@langchain/langgraph";
import { SqliteSaver } from "@langchain/langgraph-checkpoint-sqlite";

const State = Annotation.Root({
  data: Annotation<string>,
  timestamp: Annotation<number>({
    default: () => Date.now(),
    reducer: (_, curr) => curr,
  }),
});

// 创建 SQLite 检查点（持久化到文件）
const checkpointer = new SqliteSaver({
  db: "./checkpoints.db",
});

const graph = new StateGraph(State)
  .addNode("process", async (state) => {
    return {
      data: `处理: ${state.data}`,
      timestamp: Date.now(),
    };
  })
  .addEdge("__start__", "process")
  .addEdge("process", "__end__")
  .compile({ checkpointer });

// 执行图
const config = { configurable: { thread_id: "sqlite-thread" } };
const result = await graph.invoke({ data: "输入数据" }, config);

console.log("结果:", result);
// 数据已持久化到 SQLite 文件
```

## PostgreSQL 检查点

### 安装和配置

```bash
npm install @langchain/langgraph-checkpoint-postgres
```

### 基本使用

```typescript
import { StateGraph, Annotation } from "@langchain/langgraph";
import { PostgresSaver } from "@langchain/langgraph-checkpoint-postgres";

const State = Annotation.Root({
  userId: Annotation<string>,
  sessionData: Annotation<Record<string, any>>({
    reducer: (prev, curr) => ({ ...prev, ...curr }),
    default: () => ({}),
  }),
});

// PostgreSQL 连接配置
const checkpointer = new PostgresSaver({
  connectionOptions: {
    host: "localhost",
    port: 5432,
    database: "langgraph",
    user: "postgres",
    password: "password",
  },
});

// 或使用连接字符串
const checkpointer2 = new PostgresSaver({
  connectionString: "postgresql://user:password@localhost:5432/langgraph",
});

const graph = new StateGraph(State)
  .addNode("updateSession", async (state) => {
    return {
      sessionData: {
        lastAccess: Date.now(),
        actions: [...(state.sessionData.actions || []), "update"],
      },
    };
  })
  .addEdge("__start__", "updateSession")
  .addEdge("updateSession", "__end__")
  .compile({ checkpointer });

// 执行图
const config = { configurable: { thread_id: "pg-thread" } };
const result = await graph.invoke(
  {
    userId: "user-123",
    sessionData: { startTime: Date.now() },
  },
  config
);

console.log("结果:", result);
// 数据已持久化到 PostgreSQL
```

### 连接池配置

```typescript
import { StateGraph, Annotation } from "@langchain/langgraph";
import { PostgresSaver } from "@langchain/langgraph-checkpoint-postgres";

const State = Annotation.Root({
  data: Annotation<string>,
});

// 使用连接池
const checkpointer = new PostgresSaver({
  connectionOptions: {
    host: "localhost",
    port: 5432,
    database: "langgraph",
    user: "postgres",
    password: "password",
    // 连接池配置
    max: 20,           // 最大连接数
    idleTimeoutMillis: 30000,  // 空闲超时
    connectionTimeoutMillis: 2000,  // 连接超时
  },
});

const graph = new StateGraph(State)
  .addNode("process", async (state) => {
    return { data: `处理: ${state.data}` };
  })
  .addEdge("__start__", "process")
  .addEdge("process", "__end__")
  .compile({ checkpointer });

// 并发执行多个任务
const tasks = Array.from({ length: 10 }, (_, i) => {
  const config = { configurable: { thread_id: `concurrent-${i}` } };
  return graph.invoke({ data: `任务 ${i}` }, config });
});

const results = await Promise.all(tasks);
console.log("并发执行完成:", results.length);
```

## 自定义检查点

### 实现 BaseCheckpointSaver 接口

```typescript
import {
  StateGraph,
  Annotation,
  BaseCheckpointSaver,
  Checkpoint,
  CheckpointMetadata,
  SerializerProtocol,
} from "@langchain/langgraph";

// 自定义检查点实现
class CustomCheckpointer extends BaseCheckpointSaver {
  private storage: Map<string, Checkpoint> = new Map();

  constructor(serializer?: SerializerProtocol) {
    super(serializer);
  }

  async get(config: any): Promise<Checkpoint | undefined> {
    const key = this.getKey(config);
    return this.storage.get(key);
  }

  async put(
    config: any,
    checkpoint: Checkpoint,
    metadata: CheckpointMetadata
  ): Promise<any> {
    const key = this.getKey(config);
    this.storage.set(key, checkpoint);
    return {
      configurable: {
        thread_id: config.configurable?.thread_id,
        checkpoint_id: checkpoint.id,
      },
    };
  }

  async *list(config: any): AsyncGenerator<Checkpoint> {
    const threadId = config.configurable?.thread_id;
    for (const [key, checkpoint] of this.storage) {
      if (key.startsWith(threadId)) {
        yield checkpoint;
      }
    }
  }

  private getKey(config: any): string {
    const threadId = config.configurable?.thread_id || "default";
    const checkpointId = config.configurable?.checkpoint_id || "latest";
    return `${threadId}:${checkpointId}`;
  }
}

// 使用自定义检查点
const State = Annotation.Root({
  data: Annotation<string>,
});

const checkpointer = new CustomCheckpointer();

const graph = new StateGraph(State)
  .addNode("process", async (state) => {
    return { data: `处理: ${state.data}` };
  })
  .addEdge("__start__", "process")
  .addEdge("process", "__end__")
  .compile({ checkpointer });

const config = { configurable: { thread_id: "custom-thread" } };
const result = await graph.invoke({ data: "测试" }, config);

console.log("结果:", result);
```

### Redis 检查点实现

```typescript
import {
  StateGraph,
  Annotation,
  BaseCheckpointSaver,
  Checkpoint,
  CheckpointMetadata,
} from "@langchain/langgraph";
import { createClient, RedisClientType } from "redis";

class RedisCheckpointer extends BaseCheckpointSaver {
  private client: RedisClientType;
  private prefix: string;

  constructor(options: { url?: string; prefix?: string }) {
    super();
    this.client = createClient({ url: options.url || "redis://localhost:6379" });
    this.prefix = options.prefix || "langgraph:checkpoint:";
  }

  async connect(): Promise<void> {
    await this.client.connect();
  }

  async disconnect(): Promise<void> {
    await this.client.disconnect();
  }

  async get(config: any): Promise<Checkpoint | undefined> {
    const key = this.getKey(config);
    const data = await this.client.get(key);
    if (!data) return undefined;
    return JSON.parse(data);
  }

  async put(
    config: any,
    checkpoint: Checkpoint,
    metadata: CheckpointMetadata
  ): Promise<any> {
    const key = this.getKey(config);
    await this.client.set(key, JSON.stringify(checkpoint));
    // 设置过期时间（24小时）
    await this.client.expire(key, 86400);
    return {
      configurable: {
        thread_id: config.configurable?.thread_id,
        checkpoint_id: checkpoint.id,
      },
    };
  }

  async *list(config: any): AsyncGenerator<Checkpoint> {
    const threadId = config.configurable?.thread_id;
    const pattern = `${this.prefix}${threadId}:*`;
    const keys = await this.client.keys(pattern);

    for (const key of keys) {
      const data = await this.client.get(key);
      if (data) {
        yield JSON.parse(data);
      }
    }
  }

  private getKey(config: any): string {
    const threadId = config.configurable?.thread_id || "default";
    const checkpointId = config.configurable?.checkpoint_id || "latest";
    return `${this.prefix}${threadId}:${checkpointId}`;
  }
}

// 使用 Redis 检查点
const State = Annotation.Root({
  data: Annotation<string>,
});

async function main() {
  const checkpointer = new RedisCheckpointer({
    url: "redis://localhost:6379",
    prefix: "myapp:checkpoint:",
  });

  await checkpointer.connect();

  const graph = new StateGraph(State)
    .addNode("process", async (state) => {
      return { data: `处理: ${state.data}` };
    })
    .addEdge("__start__", "process")
    .addEdge("process", "__end__")
    .compile({ checkpointer });

  const config = { configurable: { thread_id: "redis-thread" } };
  const result = await graph.invoke({ data: "测试" }, config);

  console.log("结果:", result);

  await checkpointer.disconnect();
}

main().catch(console.error);
```

### 检查点清理策略

```typescript
import {
  StateGraph,
  Annotation,
  BaseCheckpointSaver,
  Checkpoint,
  CheckpointMetadata,
} from "@langchain/langgraph";

interface CleanupStrategy {
  maxAge?: number;       // 最大存活时间（毫秒）
  maxCount?: number;     // 最大检查点数量
  keepLatest?: number;   // 保留最新的 N 个检查点
}

class ManagedCheckpointer extends BaseCheckpointSaver {
  private storage: Map<string, { checkpoint: Checkpoint; timestamp: number }> =
    new Map();
  private strategy: CleanupStrategy;

  constructor(strategy: CleanupStrategy = {}) {
    super();
    this.strategy = strategy;
  }

  async get(config: any): Promise<Checkpoint | undefined> {
    const key = this.getKey(config);
    const entry = this.storage.get(key);
    if (!entry) return undefined;

    // 检查是否过期
    if (this.strategy.maxAge) {
      const age = Date.now() - entry.timestamp;
      if (age > this.strategy.maxAge) {
        this.storage.delete(key);
        return undefined;
      }
    }

    return entry.checkpoint;
  }

  async put(
    config: any,
    checkpoint: Checkpoint,
    metadata: CheckpointMetadata
  ): Promise<any> {
    const key = this.getKey(config);
    this.storage.set(key, {
      checkpoint,
      timestamp: Date.now(),
    });

    // 执行清理策略
    await this.cleanup(config);

    return {
      configurable: {
        thread_id: config.configurable?.thread_id,
        checkpoint_id: checkpoint.id,
      },
    };
  }

  async *list(config: any): AsyncGenerator<Checkpoint> {
    const threadId = config.configurable?.thread_id;
    const entries = Array.from(this.storage.entries())
      .filter(([key]) => key.startsWith(threadId))
      .sort((a, b) => b[1].timestamp - a[1].timestamp);

    for (const [_, entry] of entries) {
      yield entry.checkpoint;
    }
  }

  private async cleanup(config: any): Promise<void> {
    const threadId = config.configurable?.thread_id;
    const entries = Array.from(this.storage.entries())
      .filter(([key]) => key.startsWith(threadId))
      .sort((a, b) => b[1].timestamp - a[1].timestamp);

    // 按数量清理
    if (this.strategy.maxCount && entries.length > this.strategy.maxCount) {
      const toDelete = entries.slice(this.strategy.maxCount);
      for (const [key] of toDelete) {
        this.storage.delete(key);
      }
    }

    // 按时间清理
    if (this.strategy.maxAge) {
      const now = Date.now();
      for (const [key, entry] of entries) {
        if (now - entry.timestamp > this.strategy.maxAge) {
          this.storage.delete(key);
        }
      }
    }
  }

  private getKey(config: any): string {
    const threadId = config.configurable?.thread_id || "default";
    const checkpointId = config.configurable?.checkpoint_id || "latest";
    return `${threadId}:${checkpointId}`;
  }
}

// 使用带清理策略的检查点
const State = Annotation.Root({
  data: Annotation<string>,
});

const checkpointer = new ManagedCheckpointer({
  maxAge: 3600000,   // 1小时过期
  maxCount: 100,      // 最多保留100个检查点
  keepLatest: 10,     // 保留最新的10个
});

const graph = new StateGraph(State)
  .addNode("process", async (state) => {
    return { data: `处理: ${state.data}` };
  })
  .addEdge("__start__", "process")
  .addEdge("process", "__end__")
  .compile({ checkpointer });
```

> **注意**：`MemorySaver` 适用于开发和测试环境，生产环境建议使用持久化存储（SQLite 或 PostgreSQL）。

> **建议**：对于需要高可用性的应用，建议使用 PostgreSQL 检查点，它支持并发访问和数据持久化。

> **提示**：自定义检查点时，确保正确实现 `get`、`put` 和 `list` 方法，以支持时间旅行和状态恢复功能。

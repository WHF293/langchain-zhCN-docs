# 第六章 技能（Skills）

> 来源: [Deep Agents 官方文档](https://docs.langchain.com/deep-agents)

---

技能（Skills）是 Deep Agents 的核心扩展机制，允许你为代理添加自定义能力。本章将详细介绍技能系统的设计理念、编写方法和高级用法。

## 渐进式披露（Progressive Disclosure）

Deep Agents 采用渐进式披露的设计原则，将复杂功能分层展示：

| 层级 | 描述 | 适用场景 |
|:-----|:-----|:---------|
| **基础层** | 简单的技能调用 | 快速原型开发 |
| **进阶层** | 自定义技能编写 | 生产环境部署 |
| **高级层** | 子代理技能、命名空间 | 大型项目架构 |

```typescript
// 基础层：直接使用内置技能
import { deepAgents } from "@langchain/deep-agents";

const agent = deepAgents.create({
  skills: ["web-search", "code-interpreter"]
});

// 进阶层：自定义技能
const customSkill = {
  name: "data-analysis",
  description: "分析数据并生成报告",
  execute: async (input) => {
    // 实现细节
  }
};
```

## 技能编写（Writing Skills）

### 基本结构

每个技能都是一个包含特定属性的对象：

```typescript
interface Skill {
  // 技能名称，用于唯一标识
  name: string;

  // 技能描述，帮助代理理解何时使用
  description: string;

  // 输入参数定义
  parameters?: {
    type: "object";
    properties: Record<string, ParameterDefinition>;
    required?: string[];
  };

  // 执行函数
  execute: (params: any, context: SkillContext) => Promise<SkillResult>;
}
```

### 完整示例

```typescript
// 创建一个天气查询技能
const weatherSkill: Skill = {
  name: "weather-query",
  description: "查询指定城市的天气信息，支持当前天气和未来预报",

  parameters: {
    type: "object",
    properties: {
      city: {
        type: "string",
        description: "城市名称，如 '北京'、'上海'"
      },
      days: {
        type: "number",
        description: "预报天数，1-7天",
        default: 1
      }
    },
    required: ["city"]
  },

  execute: async (params, context) => {
    const { city, days = 1 } = params;

    // 调用天气 API
    const response = await fetch(
      `https://api.weather.com/forecast?city=${city}&days=${days}`
    );
    const data = await response.json();

    return {
      success: true,
      data: {
        temperature: data.current.temp,
        condition: data.current.condition,
        forecast: data.forecast.slice(0, days)
      }
    };
  }
};
```

## 支持资源（Supporting Resources）

技能可以依赖外部资源来完成任务：

```typescript
// 定义资源提供者
const resourceProvider = {
  // 数据库连接
  database: async () => {
    return await createDbConnection();
  },

  // API 客户端
  apiClient: async () => {
    return new ApiClient({
      baseUrl: process.env.API_BASE_URL,
      apiKey: process.env.API_KEY
    });
  },

  // 文件系统访问
  fileSystem: async () => {
    return new FileSystemAdapter({
      rootDir: "/data"
    });
  }
};

// 在技能中使用资源
const dataSkill: Skill = {
  name: "data-query",
  description: "查询数据库中的用户数据",

  execute: async (params, context) => {
    // 从上下文获取数据库连接
    const db = await context.resources.database();

    const results = await db.query(
      "SELECT * FROM users WHERE status = ?",
      [params.status]
    );

    return { success: true, data: results };
  }
};
```

## 后端（Backend）

Deep Agents 支持多种后端实现，以适应不同的部署环境：

| 后端类型 | 特点 | 适用场景 |
|:---------|:-----|:---------|
| **本地后端** | 直接在进程内执行 | 开发测试 |
| **远程后端** | 通过 API 调用 | 微服务架构 |
| **容器后端** | 在容器中隔离执行 | 安全敏感场景 |

```typescript
// 配置远程后端
import { RemoteBackend } from "@langchain/deep-agents/backends";

const backend = new RemoteBackend({
  endpoint: "https://skills-service.example.com",
  auth: {
    type: "bearer",
    token: process.env.SKILLS_TOKEN
  },
  timeout: 30000,
  retries: 3
});

const agent = deepAgents.create({
  backend: backend,
  skills: ["remote-skill-1", "remote-skill-2"]
});
```

## 运行时加载（Runtime Loading）

技能可以在运行时动态加载，无需重启应用：

```typescript
// 动态加载技能
async function loadSkill(skillPath: string): Promise<Skill> {
  const skillModule = await import(skillPath);
  return skillModule.default;
}

// 注册到代理
const agent = deepAgents.create({
  skills: ["base-skill-1", "base-skill-2"]
});

// 运行时添加新技能
const newSkill = await loadSkill("./skills/analytics.js");
agent.addSkill(newSkill);

// 移除技能
agent.removeSkill("analytics");

// 热重载技能
agent.on("skill-update", (skillName) => {
  console.log(`技能 ${skillName} 已更新`);
});
```

## 命名空间技能（Namespaced Skills）

命名空间用于组织和管理大量技能，避免名称冲突：

```typescript
// 定义命名空间
const agent = deepAgents.create({
  skills: {
    // 数据处理命名空间
    "data": {
      "analyze": dataAnalyzeSkill,
      "transform": dataTransformSkill,
      "export": dataExportSkill
    },

    // 通信命名空间
    "comm": {
      "email": emailSkill,
      "sms": smsSkill,
      "webhook": webhookSkill
    },

    // 第三方集成
    "integrations": {
      "github": githubSkill,
      "slack": slackSkill,
      "jira": jiraSkill
    }
  }
});

// 调用命名空间技能
await agent.invoke("data.analyze", { dataset: "sales-2024" });
await agent.invoke("comm.email", { to: "user@example.com" });
```

### 命名空间配置

```typescript
// 命名空间高级配置
const namespaceConfig = {
  // 命名空间元数据
  metadata: {
    "data": {
      description: "数据处理相关技能",
      version: "1.0.0",
      author: "Data Team"
    }
  },

  // 访问控制
  accessControl: {
    "data": ["admin", "analyst"],
    "integrations": ["admin"]
  },

  // 技能发现
  discovery: {
    enabled: true,
    refreshInterval: 60000
  }
};
```

## 子代理技能（Sub-Agent Skills）

子代理技能允许将复杂任务委托给专门的子代理处理：

```typescript
// 定义子代理技能
const researchSubAgent = {
  name: "research-agent",
  type: "sub-agent",

  // 子代理配置
  config: {
    // 继承父代理的部分能力
    inheritSkills: ["web-search", "file-read"],

    // 独有的技能
    exclusiveSkills: ["academic-search", "citation-format"],

    // 上下文传递规则
    contextRules: {
      // 传递给子代理的上下文
      passThrough: ["userQuery", "sessionId"],

      // 不传递的敏感信息
      exclude: ["apiKey", "userData"]
    },

    // 执行限制
    limits: {
      maxTokens: 10000,
      maxToolCalls: 20,
      timeout: 120000
    }
  },

  // 何时使用此子代理
  canHandle: (task) => {
    return task.type === "research" ||
           task.keywords?.some(k => ["论文", "研究", "分析"].includes(k));
  },

  // 处理任务
  execute: async (task, context) => {
    const subAgent = deepAgents.createSubAgent(this.config);
    return await subAgent.invoke(task);
  }
};

// 使用子代理技能
const agent = deepAgents.create({
  skills: [researchSubAgent, otherSkills]
});
```

## 权限（Permissions）

技能权限系统确保安全性和访问控制：

```typescript
// 定义权限策略
const permissionPolicy = {
  // 技能级别权限
  skills: {
    "file-read": {
      // 允许的路径
      allowedPaths: ["/data/*", "/tmp/*"],

      // 禁止的路径
      deniedPaths: ["/etc/*", "/root/*"]
    },

    "database": {
      // 允许的操作
      allowedOperations: ["SELECT"],

      // 禁止的操作
      deniedOperations: ["DELETE", "DROP"]
    },

    "api-call": {
      // 允许的域名
      allowedDomains: ["api.example.com", "data.example.com"],

      // 速率限制
      rateLimit: {
        requests: 100,
        window: 60000 // 1 分钟
      }
    }
  },

  // 全局权限
  global: {
    // 是否允许网络访问
    networkAccess: true,

    // 是否允许文件系统访问
    fileSystemAccess: true,

    // 最大内存使用
    maxMemory: "512MB"
  }
};

// 应用权限策略
const agent = deepAgents.create({
  permissions: permissionPolicy
});
```

### 动态权限

```typescript
// 运行时权限检查
agent.on("permission-check", async (event) => {
  const { skill, action, resource } = event;

  // 调用权限服务
  const allowed = await checkPermission({
    skill: skill.name,
    action: action,
    resource: resource,
    user: event.context.user
  });

  if (!allowed) {
    throw new PermissionDeniedError(
      `技能 ${skill.name} 没有权限执行 ${action}`
    );
  }
});

// 权限请求和授权
agent.on("permission-request", async (request) => {
  // 向用户请求授权
  const approved = await requestUserApproval({
    skill: request.skill,
    action: request.action,
    reason: request.reason
  });

  return approved;
});
```

## 沙箱脚本的代码执行（Sandboxed Code Execution）

在安全沙箱中执行不可信代码：

```typescript
// 配置代码执行沙箱
import { SandboxExecutor } from "@langchain/deep-agents/sandbox";

const sandbox = new SandboxExecutor({
  // 沙箱类型
  type: "docker",  // 'docker' | 'vm' | 'wasm'

  // 资源限制
  resources: {
    cpu: "0.5",      // CPU 核心数
    memory: "256MB",  // 内存限制
    disk: "1GB",      // 磁盘空间
    timeout: 30000    // 执行超时（毫秒）
  },

  // 网络策略
  network: {
    // 允许访问的外部地址
    allowedHosts: ["api.example.com"],

    // 是否禁止所有出站连接
    disableOutbound: false
  },

  // 文件系统策略
  filesystem: {
    // 可写的目录
    writable: ["/tmp", "/workspace"],

    // 可读的目录
    readable: ["/data", "/shared"]
  }
});

// 创建沙箱执行技能
const codeExecutionSkill: Skill = {
  name: "code-executor",
  description: "在安全沙箱中执行 Python 或 JavaScript 代码",

  parameters: {
    type: "object",
    properties: {
      language: {
        type: "string",
        enum: ["python", "javascript"],
        description: "编程语言"
      },
      code: {
        type: "string",
        description: "要执行的代码"
      }
    },
    required: ["language", "code"]
  },

  execute: async (params) => {
    const { language, code } = params;

    // 在沙箱中执行代码
    const result = await sandbox.execute({
      language: language,
      code: code,
      // 注入可用的库
      libraries: language === "python"
        ? ["numpy", "pandas", "matplotlib"]
        : ["lodash", "moment"]
    });

    return {
      success: result.exitCode === 0,
      output: result.stdout,
      error: result.stderr,
      // 返回生成的文件
      artifacts: result.files
    };
  }
};

// 使用沙箱技能
const agent = deepAgents.create({
  skills: [codeExecutionSkill],
  sandbox: sandbox
});
```

> **注意**：沙箱执行环境是隔离的，代码执行不会影响宿主系统。但请注意资源限制，避免执行过于消耗资源的代码。

> **提示**：在生产环境中，建议使用容器化沙箱（如 Docker）以获得更好的隔离性和安全性。

> **建议**：对于频繁执行的代码片段，可以考虑预编译或缓存机制，以提高执行效率。

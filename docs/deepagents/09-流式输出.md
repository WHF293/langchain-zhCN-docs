# 第九章 流式输出（Streaming）

> 来源: [Deep Agents 官方文档](https://docs.langchain.com/deep-agents)

---

流式输出是 Deep Agents 的核心特性之一，允许实时获取代理的处理进度和结果。本章详细介绍流式输出的各种模式和使用方法。

## 子图流式（Subgraph Streaming）

子图流式允许监控代理内部图结构的执行过程：

```typescript
// 子图流式配置
import { deepAgents } from "@langchain/deep-agents";

const agent = deepAgents.create({
  // 启用子图流式
  streaming: {
    subgraph: {
      enabled: true,

      // 监控的节点类型
      nodeTypes: ["llm", "tool", "function", "condition"],

      // 事件过滤
      filter: {
        // 只关注特定节点
        include: ["research-node", "analysis-node"],
        // 排除调试节点
        exclude: ["debug-node"]
      }
    }
  }
});

// 使用子图流式
const stream = agent.streamGraph({
  input: "分析市场趋势",
  config: {
    // 流式模式
    mode: "values",  // 'values' | 'updates' | 'events'

    // 子图深度
    depth: 3
  }
});

// 处理子图事件
for await (const event of stream) {
  switch (event.type) {
    case "node_start":
      console.log(`节点开始: ${event.nodeId}`);
      break;

    case "node_complete":
      console.log(`节点完成: ${event.nodeId}`);
      console.log(`输出: ${event.output}`);
      break;

    case "edge_traversed":
      console.log(`边遍历: ${event.from} -> ${event.to}`);
      break;

    case "subgraph_enter":
      console.log(`进入子图: ${event.subgraphId}`);
      break;

    case "subgraph_exit":
      console.log(`退出子图: ${event.subgraphId}`);
      break;
  }
}
```

## 命名空间（Namespaces）

命名空间用于组织和过滤流式事件：

```typescript
// 命名空间配置
const namespaceConfig = {
  // 定义命名空间
  namespaces: {
    // 业务逻辑命名空间
    business: {
      description: "业务处理相关事件",
      events: ["task-start", "task-progress", "task-complete"]
    },

    // 技术命名空间
    technical: {
      description: "技术实现相关事件",
      events: ["api-call", "cache-hit", "db-query"]
    },

    // 性能命名空间
    performance: {
      description: "性能监控事件",
      events: ["latency", "memory-usage", "token-count"]
    }
  },

  // 默认命名空间
  defaultNamespace: "business"
};

// 创建带命名空间的流
const stream = agent.stream({
  input: input,
  namespaces: ["business", "performance"]  // 只订阅这两个命名空间
});

// 按命名空间过滤事件
for await (const event of stream) {
  switch (event.namespace) {
    case "business":
      handleBusinessEvent(event);
      break;

    case "performance":
      handlePerformanceEvent(event);
      break;
  }
}

// 动态订阅命名空间
stream.subscribe("technical");
stream.unsubscribe("performance");
```

## 子代理进度（Sub-Agent Progress）

跟踪子代理的执行进度：

```typescript
// 子代理进度配置
const progressConfig = {
  // 启用进度跟踪
  enabled: true,

  // 进度粒度
  granularity: "detailed",  // 'basic' | 'detailed' | 'fine-grained'

  // 进度事件
  events: {
    // 子代理启动
    subagentStart: true,

    // 子代理进度更新
    progressUpdate: true,

    // 子代理完成
    subagentComplete: true,

    // 子代理错误
    subagentError: true
  }
};

// 创建带进度跟踪的代理
const agent = deepAgents.create({
  subAgentProgress: progressConfig,

  subAgents: {
    "researcher": {
      // 子代理自定义进度报告
      progressReporter: {
        // 进度阶段
        stages: [
          { name: "搜索", weight: 0.3 },
          { name: "分析", weight: 0.4 },
          { name: "总结", weight: 0.3 }
        ],

        // 报告进度
        report: (stage, progress) => ({
          stage: stage.name,
          percentage: stage.weight * progress,
          message: `正在${stage.name}...`
        })
      }
    }
  }
});

// 监听子代理进度
const stream = agent.stream({
  input: "研究AI趋势",
  progress: true
});

for await (const event of stream) {
  if (event.type === "progress") {
    const { subagent, stage, percentage, message } = event;

    // 更新进度条
    updateProgressBar(subagent, percentage);
    console.log(`[${subagent}] ${stage}: ${message} (${percentage}%)`);
  }

  if (event.type === "subagent_complete") {
    console.log(`[${event.subagent}] 完成!`);
  }
}
```

## LLM Token 流式

实时获取 LLM 生成的 token：

```typescript
// Token 流式配置
const tokenStreamingConfig = {
  // 启用 token 流式
  enabled: true,

  // Token 处理选项
  options: {
    // 是否包含特殊 token
    includeSpecialTokens: false,

    // 是否解码 token
    decode: true,

    // 批处理大小（累积多少 token 后触发回调）
    batchSize: 1,

    // 延迟（毫秒，用于控制流速）
    delay: 0
  },

  // Token 回调
  callbacks: {
    // 每个 token
    onToken: (token) => {
      process.stdout.write(token);
    },

    // 批量 token
    onBatch: (tokens) => {
      console.log(`收到 ${tokens.length} 个 token`);
    },

    // Token 统计
    onStats: (stats) => {
      console.log(`Token/秒: ${stats.tokensPerSecond}`);
    }
  }
};

// 创建带 token 流式的代理
const agent = deepAgents.create({
  llm: {
    streaming: tokenStreamingConfig
  }
});

// 使用 token 流式
const stream = agent.streamTokens({
  input: "写一首诗",
  config: {
    // 流式模式
    mode: "raw",  // 'raw' | 'parsed' | 'structured'

    // 是否累积完整响应
    accumulate: true
  }
});

let fullResponse = "";

for await (const event of stream) {
  switch (event.type) {
    case "token":
      // 实时输出 token
      process.stdout.write(event.token);
      fullResponse += event.token;
      break;

    case "done":
      // 流式完成
      console.log("\n\n完整响应:", fullResponse);
      console.log("Token 统计:", event.stats);
      break;

    case "error":
      console.error("流式错误:", event.error);
      break;
  }
}
```

## 工具调用流式（Tool Call Streaming）

实时跟踪工具调用过程：

```typescript
// 工具调用流式配置
const toolCallConfig = {
  // 启用工具调用流式
  enabled: true,

  // 工具调用事件
  events: {
    // 工具调用开始
    callStart: true,

    // 工具参数流式
    argumentsStreaming: true,

    // 工具调用完成
    callComplete: true,

    // 工具执行结果
    result: true,

    // 工具错误
    error: true
  },

  // 参数处理
  argumentHandling: {
    // 流式解析参数
    streamParsing: true,

    // 验证参数
    validation: true
  }
};

// 创建带工具调用流式的代理
const agent = deepAgents.create({
  tools: {
    streaming: toolCallConfig,

    // 工具定义
    definitions: [
      {
        name: "search",
        description: "搜索网络",
        parameters: {
          type: "object",
          properties: {
            query: { type: "string" }
          }
        }
      }
    ]
  }
});

// 监听工具调用
const stream = agent.stream({
  input: "搜索最新新闻",
  toolCalls: true
});

for await (const event of stream) {
  switch (event.type) {
    case "tool_call_start":
      console.log(`\n🔧 调用工具: ${event.toolName}`);
      console.log(`   参数: ${JSON.stringify(event.arguments, null, 2)}`);
      break;

    case "tool_call_arguments":
      // 流式接收参数
      console.log(`   参数流: ${event.chunk}`);
      break;

    case "tool_call_complete":
      console.log(`   ✅ 调用完成`);
      break;

    case "tool_result":
      console.log(`   📊 结果: ${event.result}`);
      break;

    case "tool_error":
      console.log(`   ❌ 错误: ${event.error}`);
      break;
  }
}
```

## 自定义更新（Custom Updates）

定义和发送自定义流式更新：

```typescript
// 自定义更新配置
const customUpdateConfig = {
  // 定义更新类型
  types: {
    // 进度更新
    progress: {
      schema: {
        type: "object",
        properties: {
          percentage: { type: "number" },
          message: { type: "string" }
        }
      }
    },

    // 状态更新
    status: {
      schema: {
        type: "object",
        properties: {
          status: { type: "string", enum: ["processing", "waiting", "complete"] },
          details: { type: "string" }
        }
      }
    },

    // 数据更新
    data: {
      schema: {
        type: "object",
        properties: {
          key: { type: "string" },
          value: { type: "any" }
        }
      }
    }
  }
};

// 创建带自定义更新的代理
const agent = deepAgents.create({
  customUpdates: customUpdateConfig
});

// 在代理处理器中发送自定义更新
const handler = async (input, context) => {
  // 发送进度更新
  await context.emitUpdate({
    type: "progress",
    data: {
      percentage: 0,
      message: "开始处理..."
    }
  });

  // 执行任务
  const result = await processTask(input);

  // 发送状态更新
  await context.emitUpdate({
    type: "status",
    data: {
      status: "processing",
      details: "正在生成报告"
    }
  });

  // 发送数据更新
  await context.emitUpdate({
    type: "data",
    data: {
      key: "intermediateResult",
      value: result.partial
    }
  });

  return result;
};

// 消费自定义更新
const stream = agent.stream({
  input: input,
  customUpdates: true
});

for await (const event of stream) {
  if (event.type === "custom") {
    switch (event.updateType) {
      case "progress":
        updateProgressBar(event.data.percentage);
        console.log(event.data.message);
        break;

      case "status":
        console.log(`状态: ${event.data.status} - ${event.data.details}`);
        break;

      case "data":
        console.log(`数据更新: ${event.data.key} = `, event.data.value);
        break;
    }
  }
}
```

## 多模式流式（Multi-Mode Streaming）

同时使用多种流式模式：

```typescript
// 多模式流式配置
const multiModeConfig = {
  // 启用的模式
  modes: ["tokens", "events", "progress", "tools"],

  // 模式优先级
  priority: {
    tokens: 1,
    events: 2,
    progress: 3,
    tools: 4
  },

  // 聚合策略
  aggregation: {
    // 合并相似事件
    mergeSimilar: true,

    // 去重
    deduplicate: true,

    // 排序
    sortBy: "timestamp"
  }
};

// 创建多模式流
const stream = agent.streamMulti({
  input: input,
  modes: multiModeConfig.modes
});

// 处理多模式事件
for await (const event of stream) {
  // 事件包含模式标识
  console.log(`[${event.mode}] `, event);

  // 根据模式处理
  switch (event.mode) {
    case "tokens":
      process.stdout.write(event.token);
      break;

    case "events":
      handleGraphEvent(event);
      break;

    case "progress":
      updateProgress(event.progress);
      break;

    case "tools":
      handleToolCall(event.toolCall);
      break;
  }
}

// 合并所有模式的结果
const aggregated = await stream.aggregate();
console.log("聚合结果:", aggregated);
```

## 生命周期跟踪（Lifecycle Tracking）

跟踪代理和子代理的完整生命周期：

```typescript
// 生命周期跟踪配置
const lifecycleConfig = {
  // 启用跟踪
  enabled: true,

  // 跟踪的生命周期阶段
  stages: {
    // 初始化
    initialization: {
      onStart: true,
      onEnd: true,
      onError: true
    },

    // 配置
    configuration: {
      onStart: true,
      onEnd: true
    },

    // 执行
    execution: {
      onStart: true,
      onProgress: true,
      onEnd: true,
      onError: true,
      // 子阶段
      subStages: {
        planning: true,
        reasoning: true,
        action: true,
        reflection: true
      }
    },

    // 清理
    cleanup: {
      onStart: true,
      onEnd: true
    }
  },

  // 钩子函数
  hooks: {
    // 全局钩子
    global: {
      onStart: (context) => {
        console.log(`代理 ${context.agentId} 启动`);
      },
      onEnd: (context) => {
        console.log(`代理 ${context.agentId} 完成`);
      },
      onError: (context, error) => {
        console.error(`代理 ${context.agentId} 错误:`, error);
      }
    },

    // 阶段钩子
    stage: {
      "execution.onStart": (context) => {
        console.log("开始执行...");
      },
      "execution.onProgress": (context) => {
        console.log(`执行进度: ${context.progress}%`);
      }
    }
  },

  // 指标收集
  metrics: {
    // 收集的指标
    collect: [
      "duration",
      "token_count",
      "tool_calls",
      "memory_usage",
      "error_rate"
    ],

    // 指标输出
    output: {
      // 控制台输出
      console: true,

      // 文件输出
      file: "./metrics.json",

      // 外部监控系统
      prometheus: {
        enabled: true,
        port: 9090
      }
    }
  }
};

// 创建带生命周期跟踪的代理
const agent = deepAgents.create({
  lifecycle: lifecycleConfig
});

// 使用生命周期流
const lifecycleStream = agent.streamLifecycle({
  input: input
});

for await (const event of lifecycleStream) {
  switch (event.stage) {
    case "initialization":
      console.log(`🚀 初始化: ${event.phase}`);
      break;

    case "execution":
      console.log(`⚙️ 执行: ${event.phase} - ${event.detail}`);
      break;

    case "cleanup":
      console.log(`🧹 清理: ${event.phase}`);
      break;
  }
}

// 获取生命周期报告
const report = await agent.getLifecycleReport();
console.log("生命周期报告:", {
  duration: report.duration,
  stages: report.stages,
  metrics: report.metrics
});
```

> **注意**：流式输出会增加一定的性能开销，建议在开发和调试时启用，生产环境根据实际需求选择性开启。

> **提示**：使用 `stream.pause()` 和 `stream.resume()` 可以控制流式输出的节奏，避免消费者处理不过来。

> **建议**：对于长时间运行的任务，结合生命周期跟踪和进度报告，可以提供更好的用户体验。

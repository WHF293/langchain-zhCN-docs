# 第十九章 生产部署

> 来源: [Deep Agents 官方文档](https://docs.langchain.com/deep-agents)

将代理从开发环境推向生产环境需要考虑众多因素，包括可靠性、安全性、可扩展性和可观测性。本章涵盖生产部署的关键方面，从基础设施配置到安全护栏的实施。

## 多租户

多租户（Multi-Tenancy）架构允许单个部署服务多个客户或用户组，同时保持数据隔离。

```typescript
// 多租户配置
interface TenantConfig {
  tenantId: string;
  // 租户级别的配置覆盖
  overrides: {
    maxConcurrentRequests?: number;
    allowedModels?: string[];
    rateLimits?: RateLimitConfig;
    features?: FeatureFlags;
  };
  // 数据隔离策略
  isolation: {
    type: "database" | "schema" | "row";
    connectionString: string;
  };
}

// 租户解析中间件
class TenantResolverMiddleware {
  async resolve(req: Request): Promise<TenantConfig> {
    // 从请求头、JWT 或子域名提取租户标识
    const tenantId =
      req.headers["x-tenant-id"] ||
      this.extractFromJwt(req) ||
      this.extractFromSubdomain(req);

    if (!tenantId) {
      throw new Error("无法识别租户");
    }

    // 从配置存储获取租户配置
    const config = await this.tenantStore.get(tenantId);
    if (!config) {
      throw new Error(`租户 ${tenantId} 不存在`);
    }

    return config;
  }
}

// 在代理中应用多租户
const multiTenantAgent = createAgent({
  middleware: [new TenantResolverMiddleware()],
  // 使用租户配置动态设置参数
  configure: (tenant: TenantConfig) => ({
    maxConcurrentRequests: tenant.overrides.maxConcurrentRequests,
    allowedModels: tenant.overrides.allowedModels,
    // ...其他配置
  }),
});
```

> **注意**：多租户环境中的数据隔离至关重要。确保每个租户的数据在存储、处理和传输过程中都保持隔离。

## 认证

认证（Authentication）是生产环境的第一道防线，确保只有授权用户才能访问代理服务。

```typescript
// 认证配置
interface AuthConfig {
  // JWT 配置
  jwt: {
    secret: string;
    issuer: string;
    audience: string;
    expiresIn: string;
    refreshTokenExpiresIn: string;
  };
  // API Key 配置
  apiKey: {
    headerName: string;
    prefix: string;
    hashAlgorithm: string;
  };
  // OAuth2 配置
  oauth2: {
    providers: {
      google: { clientId: string; clientSecret: string };
      github: { clientId: string; clientSecret: string };
    };
    callbackUrl: string;
  };
}

// 认证中间件
class AuthMiddleware {
  private config: AuthConfig;

  async authenticate(req: Request): Promise<AuthResult> {
    // 尝试 JWT 认证
    const jwtToken = this.extractJwt(req);
    if (jwtToken) {
      return this.verifyJwt(jwtToken);
    }

    // 尝试 API Key 认证
    const apiKey = req.headers[this.config.apiKey.headerName];
    if (apiKey) {
      return this.verifyApiKey(apiKey);
    }

    // 尝试 OAuth2 令牌
    const oauthToken = req.headers["authorization"]?.replace("Bearer ", "");
    if (oauthToken) {
      return this.verifyOAuthToken(oauthToken);
    }

    throw new UnauthorizedError("缺少有效的认证凭据");
  }

  // 权限检查
  async authorize(user: User, requiredPermission: string): Promise<boolean> {
    const permissions = await this.getPermissions(user);
    return permissions.includes(requiredPermission);
  }
}

// 使用认证中间件
const secureAgent = createAgent({
  middleware: [
    new AuthMiddleware({
      jwt: {
        secret: process.env.JWT_SECRET!,
        issuer: "deep-agents-auth",
        audience: "deep-agents-api",
        expiresIn: "1h",
        refreshTokenExpiresIn: "7d",
      },
      apiKey: {
        headerName: "X-API-Key",
        prefix: "da_",
        hashAlgorithm: "sha256",
      },
      oauth2: {
        providers: {
          google: {
            clientId: process.env.GOOGLE_CLIENT_ID!,
            clientSecret: process.env.GOOGLE_CLIENT_SECRET!,
          },
          github: {
            clientId: process.env.GITHUB_CLIENT_ID!,
            clientSecret: process.env.GITHUB_CLIENT_SECRET!,
          },
        },
        callbackUrl: "https://api.example.com/auth/callback",
      },
    }),
  ],
});
```

| 认证方式 | 安全性 | 易用性 | 适用场景 |
|----------|--------|--------|----------|
| JWT | 高 | 中 | Web 应用、API 服务 |
| API Key | 中 | 高 | 服务间通信、脚本集成 |
| OAuth2 | 高 | 中 | 第三方集成、用户认证 |
| mTLS | 极高 | 低 | 高安全要求的服务间通信 |

> **建议**：在生产环境中，建议同时支持多种认证方式，并为不同的使用场景选择合适的认证策略。

## 异步

异步处理（Asynchronous Processing）是提高代理服务吞吐量和响应性的关键。

```typescript
// 异步任务配置
interface AsyncTaskConfig {
  // 消息队列配置
  queue: {
    provider: "redis" | "rabbitmq" | "sqs";
    connectionString: string;
    defaultPriority: number;
    retryPolicy: {
      maxRetries: number;
      backoffMultiplier: number;
      initialDelay: number;
    };
  };
  // 任务处理器配置
  worker: {
    concurrency: number;
    timeout: number;
    gracefulShutdownTimeout: number;
  };
}

// 异步任务管理器
class AsyncTaskManager {
  private queue: MessageQueue;
  private worker: TaskWorker;

  // 提交异步任务
  async submitTask(task: AgentTask): Promise<TaskHandle> {
    const taskId = generateTaskId();

    await this.queue.publish({
      id: taskId,
      type: task.type,
      payload: task.payload,
      priority: task.priority || 0,
      createdAt: new Date(),
    });

    return {
      taskId,
      statusUrl: `/tasks/${taskId}/status`,
      cancelUrl: `/tasks/${taskId}/cancel`,
    };
  }

  // 查询任务状态
  async getTaskStatus(taskId: string): Promise<TaskStatus> {
    const status = await this.taskStore.get(taskId);
    return {
      id: taskId,
      status: status.state,
      progress: status.progress,
      result: status.result,
      error: status.error,
      startedAt: status.startedAt,
      completedAt: status.completedAt,
    };
  }

  // 取消任务
  async cancelTask(taskId: string): Promise<void> {
    await this.queue.publish({
      id: taskId,
      type: "CANCEL",
      priority: 100, // 高优先级
    });
  }
}

// 异步代理调用示例
const asyncAgent = createAgent({
  taskManager: new AsyncTaskManager({
    queue: {
      provider: "redis",
      connectionString: process.env.REDIS_URL!,
      defaultPriority: 0,
      retryPolicy: {
        maxRetries: 3,
        backoffMultiplier: 2,
        initialDelay: 1000,
      },
    },
    worker: {
      concurrency: 10,
      timeout: 300000, // 5 分钟
      gracefulShutdownTimeout: 30000,
    },
  }),
});
```

> **提示**：对于长时间运行的任务，建议实现进度报告机制，让用户能够了解任务的执行状态。

## 耐久性

耐久性（Durability）确保代理的状态和数据在故障情况下不会丢失。

```typescript
// 耐久性配置
interface DurabilityConfig {
  // 检查点配置
  checkpointing: {
    enabled: boolean;
    interval: number; // 检查点间隔（毫秒）
    storage: {
      type: "database" | "s3" | "filesystem";
      connectionString: string;
    };
    maxCheckpoints: number; // 保留的最大检查点数
  };
  // 写入确认
  writeConcern: {
    level: "none" | "acknowledged" | "journaled" | "replicated";
    timeout: number;
  };
}

// 检查点管理器
class CheckpointManager {
  private storage: CheckpointStorage;
  private config: DurabilityConfig;

  // 创建检查点
  async createCheckpoint(state: AgentState): Promise<Checkpoint> {
    const checkpoint: Checkpoint = {
      id: generateCheckpointId(),
      timestamp: new Date(),
      state: this.serializeState(state),
      metadata: {
        version: state.version,
        turnNumber: state.turnNumber,
      },
    };

    // 持久化检查点
    await this.storage.save(checkpoint, {
      writeConcern: this.config.writeConcern.level,
      timeout: this.config.writeConcern.timeout,
    });

    // 清理旧检查点
    await this.cleanupOldCheckpoints();

    return checkpoint;
  }

  // 恢复检查点
  async restoreCheckpoint(checkpointId?: string): Promise<AgentState> {
    const checkpoint = checkpointId
      ? await this.storage.get(checkpointId)
      : await this.storage.getLatest();

    if (!checkpoint) {
      throw new Error("找不到可用的检查点");
    }

    return this.deserializeState(checkpoint.state);
  }

  // 自动检查点调度
  startAutoCheckpoint(getState: () => AgentState): void {
    setInterval(async () => {
      try {
        const state = getState();
        await this.createCheckpoint(state);
      } catch (error) {
        console.error("自动检查点创建失败:", error);
      }
    }, this.config.checkpointing.interval);
  }
}
```

| 耐久性级别 | 数据丢失风险 | 性能影响 | 适用场景 |
|------------|--------------|----------|----------|
| 无耐久性 | 高 | 无 | 临时缓存、非关键数据 |
| 基本耐久性 | 中 | 低 | 一般业务数据 |
| 高耐久性 | 低 | 中 | 重要业务数据 |
| 极致耐久性 | 极低 | 高 | 关键业务数据、金融交易 |

> **建议**：根据业务数据的重要性选择合适的耐久性级别。对于代理的对话历史和状态，建议至少使用高耐久性级别。

## 记忆范围

记忆范围（Memory Scope）定义了记忆在不同维度上的可见性和生命周期。

```typescript
// 记忆范围配置
interface MemoryScopeConfig {
  // 用户级别记忆
  user: {
    enabled: boolean;
    ttl: number; // 生存时间（秒）
    maxSize: number; // 最大存储大小（字节）
    encryption: boolean;
  };
  // 会话级别记忆
  session: {
    enabled: boolean;
    ttl: number;
    maxSize: number;
  };
  // 组织级别记忆
  organization: {
    enabled: boolean;
    ttl: number;
    maxSize: number;
    sharedAcrossUsers: boolean;
  };
  // 全局记忆
  global: {
    enabled: boolean;
    ttl: number;
    maxSize: number;
    readOnly: boolean;
  };
}

// 记忆范围管理器
class MemoryScopeManager {
  private config: MemoryScopeConfig;
  private stores: Map<string, MemoryStore>;

  constructor(config: MemoryScopeConfig) {
    this.config = config;
    this.stores = new Map();

    // 初始化各范围的存储
    if (config.user.enabled) {
      this.stores.set("user", new MemoryStore(config.user));
    }
    if (config.session.enabled) {
      this.stores.set("session", new MemoryStore(config.session));
    }
    if (config.organization.enabled) {
      this.stores.set("organization", new MemoryStore(config.organization));
    }
    if (config.global.enabled) {
      this.stores.set("global", new MemoryStore(config.global));
    }
  }

  // 获取记忆，按优先级从多个范围检索
  async recall(key: string, context: MemoryContext): Promise<Memory | null> {
    const scopes = this.getRelevantScopes(context);

    for (const scope of scopes) {
      const store = this.stores.get(scope);
      if (store) {
        const memory = await store.get(key);
        if (memory) {
          return { ...memory, scope };
        }
      }
    }

    return null;
  }

  // 存储记忆到指定范围
  async remember(
    key: string,
    value: any,
    scope: string,
    context: MemoryContext
  ): Promise<void> {
    const store = this.stores.get(scope);
    if (!store) {
      throw new Error(`记忆范围 ${scope} 不存在或未启用`);
    }

    // 检查写入权限
    if (!this.canWrite(scope, context)) {
      throw new Error(`无权写入记忆范围 ${scope}`);
    }

    await store.set(key, {
      value,
      createdAt: new Date(),
      updatedAt: new Date(),
      createdBy: context.userId,
    });
  }

  private getRelevantScopes(context: MemoryContext): string[] {
    // 按优先级排序：用户 > 会话 > 组织 > 全局
    const scopes: string[] = [];

    if (this.config.user.enabled) scopes.push("user");
    if (this.config.session.enabled) scopes.push("session");
    if (this.config.organization.enabled) scopes.push("organization");
    if (this.config.global.enabled) scopes.push("global");

    return scopes;
  }
}
```

## 沙箱

沙箱（Sandbox）为代理提供安全的执行环境，隔离潜在的危险操作。

### 生命周期

```typescript
// 沙箱生命周期管理
interface SandboxLifecycleConfig {
  // 创建配置
  creation: {
    template: string; // 基础镜像模板
    resources: {
      cpu: string; // CPU 限制，如 "1" 或 "500m"
      memory: string; // 内存限制，如 "512Mi"
      disk: string; // 磁盘限制，如 "1Gi"
    };
    networkPolicy: "isolated" | "restricted" | "open";
  };
  // 运行时配置
  runtime: {
    maxLifetime: number; // 最大生存时间（秒）
    idleTimeout: number; // 空闲超时（秒）
    maxConcurrentProcesses: number;
  };
  // 销毁配置
  destruction: {
    gracefulShutdownTimeout: number;
    cleanupTimeout: number;
    preserveArtifacts: boolean;
  };
}

// 沙箱管理器
class SandboxManager {
  private config: SandboxLifecycleConfig;
  private activeSandboxes: Map<string, Sandbox>;

  // 创建沙箱
  async createSandbox(options?: CreateSandboxOptions): Promise<Sandbox> {
    const sandboxId = generateSandboxId();

    const sandbox = await this.containerRuntime.create({
      id: sandboxId,
      image: this.config.creation.template,
      resources: this.config.creation.resources,
      networkPolicy: this.config.creation.networkPolicy,
      ...options,
    });

    // 设置生命周期钩子
    sandbox.on("idle", () => this.handleIdle(sandboxId));
    sandbox.on("error", (err) => this.handleError(sandboxId, err));

    // 设置自动销毁定时器
    setTimeout(() => {
      this.destroySandbox(sandboxId);
    }, this.config.runtime.maxLifetime * 1000);

    this.activeSandboxes.set(sandboxId, sandbox);

    return {
      id: sandboxId,
      endpoint: sandbox.getEndpoint(),
      status: "ready",
    };
  }

  // 销毁沙箱
  async destroySandbox(sandboxId: string): Promise<void> {
    const sandbox = this.activeSandboxes.get(sandboxId);
    if (!sandbox) return;

    try {
      // 优雅关闭
      await sandbox.shutdown({
        timeout: this.config.destruction.gracefulShutdownTimeout,
      });

      // 保存产物（如果配置了）
      if (this.config.destruction.preserveArtifacts) {
        await this.preserveArtifacts(sandboxId);
      }

      // 清理资源
      await sandbox.cleanup({
        timeout: this.config.destruction.cleanupTimeout,
      });

      this.activeSandboxes.delete(sandboxId);
    } catch (error) {
      console.error(`沙箱 ${sandboxId} 销毁失败:`, error);
      // 强制销毁
      await sandbox.forceDestroy();
    }
  }
}
```

### 文件传输

```typescript
// 沙箱文件传输配置
interface FileTransferConfig {
  // 上传配置
  upload: {
    maxSize: number; // 最大文件大小（字节）
    allowedTypes: string[]; // 允许的文件类型
    scanForViruses: boolean;
    quarantineOnSuspicious: boolean;
  };
  // 下载配置
  download: {
    maxFiles: number; // 单次最大下载文件数
    maxTotalSize: number;
    requireApproval: boolean;
  };
}

// 文件传输管理器
class SandboxFileTransfer {
  private config: FileTransferConfig;

  // 上传文件到沙箱
  async uploadToSandbox(
    sandboxId: string,
    file: File
  ): Promise<UploadResult> {
    // 验证文件
    await this.validateFile(file);

    // 病毒扫描
    if (this.config.upload.scanForViruses) {
      const scanResult = await this.virusScanner.scan(file);
      if (scanResult.isSuspicious) {
        if (this.config.upload.quarantineOnSuspicious) {
          await this.quarantineFile(file);
          throw new Error("文件被隔离：检测到可疑内容");
        }
      }
    }

    // 传输文件
    const result = await this.transferToSandbox(sandboxId, file);

    return {
      fileId: result.fileId,
      path: result.path,
      size: file.size,
      checksum: result.checksum,
    };
  }

  // 从沙箱下载文件
  async downloadFromSandbox(
    sandboxId: string,
    filePath: string
  ): Promise<Buffer> {
    // 检查权限
    if (this.config.download.requireApproval) {
      const approved = await this.requestApproval(sandboxId, filePath);
      if (!approved) {
        throw new Error("文件下载未获批准");
      }
    }

    // 下载文件
    const content = await this.transferFromSandbox(sandboxId, filePath);

    // 内容验证
    await this.validateContent(content);

    return content;
  }

  private async validateFile(file: File): Promise<void> {
    if (file.size > this.config.upload.maxSize) {
      throw new Error(`文件大小超过限制：${this.config.upload.maxSize} 字节`);
    }

    const ext = path.extname(file.name).toLowerCase();
    if (!this.config.upload.allowedTypes.includes(ext)) {
      throw new Error(`不允许的文件类型：${ext}`);
    }
  }
}
```

### 密钥

```typescript
// 沙箱密钥管理
interface SandboxSecretsConfig {
  // 密钥存储
  storage: {
    provider: "vault" | "aws-secrets-manager" | "azure-key-vault";
    connectionString: string;
    encryptionKey: string;
  };
  // 密钥注入策略
  injection: {
    method: "environment" | "file" | "api";
    maskingEnabled: boolean; // 在日志中掩码显示
  };
  // 密钥轮换
  rotation: {
    enabled: boolean;
    interval: number; // 轮换间隔（天）
    overlapPeriod: number; // 新旧密钥共存期（小时）
  };
}

// 沙箱密钥管理器
class SandboxSecretsManager {
  private config: SandboxSecretsConfig;
  private vault: SecretStore;

  // 向沙箱注入密钥
  async injectSecrets(
    sandboxId: string,
    secrets: SecretReference[]
  ): Promise<void> {
    // 从密钥存储获取实际值
    const secretValues = await Promise.all(
      secrets.map(async (ref) => ({
        name: ref.name,
        value: await this.vault.get(ref.path),
      }))
    );

    // 根据注入策略注入密钥
    switch (this.config.injection.method) {
      case "environment":
        await this.injectAsEnvironment(sandboxId, secretValues);
        break;
      case "file":
        await this.injectAsFile(sandboxId, secretValues);
        break;
      case "api":
        await this.injectAsApi(sandboxId, secretValues);
        break;
    }

    // 记录注入日志（使用掩码）
    this.logSecretInjection(sandboxId, secrets, {
      masking: this.config.injection.maskingEnabled,
    });
  }

  // 轮换沙箱密钥
  async rotateSecrets(sandboxId: string): Promise<void> {
    const currentSecrets = await this.getSandboxSecrets(sandboxId);

    for (const secret of currentSecrets) {
      // 生成新密钥
      const newValue = await this.generateNewSecret(secret);

      // 更新密钥存储
      await this.vault.update(secret.path, newValue, {
        // 保留旧值直到轮换完成
        preservePrevious: true,
        overlapPeriod: this.config.rotation.overlapPeriod,
      });

      // 更新沙箱中的密钥
      await this.updateSandboxSecret(sandboxId, secret.name, newValue);
    }
  }

  // 清理沙箱密钥
  async cleanupSecrets(sandboxId: string): Promise<void> {
    const secrets = await this.getSandboxSecrets(sandboxId);

    // 从沙箱中移除密钥
    await this.removeAllSecrets(sandboxId);

    // 记录清理操作
    this.logSecretCleanup(sandboxId, secrets);
  }
}
```

```typescript
  // 检查是否超出速率限制
  async checkLimit(request: Request): Promise<RateLimitResult> {

    const globalLimit = await this.checkGlobalLimit();

    if (!tenantLimit.allowed) {
      return {
        allowed: false,
        reason: "租户速率限制",
        retryAfter: tenantLimit.retryAfter,
      };
    }

    // 消耗配额
    await this.consumeQuota(userId, tenantId);
```

```typescript
interface ErrorHandlingConfig {
  // 错误分类
  classification: {
    retryable: string[]; // 可重试的错误类型
    fatal: string[]; // 致命错误类型

      type: "exponential" | "linear" | "fixed";
      initialDelay: number;
      maxDelay: number;
    };

    enabled: boolean;
    strategy: "cached" | "default" | "simplified";

// 错误处理器
class ErrorHandler {
  private config: ErrorHandlingConfig;

  // 处理错误
  async handleError(error: Error, context: ErrorContext): Promise<ErrorResult> {
    // 分类错误
    const classification = this.classifyError(error);

    // 记录错误
    await this.logError(error, context, classification);

    // 根据分类处理
    switch (classification.type) {
      case "retryable":
        return this.handleRetryableError(error, context);
      case "fatal":
        return this.handleFatalError(error, context);
      case "userFacing":
        return this.handleUserFacingError(error, context);
      default:
        return this.handleUnknownError(error, context);
    }
  }

  // 处理可重试错误
  private async handleRetryableError(
    error: Error,
    context: ErrorContext
  ): Promise<ErrorResult> {
    const retryCount = context.retryCount || 0;

    if (retryCount < this.config.retry.maxRetries) {
      // 计算退避延迟
      const delay = this.calculateBackoff(retryCount);

      // 等待后重试
      await this.sleep(delay);

      return {
        action: "retry",
        delay,
        retryCount: retryCount + 1,
      };
    }

    // 超过重试次数，使用降级策略
    if (this.config.fallback.enabled) {
      return {
        action: "fallback",
        strategy: this.config.fallback.strategy,
      };
    }

    return { action: "fail", error };
  }

  // 处理面向用户的错误
  private async handleUserFacingError(
    error: Error,
    context: ErrorContext
  ): Promise<ErrorResult> {
    // 生成用户友好的错误消息
    const userMessage = this.generateUserMessage(error);

    return {
      action: "respond",
      response: {
        error: true,
        message: userMessage,
        code: error.code,
        supportId: context.requestId,
      },
    };
  }
```

```typescript
class DataPrivacyManager {
  private config: DataPrivacyConfig;

  // 脱敏敏感数据
  maskSensitiveData(text: string): string {
    let masked = text;

    // 应用所有掩码模式
    for (const [type, pattern] of Object.entries(this.config.masking.patterns)) {
      masked = masked.replace(pattern, (match) => {
        return this.applyMask(match);
      });
    }

    userId: string,
    request: DeletionRequest
  ): Promise<DeletionResult> {

      if (!request.confirmed) {
        return {
          status: "pending_confirmation",
          message: "请确认删除请求",
        };
      }
    }

    // 收集要删除的数据
    const dataToDelete = await this.collectUserData(userId);

    // 执行删除
    const results = await Promise.all([
      this.deleteConversations(userId),
      this.deleteLogs(userId),
      this.deleteMemories(userId),
      this.deleteFiles(userId),
    ]);

    // 记录删除操作（用于审计）
    await this.logDeletion(userId, dataToDelete);

    return {

      deletedItems: results.reduce((sum, r) => sum + r.count, 0),

  private applyMask(value: string): string {
    const { maskChar, preservePrefix, preserveSuffix } = this.config.masking;

    if (value.length <= preservePrefix + preserveSuffix) {
      return maskChar.repeat(value.length);
    }

    const prefix = value.slice(0, preservePrefix);
    const suffix = value.slice(-preserveSuffix);
    const maskedLength = value.length - preservePrefix - preserveSuffix;

    return `${prefix}${maskChar.repeat(maskedLength)}${suffix}`;
```

```typescript
class DeepAgentSDK {
  private config: FrontendSDKConfig;
  private ws: WebSocket | null = null;

  // 初始化连接
  async connect(): Promise<void> {
    // 建立 WebSocket 连接
    this.ws = new WebSocket(`${this.config.endpoint}/ws`);

    this.ws.onopen = () => {
      console.log("已连接到代理服务");
      this.authenticate();
    };

    this.ws.onmessage = (event) => {
      this.handleMessage(JSON.parse(event.data));
    };

    this.ws.onerror = (error) => {
      console.error("连接错误:", error);
      this.handleError(error);
    };

    this.ws.onclose = () => {
      console.log("连接已关闭");
      this.handleDisconnect();
    };
  }

  // 发送消息
  async sendMessage(message: string): Promise<AgentResponse> {
    return new Promise((resolve, reject) => {
      const messageId = generateMessageId();

      // 设置响应监听
      this.once(`response:${messageId}`, (response) => {
        if (response.error) {
          reject(new Error(response.error));
        } else {
          resolve(response);
        }
      });

      // 发送消息
      this.ws?.send(
        JSON.stringify({
          type: "message",
          id: messageId,
          content: message,
          timestamp: new Date(),
        })
      );
    });
  }

  // 流式接收响应
  async *streamResponse(message: string): AsyncGenerator<StreamChunk> {
    const streamId = generateStreamId();

    // 发起流式请求
    this.ws?.send(
      JSON.stringify({
        type: "stream",
        id: streamId,
        content: message,
      })
    );

    // 生成器：逐块接收响应
    while (true) {
      const chunk = await this.waitForChunk(streamId);

      if (chunk.done) {
        break;
      }

      yield chunk;
    }
  }

  // React 集成示例
  static createReactHook(config: FrontendSDKConfig) {
    return function useDeepAgent() {
      const [sdk] = useState(() => new DeepAgentSDK(config));
      const [isConnected, setIsConnected] = useState(false);
      const [messages, setMessages] = useState<Message[]>([]);

      useEffect(() => {
        sdk.connect().then(() => setIsConnected(true));

        return () => {
          sdk.disconnect();
        };
      }, [sdk]);

      const sendMessage = useCallback(
        async (content: string) => {
          const userMessage: Message = {
            id: generateId(),
            role: "user",
            content,
            timestamp: new Date(),
          };

          setMessages((prev) => [...prev, userMessage]);

          // 流式接收响应
          let agentContent = "";
          const agentMessage: Message = {
            id: generateId(),
            role: "agent",
            content: "",
            timestamp: new Date(),
          };

          setMessages((prev) => [...prev, agentMessage]);

          for await (const chunk of sdk.streamResponse(content)) {
            agentContent += chunk.content;
            setMessages((prev) =>
              prev.map((m) =>
                m.id === agentMessage.id
                  ? { ...m, content: agentContent }
                  : m
              )
            );
          }
        },
        [sdk]
      );

      return {
        isConnected,
        messages,
        sendMessage,
      };
    };
  }
}
```

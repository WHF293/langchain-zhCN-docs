## 🔧 创建搜索工具

首先，我们创建一个简单的搜索工具，让代理能够搜索互联网：

```typescript
import { createTool } from "@langchain/deep-agents";

// 创建搜索工具
const searchTool = createTool({
  name: "web_search",
  description: "搜索互联网获取信息",
  parameters: {
    query: {
      type: "string",
      description: "搜索关键词",
    },
  },
  execute: async ({ query }) => {
    // 这里使用示例实现，实际项目中接入真实搜索 API
    console.log(`搜索: ${query}`);

    // 模拟搜索结果
    return {
      results: [
        { title: "搜索结果1", snippet: "相关内容摘要..." },
        { title: "搜索结果2", snippet: "相关内容摘要..." },
      ],
    };
  },
});

export { searchTool };
```

---

## 🤖 创建 Deep Agent

接下来，创建一个 Deep Agent 实例：

```typescript
import { DeepAgent } from "@langchain/deep-agents";
import { searchTool } from "./tools/search";

// 创建 Deep Agent
const agent = new DeepAgent({
  // 模型配置
  model: "anthropic/claude-sonnet-4-20250514",

  // 系统提示
  systemPrompt: `你是一个 helpful 的 AI 助手。
你可以搜索互联网获取信息，并回答用户的问题。
回答时请使用中文，并保持简洁明了。`,

  // 工具配置
  tools: [searchTool],

  // 记忆配置（可选）
  memory: {
    type: "short-term",
  },
});

export { agent };
```

## ▶️ 运行 Agent

现在，让我们运行这个 Agent：

```typescript
import { agent } from "./agent";

async function main() {
  try {
    // 运行代理，发送消息
    const response = await agent.run({
      messages: [
        {
          role: "user",
          content: "什么是 Deep Agents？它有什么特点？",
        },
      ],
    });

    // 输出响应
    console.log("Agent 回复:");
    console.log(response.content);

    // 如果有工具调用，输出工具调用信息
    if (response.toolCalls) {
      console.log("\n工具调用:");
      response.toolCalls.forEach((call) => {
        console.log(`- ${call.name}: ${JSON.stringify(call.arguments)}`);
      });
    }
  } catch (error) {
    console.error("运行出错:", error);
  }
}

main();
```

## 📝 完整示例

将上述代码整合到一个文件中：

```typescript
// main.ts
import { DeepAgent, createTool } from "@langchain/deep-agents";

// 1. 创建搜索工具
const searchTool = createTool({
  name: "web_search",
  description: "搜索互联网获取信息",
  parameters: {
    query: {
      type: "string",
      description: "搜索关键词",
    },
  },
  execute: async ({ query }) => {
    console.log(`搜索: ${query}`);
    return {
      results: [
        { title: "搜索结果1", snippet: "相关内容摘要..." },
      ],
    };
  },
});

// 2. 创建 Agent
const agent = new DeepAgent({
  model: "anthropic/claude-sonnet-4-20250514",
  systemPrompt: "你是一个 helpful 的 AI 助手，可以搜索互联网获取信息。",
  tools: [searchTool],
});

// 3. 运行 Agent
async function main() {
  const response = await agent.run({
    messages: [
      { role: "user", content: "什么是 Deep Agents？" },
    ],
  });

  console.log("回复:", response.content);
}

main().catch(console.error);
```

运行命令：

```bash
npx ts-node main.ts
```

## ❓ 常见问题

### 如何设置 API Key？

Deep Agents 使用环境变量或配置文件设置 API Key：

```bash
# 方式1: 环境变量
export ANTHROPIC_API_KEY="your-api-key"

# 方式2: .env 文件
echo 'ANTHROPIC_API_KEY=your-api-key' > .env
```

### 支持哪些模型？

Deep Agents 支持多种模型，详见 [模型配置](./04-模型配置.md) 文档。

### 如何调试？

启用调试模式查看详细日志：

```typescript
const agent = new DeepAgent({
  model: "anthropic/claude-sonnet-4-20250514",
  debug: true, // 启用调试模式
});
```

---

> **提示**：建议在开发过程中启用 `debug` 模式，以便查看代理的推理过程和工具调用细节。

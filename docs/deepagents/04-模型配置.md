## 📋 支持的模型列表

Deep Agents 支持多种模型提供者，以下是完整列表：

### Anthropic Claude

| 模型 | 说明 | 推荐场景 |
|------|------|----------|
| `claude-opus-4-20250514` | Claude Opus 4 | 复杂推理、代码生成 |
| `claude-sonnet-4-20250514` | Claude Sonnet 4 | 通用任务、性价比高 |
| `claude-haiku-3.5` | Claude Haiku 3.5 | 快速响应、低成本 |

### OpenAI

| 模型 | 说明 | 推荐场景 |
|------|------|----------|
| `gpt-4o` | GPT-4o | 多模态、通用任务 |
| `gpt-4o-mini` | GPT-4o Mini | 快速响应、低成本 |
| `gpt-4-turbo` | GPT-4 Turbo | 长上下文、复杂任务 |
| `o1` | o1 | 复杂推理 |
| `o3-mini` | o3-mini | 快速推理 |

### Google

| 模型 | 说明 | 推荐场景 |
|------|------|----------|
| `gemini-2.5-pro` | Gemini 2.5 Pro | 复杂任务、长上下文 |
| `gemini-2.5-flash` | Gemini 2.5 Flash | 快速响应 |
| `gemini-2.0-flash` | Gemini 2.0 Flash | 通用任务 |

### 本地模型 (Ollama)

| 模型 | 说明 | 推荐场景 |
|------|------|----------|
| `llama3` | Llama 3 | 本地部署、隐私保护 |
| `mistral` | Mistral | 本地部署、快速响应 |
| `codellama` | Code Llama | 代码生成 |

## ⭐ 推荐模型

### 生产环境推荐

```typescript

  model: "anthropic/claude-sonnet-4-20250514",
  temperature: 0.7,
```

## 📝 提供者配置文件

## 🔄 运行时模型选择

### 静态配置

```typescript
// 固定使用一个模型
const agent = new DeepAgent({
  model: "anthropic/claude-sonnet-4-20250514",
});
```

### 动态选择

```typescript
// 根据任务动态选择模型
const agent = new DeepAgent({
  modelSelector: (task) => {
    // 简单任务使用快速模型
    if (task.complexity === "low") {
      return "anthropic/claude-haiku-3.5";
    }
    // 代码任务使用代码能力强的模型
    if (task.type === "coding") {
      return "anthropic/claude-sonnet-4-20250514";
    }
    // 复杂推理使用最强模型
    return "anthropic/claude-opus-4-20250514";
  },
});
```

### 模型回退

```typescript
// 配置模型回退机制
const agent = new DeepAgent({
  model: "anthropic/claude-opus-4-20250514",
  fallback: [
    "anthropic/claude-sonnet-4-20250514",
    "openai/gpt-4o",
  ],
  retry: {
    maxAttempts: 3,
    delay: 1000,
  },
});
```

## 🔧 高级配置

### 温度与采样

```typescript
const agent = new DeepAgent({
  model: "anthropic/claude-sonnet-4-20250514",
  temperature: 0.7,      // 0-1，越高越随机
  topP: 0.9,            // 核采样
  maxTokens: 4096,      // 最大输出 token 数
});
```

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| `temperature` | 控制输出随机性 | 0.7 (通用), 0 (精确) |
| `topP` | 核采样阈值 | 0.9 |
| `maxTokens` | 最大输出长度 | 4096 |

### 上下文窗口

```typescript
const agent = new DeepAgent({
  model: "anthropic/claude-sonnet-4-20250514",
  contextWindow: {
    maxTokens: 200000,  // 最大上下文长度
    strategy: "sliding", // 滑动窗口策略
  },
});
```


## 💡 最佳实践

1. **生产环境**：使用 Claude Sonnet 4，平衡性能与成本
2. **开发环境**：使用 Claude Haiku 3.5，快速迭代
3. **复杂任务**：使用 Claude Opus 4，确保质量
4. **隐私敏感**：使用 Ollama 本地部署
5. **多模型策略**：配置动态选择，优化成本

---

> **建议**：建议在项目中配置模型回退机制，当主模型不可用时自动切换到备用模型，提高系统可用性。

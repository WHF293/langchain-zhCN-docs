# 第十四章 沙箱（Sandbox）

> 来源: [Deep Agents 官方文档](https://docs.langchain.com/deep-agents)

---

沙箱为代理提供隔离的代码执行环境，确保代理生成的代码在安全的容器中运行，不会影响宿主系统。Deep Agents 支持多种沙箱提供者，满足不同场景的需求。

## 沙箱提供者概览

| 提供者 | 特点 | 适用场景 |
|:-------|:-----|:---------|
| **LangSmith** | LangChain 官方托管，开箱即用 | 快速原型开发、测试 |
| **AgentCore** | 企业级，高安全性 | 生产环境、敏感数据处理 |
| **Daytona** | 基于容器，支持自定义镜像 | 需要特定环境的场景 |
| **E2B** | 轻量级，快速启动 | 代码片段执行、快速验证 |
| **Modal** | 支持 GPU，适合计算密集型任务 | AI 推理、数据处理 |
| **Runloop** | 持久化会话，支持长时间运行 | 复杂工作流、多步骤任务 |
| **Vercel** | 边缘计算，低延迟 | Web 应用、API 服务 |

## 提供者配置

### LangSmith 沙箱

```typescript
import { Agent, LangSmithSandbox } from "deepagents";

const sandbox = new LangSmithSandbox({
  // LangSmith API 密钥
  apiKey: process.env.LANGSMITH_API_KEY,
  // 运行时环境
  runtime: "node", // 或 "python"
  // 超时时间（秒）
  timeout: 60,
  // 内存限制（MB）
  memoryLimit: 512,
});

const agent = new Agent({
  model: "gpt-4",
  sandbox,
});
```

### E2B 沙箱

```typescript
import { Agent, E2BSandbox } from "deepagents";

const sandbox = new E2BSandbox({
  // E2B API 密钥
  apiKey: process.env.E2B_API_KEY,
  // 沙箱模板
  template: "node", // 或 "python"、"custom"
  // 生命周期（秒）：无操作后自动销毁
  idleTimeout: 300,
  // 环境变量
  env: {
    NODE_ENV: "development",
  },
});

const agent = new Agent({
  model: "gpt-4",
  sandbox,
});
```

### Modal 沙箱（支持 GPU）

```typescript
import { Agent, ModalSandbox } from "deepagents";

const sandbox = new ModalSandbox({
  // Modal API 密钥
  token: process.env.MODAL_TOKEN,
  // GPU 类型
  gpu: "A10G", // 或 "T4"、"A100"
  // 容器镜像
  image: "python:3.11-slim",
  // 安装依赖
  packages: ["torch", "transformers", "pandas"],
  // 超时时间（秒）
  timeout: 300,
});

const agent = new Agent({
  model: "gpt-4",
  sandbox,
});
```

### Daytona 沙箱

```typescript
import { Agent, DaytonaSandbox } from "deepagents";

const sandbox = new DaytonaSandbox({
  // Daytona API 配置
  apiUrl: "https://api.daytona.io",
  apiKey: process.env.DAYTONA_API_KEY,
  // 自定义镜像
  image: "my-custom-image:latest",
  // 挂载卷
  volumes: [
    { host: "/data", container: "/workspace/data", readonly: false },
  ],
  // 网络配置
  network: "isolated",
});

const agent = new Agent({
  model: "gpt-4",
  sandbox,
});
```

## 沙箱生命周期

沙箱实例有明确的生命周期管理：

```
创建（Create）
    │
    ▼
初始化（Initialize）← 安装依赖、配置环境
    │
    ▼
运行（Running）← 执行代码、文件操作
    │
    ▼
暂停（Paused）← 可选：资源释放但状态保留
    │
    ▼
销毁（Destroy）← 清理资源、删除临时文件
```

### 生命周期管理

```typescript
// 创建沙箱
const sandbox = new E2BSandbox({ apiKey: "..." });

// 初始化（自动或手动）
await sandbox.initialize();

// 执行代码
const result = await sandbox.run("console.log('Hello')");

// 暂停（释放资源但保留状态）
await sandbox.pause();

// 恢复
await sandbox.resume();

// 销毁
await sandbox.destroy();

// 使用 withAgent 自动管理生命周期
await sandbox.withAgent(async (agent) => {
  // agent 在此作用域内可用
  const result = await agent.run("分析这段代码");
  return result;
  // 离开作用域后沙箱自动销毁
});
```

> **注意**：沙箱在空闲超时后会自动销毁，请确保重要数据已持久化到外部存储。

## 集成模式

### 与代理集成

```typescript
import { Agent, E2BSandbox } from "deepagents";

// 创建带沙箱的代理
const agent = new Agent({
  model: "gpt-4",
  sandbox: new E2BSandbox({ apiKey: "..." }),
  // 沙箱配置
  sandboxConfig: {
    // 自动安装依赖
    autoInstall: true,
    // 依赖列表
    dependencies: ["lodash", "axios"],
    // 启动脚本
    startupScript: "console.log('Sandbox ready')",
  },
});

// 代理执行代码时自动使用沙箱
const result = await agent.run(`
  请分析这段 Python 代码的性能：
  def fibonacci(n):
      if n <= 1:
          return n
      return fibonacci(n-1) + fibonacci(n-2)
`);
```

### 与后端集成

```typescript
import { CompositeBackend, FilesystemBackend, SandboxShellBackend } from "deepagents";

const backend = new CompositeBackend({
  backends: {
    // 本地文件系统
    fs: new FilesystemBackend({ root: "/workspace" }),
    // 沙箱 Shell（代码在沙箱中执行）
    shell: new SandboxShellBackend({
      sandbox: new E2BSandbox({ apiKey: "..." }),
      // 同步本地文件到沙箱
      syncFiles: ["/workspace/src/**"],
    }),
  },
});
```

## 文件传输

沙箱支持与本地文件系统之间的文件传输：

```typescript
const sandbox = new E2BSandbox({ apiKey: "..." });

// 上传文件到沙箱
await sandbox.uploadFile({
  localPath: "/local/data.csv",
  remotePath: "/workspace/data.csv",
});

// 批量上传
await sandbox.uploadFiles([
  { localPath: "/local/file1.txt", remotePath: "/workspace/file1.txt" },
  { localPath: "/local/file2.txt", remotePath: "/workspace/file2.txt" },
]);

// 从沙箱下载文件
await sandbox.downloadFile({
  remotePath: "/workspace/output/result.json",
  localPath: "/local/result.json",
});

// 读取沙箱中的文件内容（不下载到本地）
const content = await sandbox.readFile("/workspace/config.yaml");

// 写入内容到沙箱文件
await sandbox.writeFile("/workspace/data.json", JSON.stringify({ key: "value" }));
```

### 自动文件同步

```typescript
const sandbox = new E2BSandbox({
  apiKey: "...",
  // 自动同步配置
  autoSync: {
    // 监听本地目录变化，自动同步到沙箱
    watch: ["/workspace/src"],
    // 同步方向
    direction: "upload", // "upload"、"download"、"bidirectional"
    // 忽略的文件
    ignore: ["*.log", "node_modules/**"],
    // 同步间隔（毫秒）
    interval: 5000,
  },
});
```

## 安全考虑

### 网络隔离

```typescript
const sandbox = new E2BSandbox({
  apiKey: "...",
  network: {
    // 禁止外部网络访问
    outbound: false,
    // 允许的域名白名单
    allowedDomains: ["api.openai.com", "registry.npmjs.org"],
    // 禁止的端口
    blockedPorts: [22, 3306, 5432],
  },
});
```

### 文件系统隔离

```typescript
const sandbox = new E2BSandbox({
  apiKey: "...",
  filesystem: {
    // 只读根文件系统
    readonlyRoot: true,
    // 可写目录（白名单）
    writablePaths: ["/workspace", "/tmp"],
    // 禁止的路径
    blockedPaths: ["/etc", "/var", "/root"],
    // 最大文件大小
    maxFileSize: 10 * 1024 * 1024, // 10MB
    // 最大磁盘使用
    maxDiskUsage: 100 * 1024 * 1024, // 100MB
  },
});
```

### 资源限制

```typescript
const sandbox = new E2BSandbox({
  apiKey: "...",
  resources: {
    // CPU 限制（核心数）
    cpu: 2,
    // 内存限制（MB）
    memory: 1024,
    // 进程数限制
    maxProcesses: 50,
    // 执行超时（秒）
    executionTimeout: 60,
    // 网络带宽限制（MB/s）
    networkBandwidth: 10,
  },
});
```

### 权限控制

```typescript
const sandbox = new E2BSandbox({
  apiKey: "...",
  security: {
    // 禁止特权操作
    privileged: false,
    // 禁止容器逃逸
    allowEscalation: false,
    // 禁止访问宿主设备
    devices: [],
    // Seccomp 配置
    seccomp: "default",
    // AppArmor 配置
    apparmor: "default",
  },
});
```

> **建议**：生产环境中，建议始终启用网络隔离和文件系统白名单，仅允许必要的访问。

## 沙箱监控

```typescript
const sandbox = new E2BSandbox({ apiKey: "..." });

// 获取沙箱状态
const status = await sandbox.getStatus();
console.log(status);
// {
//   state: "running",
//   uptime: 120,
//   cpu: 0.45,
//   memory: 512,
//   disk: 1024,
//   processes: 12,
// }

// 获取资源使用情况
const metrics = await sandbox.getMetrics();
console.log(metrics);

// 设置资源告警
sandbox.onAlert("memory", 80, async (usage) => {
  console.warn(`内存使用率超过 ${usage}%`);
});
```

## 最佳实践

1. **按需创建**：只在需要执行代码时创建沙箱，用完及时销毁
2. **资源限制**：始终设置合理的资源限制，防止资源耗尽
3. **网络隔离**：生产环境禁止开放不必要的网络访问
4. **文件白名单**：使用文件系统白名单而非黑名单
5. **超时设置**：设置合理的执行超时，防止代码无限运行
6. **日志记录**：启用沙箱执行日志，便于问题排查
7. **镜像固定**：使用固定版本的容器镜像，避免环境变化导致的问题

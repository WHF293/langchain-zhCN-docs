# 第二十一章 异步子代理

> 来源: [Deep Agents 官方文档](https://docs.langchain.com/deep-agents)

---

异步子代理（Asynchronous Sub-Agent）是一种在后台执行长时间运行任务的代理模式。与同步子代理不同，异步子代理启动后立即返回，允许主管代理继续处理其他任务。本章介绍异步子代理的工作原理、传输机制、生命周期管理和部署拓扑。

## 概述

异步子代理模式解决了长时间运行任务阻塞主管代理的问题。通过将耗时任务委托给后台子代理，主管代理可以继续响应用户请求，提高系统的整体吞吐量和响应性。

### 工作流程

```typescript
// 异步子代理的工作流程
interface AsyncSubAgentWorkflow {
  // 1. 主管代理接收用户请求
  receiveRequest(request: UserRequest): void;

  // 2. 主管代理判断是否需要异步处理
  shouldDelegateAsync(task: Task): boolean;

  // 3. 启动异步子代理，立即返回任务句柄
  launchSubAgent(task: Task): TaskHandle;

  // 4. 主管代理继续处理其他任务
  continueProcessing(): void;

  // 5. 子代理在后台执行任务
  // （主管代理不阻塞等待）

  // 6. 子代理完成后通知主管代理
  notifyCompletion(result: TaskResult): void;

  // 7. 主管代理整合结果并响应用户
  integrateResult(result: TaskResult): Response;
}
```

| 同步子代理 | 异步子代理 |
|------------|------------|
| 主管代理阻塞等待 | 主管代理立即返回 |
| 适用于短时间任务 | 适用于长时间任务 |
| 实现简单 | 需要状态管理 |
| 结果立即可用 | 结果延迟可用 |
| 资源占用集中 | 资源占用分散 |

## 启动后台任务

异步子代理的启动涉及任务定义、资源分配和状态初始化。

```typescript
// 异步子代理任务定义
interface AsyncSubAgentTask {
  // 任务标识
  taskId: string;
  // 任务类型
  taskType: string;
  // 任务输入
  input: TaskInput;
  // 任务配置
  config: {
    // 优先级（0-10，数字越大优先级越高）
    priority: number;
    // 超时时间（毫秒）
    timeout: number;
    // 重试策略
    retryPolicy: {
      maxRetries: number;
      backoffMs: number;
    };
    // 回调配置
    callback: {
      // 完成回调 URL
      url?: string;
      // 完成回调事件
      event?: string;
    };
  };
  // 元数据
  metadata: {
    parentTaskId: string;
    userId: string;
    createdAt: Date;
  };
}

// 异步子代理启动器
class AsyncSubAgentLauncher {
  private taskQueue: TaskQueue;
  private stateManager: StateManager;
  private eventBus: EventBus;

  // 启动异步子代理
  async launch(task: AsyncSubAgentTask): Promise<TaskHandle> {
    // 1. 验证任务
    await this.validateTask(task);

    // 2. 初始化任务状态
    await this.stateManager.initializeTask(task.taskId, {
      status: "pending",
      input: task.input,
      createdAt: new Date(),
    });

    // 3. 将任务加入队列
    await this.taskQueue.enqueue(task);

    // 4. 发布任务启动事件
    await this.eventBus.publish("task.launched", {
      taskId: task.taskId,
      taskType: task.taskType,
      timestamp: new Date(),
    });

    // 5. 立即返回任务句柄
    return {
      taskId: task.taskId,
      status: "pending",
      statusUrl: `/tasks/${task.taskId}/status`,
      cancelUrl: `/tasks/${task.taskId}/cancel`,
      resultUrl: `/tasks/${task.taskId}/result`,
    };
  }

  // 验证任务
  private async validateTask(task: AsyncSubAgentTask): Promise<void> {
    // 检查任务类型是否支持
    if (!this.isTaskTypeSupported(task.taskType)) {
      throw new Error(`不支持的任务类型: ${task.taskType}`);
    }

    // 检查资源配额
    const quota = await this.checkQuota(task.metadata.userId);
    if (!quota.available) {
      throw new Error("已达到并发任务限制");
    }

    // 检查输入有效性
    await this.validateInput(task.input);
  }
}

// 主管代理使用示例
class SupervisorAgent {
  private launcher: AsyncSubAgentLauncher;

  async handleRequest(request: UserRequest): Promise<Response> {
    const task = this.analyzeTask(request);

    if (this.shouldDelegateAsync(task)) {
      // 启动异步子代理
      const handle = await this.launcher.launch({
        taskId: generateTaskId(),
        taskType: task.type,
        input: task.input,
        config: {
          priority: task.priority,
          timeout: 300000, // 5 分钟
          retryPolicy: {
            maxRetries: 3,
            backoffMs: 1000,
          },
          callback: {
            event: "task.completed",
          },
        },
        metadata: {
          parentTaskId: request.id,
          userId: request.userId,
          createdAt: new Date(),
        },
      });

      // 立即返回给用户，告知任务已启动
      return {
        type: "async_task_started",
        message: "任务已启动，正在后台处理...",
        taskId: handle.taskId,
        statusUrl: handle.statusUrl,
        estimatedTime: this.estimateCompletionTime(task),
      };
    }

    // 同步处理短任务
    return this.processSync(task);
  }

  private shouldDelegateAsync(task: Task): boolean {
    // 判断是否应该异步处理
    return (
      task.estimatedDuration > 30000 || // 超过 30 秒
      task.requiresExternalAPI ||
      task.resourceIntensive
    );
  }
}
```

> **注意**：异步任务启动后，主管代理应立即返回响应，告知用户任务已开始处理，并提供任务状态查询方式。

## 立即返回，主管继续

主管代理在启动异步子代理后，可以继续处理其他请求，提高系统并发能力。

```typescript
// 主管代理的并发处理
class ConcurrentSupervisorAgent {
  private activeTasks: Map<string, TaskHandle> = new Map();
  private taskResults: Map<string, TaskResult> = new Map();

  // 处理用户请求
  async handleRequest(request: UserRequest): Promise<Response> {
    // 检查是否有已完成的任务结果需要返回
    const pendingResult = this.checkPendingResults(request.userId);
    if (pendingResult) {
      return this.formatResult(pendingResult);
    }

    // 分析请求类型
    const requestType = this.classifyRequest(request);

    switch (requestType) {
      case "quick_task":
        // 快速任务：同步处理
        return this.processQuickly(request);

      case "long_running_task":
        // 长时间任务：启动异步子代理
        return this.launchAsyncTask(request);

      case "status_check":
        // 状态查询：返回任务状态
        return this.getTaskStatus(request);

      default:
        return this.processQuickly(request);
    }
  }

  // 启动异步任务
  private async launchAsyncTask(request: UserRequest): Promise<Response> {
    const task = this.createTask(request);

    // 启动子代理
    const handle = await this.launcher.launch(task);

    // 记录活跃任务
    this.activeTasks.set(task.taskId, handle);

    // 注册完成回调
    this.registerCallback(task.taskId, async (result) => {
      this.taskResults.set(task.taskId, result);
      this.activeTasks.delete(task.taskId);

      // 通知用户（如果用户在线）
      await this.notifyUser(request.userId, result);
    });

    // 立即返回
    return {
      type: "task_started",
      taskId: task.taskId,
      message: "任务已启动，我会在完成后通知您。",
      estimatedTime: this.estimateTime(task),
      checkStatusUrl: `/api/tasks/${task.taskId}/status`,
    };
  }

  // 检查待返回的结果
  private checkPendingResults(userId: string): TaskResult | null {
    for (const [taskId, result] of this.taskResults) {
      if (result.userId === userId && !result.delivered) {
        return result;
      }
    }
    return null;
  }

  // 获取任务状态
  private async getTaskStatus(request: UserRequest): Promise<Response> {
    const taskId = request.params.taskId;
    const handle = this.activeTasks.get(taskId);

    if (!handle) {
      const result = this.taskResults.get(taskId);
      if (result) {
        return this.formatResult(result);
      }
      throw new Error("任务不存在");
    }

    const status = await this.stateManager.getTaskStatus(taskId);

    return {
      type: "task_status",
      taskId,
      status: status.status,
      progress: status.progress,
      estimatedRemaining: status.estimatedRemaining,
    };
  }
}

// 用户端处理示例
class UserClient {
  private supervisor: ConcurrentSupervisorAgent;

  async interact(userInput: string): Promise<void> {
    const response = await this.supervisor.handleRequest({
      userId: this.userId,
      input: userInput,
    });

    switch (response.type) {
      case "task_started":
        console.log(`任务已启动: ${response.taskId}`);
        console.log(`预计完成时间: ${response.estimatedTime}秒`);
        console.log(`状态查询: ${response.checkStatusUrl}`);

        // 可选：轮询状态
        this.pollStatus(response.taskId);
        break;

      case "task_completed":
        console.log(`任务完成: ${response.result}`);
        break;

      case "quick_response":
        console.log(`响应: ${response.message}`);
        break;
    }
  }

  // 轮询任务状态
  private async pollStatus(taskId: string): Promise<void> {
    const interval = setInterval(async () => {
      const status = await this.getTaskStatus(taskId);

      if (status.status === "completed") {
        clearInterval(interval);
        console.log(`任务完成: ${status.result}`);
      } else if (status.status === "failed") {
        clearInterval(interval);
        console.error(`任务失败: ${status.error}`);
      } else {
        console.log(`进度: ${status.progress}%`);
      }
    }, 5000); // 每 5 秒轮询一次
  }
}
```

> **提示**：为了避免轮询带来的资源消耗，建议使用 WebSocket 或 Server-Sent Events 实现实时通知。

## ASGI/HTTP 传输

异步子代理可以通过 ASGI（Asynchronous Server Gateway Interface）或 HTTP 协议进行通信。

```typescript
// ASGI 传输层实现
class ASGITransport {
  private app: ASGIApplication;
  private connections: Map<string, WebSocket> = new Map();

  // 创建 ASGI 应用
  createApp(): ASGIApplication {
    return async (scope, receive, send) => {
      if (scope.type === "http") {
        await this.handleHTTP(scope, receive, send);
      } else if (scope.type === "websocket") {
        await this.handleWebSocket(scope, receive, send);
      }
    };
  }

  // 处理 HTTP 请求
  private async handleHTTP(scope, receive, send): Promise<void> {
    const request = new Request(scope, receive);

    // 路由处理
    const response = await this.route(request);

    // 发送响应
    await send({
      type: "http.response.start",
      status: response.status,
      headers: response.headers,
    });
    await send({
      type: "http.response.body",
      body: response.body,
    });
  }

  // 处理 WebSocket 连接
  private async handleWebSocket(scope, receive, send): Promise<void> {
    await send({ type: "websocket.accept" });

    const connectionId = generateConnectionId();
    const ws = new WebSocket(scope, receive, send);
    this.connections.set(connectionId, ws);

    // 监听消息
    ws.on("message", async (message) => {
      await this.handleMessage(connectionId, message);
    });

    // 监听关闭
    ws.on("close", () => {
      this.connections.delete(connectionId);
    });
  }

  // 发送任务更新
  async sendTaskUpdate(taskId: string, update: TaskUpdate): Promise<void> {
    // 查找相关连接
    const connections = await this.getTaskConnections(taskId);

    for (const connectionId of connections) {
      const ws = this.connections.get(connectionId);
      if (ws) {
        await ws.send(JSON.stringify({
          type: "task_update",
          taskId,
          update,
          timestamp: new Date(),
        }));
      }
    }
  }
}

// HTTP 传输层实现
class HTTPTransport {
  private baseUrl: string;
  private apiKey: string;

  // 启动异步任务
  async launchTask(task: AsyncSubAgentTask): Promise<TaskHandle> {
    const response = await fetch(`${this.baseUrl}/tasks`, {
      method: "POST",
      headers: {
        "Content-Type": "application/json",
        "Authorization": `Bearer ${this.apiKey}`,
      },
      body: JSON.stringify(task),
    });

    if (!response.ok) {
      throw new Error(`任务启动失败: ${response.statusText}`);
    }

    return response.json();
  }

  // 查询任务状态
  async getTaskStatus(taskId: string): Promise<TaskStatus> {
    const response = await fetch(`${this.baseUrl}/tasks/${taskId}/status`, {
      headers: {
        "Authorization": `Bearer ${this.apiKey}`,
      },
    });

    if (!response.ok) {
      throw new Error(`状态查询失败: ${response.statusText}`);
    }

    return response.json();
  }

  // 获取任务结果
  async getTaskResult(taskId: string): Promise<TaskResult> {
    const response = await fetch(`${this.baseUrl}/tasks/${taskId}/result`, {
      headers: {
        "Authorization": `Bearer ${this.apiKey}`,
      },
    });

    if (!response.ok) {
      throw new Error(`结果获取失败: ${response.statusText}`);
    }

    return response.json();
  }

  // 取消任务
  async cancelTask(taskId: string): Promise<void> {
    await fetch(`${this.baseUrl}/tasks/${taskId}/cancel`, {
      method: "POST",
      headers: {
        "Authorization": `Bearer ${this.apiKey}`,
      },
    });
  }
}
```

| 传输方式 | 实时性 | 复杂度 | 适用场景 |
|----------|--------|--------|----------|
| HTTP 轮询 | 低 | 低 | 简单任务、低频查询 |
| HTTP 长轮询 | 中 | 中 | 中等实时性要求 |
| WebSocket | 高 | 高 | 实时通知、双向通信 |
| Server-Sent Events | 高 | 中 | 单向实时推送 |

## 生命周期

异步子代理的生命周期包括创建、运行、暂停、恢复和销毁等阶段。

```typescript
// 子代理生命周期状态
enum SubAgentState {
  CREATED = "created",
  PENDING = "pending",
  RUNNING = "running",
  PAUSED = "paused",
  COMPLETED = "completed",
  FAILED = "failed",
  CANCELLED = "cancelled",
  TIMEOUT = "timeout",
}

// 生命周期管理器
class SubAgentLifecycleManager {
  private stateManager: StateManager;
  private eventBus: EventBus;

  // 创建子代理
  async create(task: AsyncSubAgentTask): Promise<SubAgentInstance> {
    const instance: SubAgentInstance = {
      id: generateInstanceId(),
      taskId: task.taskId,
      state: SubAgentState.CREATED,
      createdAt: new Date(),
      config: task.config,
    };

    // 持久化初始状态
    await this.stateManager.saveInstance(instance);

    // 发布创建事件
    await this.eventBus.publish("subagent.created", {
      instanceId: instance.id,
      taskId: task.taskId,
    });

    return instance;
  }

  // 启动子代理
  async start(instanceId: string): Promise<void> {
    const instance = await this.stateManager.getInstance(instanceId);

    // 状态检查
    if (instance.state !== SubAgentState.CREATED &&
        instance.state !== SubAgentState.PAUSED) {
      throw new Error(`无法启动: 当前状态为 ${instance.state}`);
    }

    // 更新状态
    instance.state = SubAgentState.RUNNING;
    instance.startedAt = new Date();
    await this.stateManager.saveInstance(instance);

    // 发布启动事件
    await this.eventBus.publish("subagent.started", {
      instanceId,
      timestamp: new Date(),
    });

    // 设置超时定时器
    if (instance.config.timeout > 0) {
      setTimeout(() => {
        this.handleTimeout(instanceId);
      }, instance.config.timeout);
    }
  }

  // 暂停子代理
  async pause(instanceId: string): Promise<void> {
    const instance = await this.stateManager.getInstance(instanceId);

    if (instance.state !== SubAgentState.RUNNING) {
      throw new Error(`无法暂停: 当前状态为 ${instance.state}`);
    }

    // 保存当前状态以便恢复
    instance.state = SubAgentState.PAUSED;
    instance.pausedAt = new Date();
    instance.pausedState = await this.captureState(instanceId);
    await this.stateManager.saveInstance(instance);

    // 释放资源
    await this.releaseResources(instanceId);

    await this.eventBus.publish("subagent.paused", { instanceId });
  }

  // 恢复子代理
  async resume(instanceId: string): Promise<void> {
    const instance = await this.stateManager.getInstance(instanceId);

    if (instance.state !== SubAgentState.PAUSED) {
      throw new Error(`无法恢复: 当前状态为 ${instance.state}`);
    }

    // 恢复资源
    await this.allocateResources(instanceId);

    // 恢复状态
    await this.restoreState(instanceId, instance.pausedState);

    instance.state = SubAgentState.RUNNING;
    instance.resumedAt = new Date();
    await this.stateManager.saveInstance(instance);

    await this.eventBus.publish("subagent.resumed", { instanceId });
  }

  // 完成子代理
  async complete(instanceId: string, result: TaskResult): Promise<void> {
    const instance = await this.stateManager.getInstance(instanceId);

    instance.state = SubAgentState.COMPLETED;
    instance.completedAt = new Date();
    instance.result = result;
    await this.stateManager.saveInstance(instance);

    // 清理资源
    await this.cleanupResources(instanceId);

    // 发布完成事件
    await this.eventBus.publish("subagent.completed", {
      instanceId,
      result,
      duration: instance.completedAt.getTime() - instance.startedAt.getTime(),
    });
  }

  // 处理失败
  async fail(instanceId: string, error: Error): Promise<void> {
    const instance = await this.stateManager.getInstance(instanceId);

    instance.state = SubAgentState.FAILED;
    instance.failedAt = new Date();
    instance.error = {
      message: error.message,
      stack: error.stack,
    };
    await this.stateManager.saveInstance(instance);

    // 清理资源
    await this.cleanupResources(instanceId);

    // 检查是否需要重试
    if (this.shouldRetry(instance)) {
      await this.retry(instanceId);
    } else {
      await this.eventBus.publish("subagent.failed", {
        instanceId,
        error: instance.error,
      });
    }
  }

  // 处理超时
  private async handleTimeout(instanceId: string): Promise<void> {
    const instance = await this.stateManager.getInstance(instanceId);

    if (instance.state === SubAgentState.RUNNING) {
      instance.state = SubAgentState.TIMEOUT;
      instance.timeoutAt = new Date();
      await this.stateManager.saveInstance(instance);

      // 终止执行
      await this.terminate(instanceId);

      await this.eventBus.publish("subagent.timeout", { instanceId });
    }
  }

  // 重试逻辑
  private async retry(instanceId: string): Promise<void> {
    const instance = await this.stateManager.getInstance(instanceId);
    const retryCount = (instance.retryCount || 0) + 1;

    if (retryCount > instance.config.retryPolicy.maxRetries) {
      await this.eventBus.publish("subagent.retry_exhausted", {
        instanceId,
        retryCount,
      });
      return;
    }

    // 指数退避
    const backoff = instance.config.retryPolicy.backoffMs * Math.pow(2, retryCount - 1);

    setTimeout(async () => {
      instance.retryCount = retryCount;
      instance.state = SubAgentState.PENDING;
      await this.stateManager.saveInstance(instance);

      // 重新启动
      await this.start(instanceId);

      await this.eventBus.publish("subagent.retried", {
        instanceId,
        retryCount,
      });
    }, backoff);
  }
}
```

> **建议**：在生产环境中，建议实现完善的生命周期事件日志，以便追踪和调试子代理的执行过程。

## 状态管理

异步子代理的状态管理是确保任务可靠执行的关键。

```typescript
// 状态存储接口
interface StateStore {
  // 保存状态
  save(key: string, state: any, options?: SaveOptions): Promise<void>;
  // 获取状态
  get(key: string): Promise<any>;
  // 删除状态
  delete(key: string): Promise<void>;
  // 列出状态
  list(prefix: string, options?: ListOptions): Promise<string[]>;
}

// 状态管理器实现
class SubAgentStateManager {
  private store: StateStore;
  private checkpointInterval: number;

  constructor(store: StateStore, checkpointInterval: number = 30000) {
    this.store = store;
    this.checkpointInterval = checkpointInterval;
  }

  // 初始化任务状态
  async initializeTask(taskId: string, initialState: TaskState): Promise<void> {
    await this.store.save(`task:${taskId}:state`, {
      ...initialState,
      version: 1,
      createdAt: new Date(),
    });

    // 创建初始检查点
    await this.createCheckpoint(taskId, initialState);
  }

  // 更新任务状态
  async updateTaskState(
    taskId: string,
    updates: Partial<TaskState>
  ): Promise<TaskState> {
    const currentState = await this.getTaskState(taskId);

    const newState: TaskState = {
      ...currentState,
      ...updates,
      version: currentState.version + 1,
      updatedAt: new Date(),
    };

    await this.store.save(`task:${taskId}:state`, newState);

    return newState;
  }

  // 获取任务状态
  async getTaskState(taskId: string): Promise<TaskState> {
    const state = await this.store.get(`task:${taskId}:state`);
    if (!state) {
      throw new Error(`任务 ${taskId} 状态不存在`);
    }
    return state;
  }

  // 创建检查点
  async createCheckpoint(taskId: string, state: any): Promise<Checkpoint> {
    const checkpoint: Checkpoint = {
      id: generateCheckpointId(),
      taskId,
      timestamp: new Date(),
      state: JSON.parse(JSON.stringify(state)), // 深拷贝
    };

    await this.store.save(
      `task:${taskId}:checkpoint:${checkpoint.id}`,
      checkpoint
    );

    // 更新检查点索引
    const checkpoints = await this.getCheckpointList(taskId);
    checkpoints.push(checkpoint.id);
    await this.store.save(`task:${taskId}:checkpoints`, checkpoints);

    // 清理旧检查点
    await this.cleanupOldCheckpoints(taskId);

    return checkpoint;
  }

  // 从检查点恢复
  async restoreFromCheckpoint(
    taskId: string,
    checkpointId?: string
  ): Promise<TaskState> {
    let checkpoint: Checkpoint;

    if (checkpointId) {
      // 恢复到指定检查点
      checkpoint = await this.store.get(
        `task:${taskId}:checkpoint:${checkpointId}`
      );
    } else {
      // 恢复到最新检查点
      const checkpoints = await this.getCheckpointList(taskId);
      if (checkpoints.length === 0) {
        throw new Error("没有可用的检查点");
      }
      const latestCheckpointId = checkpoints[checkpoints.length - 1];
      checkpoint = await this.store.get(
        `task:${taskId}:checkpoint:${latestCheckpointId}`
      );
    }

    // 恢复状态
    await this.store.save(`task:${taskId}:state`, checkpoint.state);

    return checkpoint.state;
  }

  // 保存执行上下文
  async saveExecutionContext(
    taskId: string,
    context: ExecutionContext
  ): Promise<void> {
    await this.store.save(`task:${taskId}:context`, {
      ...context,
      savedAt: new Date(),
    });
  }

  // 获取执行上下文
  async getExecutionContext(taskId: string): Promise<ExecutionContext> {
    return this.store.get(`task:${taskId}:context`);
  }

  // 清理旧检查点
  private async cleanupOldCheckpoints(taskId: string): Promise<void> {
    const checkpoints = await this.getCheckpointList(taskId);
    const maxCheckpoints = 10; // 保留最近 10 个检查点

    if (checkpoints.length > maxCheckpoints) {
      const toDelete = checkpoints.slice(0, checkpoints.length - maxCheckpoints);

      for (const checkpointId of toDelete) {
        await this.store.delete(`task:${taskId}:checkpoint:${checkpointId}`);
      }

      // 更新索引
      await this.store.save(
        `task:${taskId}:checkpoints`,
        checkpoints.slice(checkpoints.length - maxCheckpoints)
      );
    }
  }

  // 获取检查点列表
  private async getCheckpointList(taskId: string): Promise<string[]> {
    return (await this.store.get(`task:${taskId}:checkpoints`)) || [];
  }
}
```

```typescript
// 分布式任务调度器
class DistributedTaskScheduler {
  private topology: DeploymentTopology;
  private messageQueue: MessageQueue;
  private stateStore: StateStore;
  private loadBalancer: LoadBalancer;

  // 调度任务
  async schedule(task: AsyncSubAgentTask): Promise<string> {
    // 1. 选择目标工作节点池
    const pool = this.selectPool(task);

    // 2. 选择具体工作节点
    const worker = await this.selectWorker(pool, task);

    // 3. 将任务分配到消息队列
    const queueName = `tasks:${pool.name}`;
    await this.messageQueue.publish(queueName, {
      ...task,
      assignedWorker: worker.id,
      scheduledAt: new Date(),
    });

    // 4. 更新任务状态
    await this.stateStore.save(`task:${task.taskId}:assignment`, {
      pool: pool.name,
      worker: worker.id,
      scheduledAt: new Date(),
    });

    return worker.id;
  }

  // 选择工作节点池
  private selectPool(task: AsyncSubAgentTask): WorkerPool {
    // 根据任务类型选择池
    const eligiblePools = this.topology.workers.pools.filter(
      (pool) =>
        pool.taskTypes.includes(task.taskType) ||
        pool.taskTypes.includes("*")
    );

    if (eligiblePools.length === 0) {
      throw new Error(`没有支持任务类型 ${task.taskType} 的工作节点池`);
    }

    // 根据负载选择最优池
    return this.selectLeastLoadedPool(eligiblePools);
  }

  // 选择工作节点
  private async selectWorker(
    pool: WorkerPool,
    task: AsyncSubAgentTask
  ): Promise<WorkerInfo> {
    const workers = await this.getPoolWorkers(pool.name);

    // 过滤可用节点
    const availableWorkers = workers.filter(
      (w) => w.status === "healthy" && w.currentTasks < pool.resources.maxConcurrentTasks
    );

    if (availableWorkers.length === 0) {
      // 触发自动扩缩容
      await this.scaleUp(pool);
      // 等待新节点就绪
      await this.waitForNewWorker(pool);
      return this.selectWorker(pool, task);
    }

    // 使用负载均衡器选择节点
    return this.loadBalancer.select(availableWorkers, task);
  }

  // 获取任务结果
  async getResult(taskId: string): Promise<TaskResult> {
    // 从结果队列获取
    const result = await this.messageQueue.consume(`results:${taskId}`);

    if (!result) {
      // 检查任务状态
      const assignment = await this.stateStore.get(`task:${taskId}:assignment`);
      if (!assignment) {
        throw new Error("任务不存在");
      }

      // 查询工作节点状态
      const workerStatus = await this.getWorkerStatus(assignment.worker);
      return {
        status: workerStatus.taskStatus,
        progress: workerStatus.progress,
      };
    }

    return result;
  }
}
```

异步子代理在生产环境中可能遇到各种问题，本节提供常见问题的诊断和解决方案。

```typescript
// 故障诊断器
class SubAgentDiagnostics {
  private stateManager: SubAgentStateManager;
  private logger: Logger;
  private metrics: MetricsCollector;

  // 诊断任务问题
  async diagnoseTask(taskId: string): Promise<DiagnosisReport> {
    const report: DiagnosisReport = {
      taskId,
      timestamp: new Date(),
      issues: [],
      recommendations: [],
    };

    // 1. 检查任务状态
    const state = await this.stateManager.getTaskState(taskId).catch(() => null);
    if (!state) {
      report.issues.push({
        severity: "critical",
        type: "state_missing",
        message: "任务状态不存在",
      });
      return report;
    }

    // 2. 检查任务是否卡住
    if (this.isTaskStuck(state)) {
      report.issues.push({
        severity: "high",
        type: "task_stuck",
        message: `任务已卡住 ${this.getStuckDuration(state)} 毫秒`,
        details: {
          lastUpdate: state.updatedAt,
          currentStep: state.currentStep,
        },
      });
      report.recommendations.push("考虑终止并重试任务");
    }

    // 3. 检查工作节点健康状态
    const workerHealth = await this.checkWorkerHealth(taskId);
    if (!workerHealth.healthy) {
      report.issues.push({
        severity: "high",
        type: "worker_unhealthy",
        message: `工作节点不健康: ${workerHealth.reason}`,
      });
      report.recommendations.push("检查工作节点日志和资源使用情况");
    }

    // 4. 检查消息队列状态
    const queueHealth = await this.checkQueueHealth(taskId);
    if (!queueHealth.healthy) {
      report.issues.push({
        severity: "medium",
        type: "queue_issue",
        message: `消息队列问题: ${queueHealth.reason}`,
      });
      report.recommendations.push("检查消息队列连接和配置");
    }

    // 5. 检查资源使用情况
    const resourceUsage = await this.checkResourceUsage(taskId);
    if (resourceUsage.memoryUsage > 0.9) {
      report.issues.push({
        severity: "high",
        type: "memory_exhaustion",
        message: `内存使用率过高: ${(resourceUsage.memoryUsage * 100).toFixed(1)}%`,
      });
      report.recommendations.push("增加内存限制或优化任务内存使用");
    }

    // 6. 检查错误日志
    const errors = await this.getRecentErrors(taskId);
    if (errors.length > 0) {
      report.issues.push({
        severity: "high",
        type: "errors_detected",
        message: `检测到 ${errors.length} 个错误`,
        details: { errors: errors.slice(0, 5) },
      });
      report.recommendations.push("检查错误日志并修复根本原因");
    }

    return report;
  }

  // 检查任务是否卡住
  private isTaskStuck(state: TaskState): boolean {
    const stuckThreshold = 5 * 60 * 1000; // 5 分钟
    const timeSinceUpdate = Date.now() - new Date(state.updatedAt).getTime();
    return (
      state.status === SubAgentState.RUNNING &&
      timeSinceUpdate > stuckThreshold
    );
  }

  // 生成诊断报告
  async generateFullReport(): Promise<SystemReport> {
    const tasks = await this.stateManager.listTasks();
    const taskReports = await Promise.all(
      tasks.map((task) => this.diagnoseTask(task.id))
    );

    const stuckTasks = taskReports.filter((r) =>
      r.issues.some((i) => i.type === "task_stuck")
    );

    const failedTasks = taskReports.filter((r) =>
      r.issues.some((i) => i.severity === "critical")
    );

    return {
      timestamp: new Date(),
      totalTasks: tasks.length,
      stuckTasks: stuckTasks.length,
      failedTasks: failedTasks.length,
      reports: taskReports,
      summary: this.generateSummary(taskReports),
    };
  }
}
```

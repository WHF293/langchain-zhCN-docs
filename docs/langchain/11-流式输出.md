# 第11章 流式输出

> 来源: [LangChain 官方文档](https://python.langchain.com/)

---

## 概述

流式输出 (Streaming) 是指逐块接收模型输出，而不是等待完整响应。这可以显著改善用户体验，因为用户可以立即看到部分结果，而不需要等待整个响应生成完毕。在构建交互式应用时，流式输出是必不可少的功能。

## 流式模式

### 基本流式调用

```typescript
import { ChatOpenAI } from "@langchain/openai";

const model = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0,
});

// 使用 stream 方法进行流式调用
const stream = await model.stream("写一首关于春天的诗");

// 逐块接收输出
for await (const chunk of stream) {
  process.stdout.write(chunk.content as string);
}
console.log(); // 换行
```

### 使用 invoke 的流式回调

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { AIMessageChunk } from "@langchain/core/messages";

const model = new ChatOpenAI({

```

### 流式处理链

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatPromptTemplate } from "@langchain/core/prompts";
import { StringOutputParser } from "@langchain/core/output_parsers";

const model = new ChatOpenAI({

// 创建链

const stream = await chain.stream({ concept: "机器学习" });

  process.stdout.write(chunk);
}
```

## 流式事件

### 使用 EventStream

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatPromptTemplate } from "@langchain/core/prompts";

const model = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0,
});

const prompt = ChatPromptTemplate.fromTemplate("告诉我关于{topic}的有趣事实");
const chain = prompt.pipe(model);

// 使用 streamEvents 获取详细事件
const eventStream = await chain.streamEvents(
  { topic: "太空" },
  { version: "v2" }
);

for await (const event of eventStream) {
  // 事件类型
  const eventType = event.event;

  switch (eventType) {
    case "on_chat_model_start":
      console.log("模型开始生成...");
      break;

    case "on_chat_model_stream":
      // 流式 token
      process.stdout.write(event.data?.chunk?.content as string || "");
      break;

    case "on_chat_model_end":
      console.log("\n模型生成完成");
      break;

    case "on_llm_start":
      console.log("LLM 开始处理");
      break;

    case "on_llm_end":
      console.log("LLM 处理完成");
      break;
  }
}
```

### 事件类型详解

| 事件类型 | 说明 | 触发时机 |
|---------|------|---------|
| `on_llm_start` | LLM 开始处理 | 调用开始时 |
| `on_llm_new_token` | 新 token 生成 | 每个 token 生成时 |
| `on_llm_end` | LLM 处理完成 | 调用结束时 |
| `on_chat_model_start` | 聊天模型开始 | 聊天模型调用开始 |
| `on_chat_model_stream` | 聊天模型流式输出 | 每个 chunk 生成时 |
| `on_chat_model_end` | 聊天模型完成 | 聊天模型调用结束 |
| `on_chain_start` | 链开始执行 | 链调用开始 |
| `on_chain_end` | 链执行完成 | 链调用结束 |
| `on_tool_start` | 工具开始执行 | 工具调用开始 |
| `on_tool_end` | 工具执行完成 | 工具调用结束 |

### 过滤事件

```typescript
import { ChatOpenAI } from "@langchain/openai";

const model = new ChatOpenAI({ modelName: "gpt-4" });

const eventStream = await model.streamEvents(

      timestamp: new Date().toISOString(),
```

## 流式工具调用

### 流式处理工具调用

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { tool } from "@langchain/core/tools";
import { z } from "zod";

// 定义工具
const weatherTool = tool(
  async ({ city }) => {
    return `${city} 今天晴朗，温度 25°C`;
  },
  {
    name: "get_weather",
    description: "获取天气信息",
    schema: z.object({
      city: z.string().describe("城市名称"),
    }),
  }
);

const model = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0,
}).bindTools([weatherTool]);

// 流式处理工具调用
const stream = await model.stream("北京和上海今天天气怎么样？");

let toolCalls: any[] = [];
let currentContent = "";

for await (const chunk of stream) {
  // 处理文本内容
  if (chunk.content) {
    currentContent += chunk.content;
    process.stdout.write(chunk.content as string);
  }

  // 处理工具调用
  if (chunk.tool_calls && chunk.tool_calls.length > 0) {
    for (const toolCall of chunk.tool_calls) {
      // 累积工具调用参数
      const existingCall = toolCalls.find((tc) => tc.id === toolCall.id);
      if (existingCall) {
        // 合并参数
        existingCall.args = { ...existingCall.args, ...toolCall.args };
      } else {
        toolCalls.push({ ...toolCall });
      }
    }
  }
}

console.log("\n\n检测到的工具调用:", toolCalls);

// 执行工具调用
for (const toolCall of toolCalls) {
  const result = await weatherTool.invoke(toolCall.args);
  console.log(`工具 ${toolCall.name} 结果:`, result);
}
```

### 累积工具调用参数

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { AIMessageChunk } from "@langchain/core/messages";

interface ToolCallAccumulator {
  id: string;
  name: string;
  args: string; // JSON 字符串形式
}

function accumulateToolCalls(chunks: AIMessageChunk[]): ToolCallAccumulator[] {
  const toolCallMap = new Map<string, ToolCallAccumulator>();

  for (const chunk of chunks) {
    if (!chunk.tool_calls) continue;

    for (const toolCall of chunk.tool_calls) {
      const existing = toolCallMap.get(toolCall.id!);

      if (existing) {
        // 累积参数字符串
        existing.args += typeof toolCall.args === "string"
          ? toolCall.args
          : JSON.stringify(toolCall.args);
      } else {
        toolCallMap.set(toolCall.id!, {
          id: toolCall.id!,
          name: toolCall.name,
          args: typeof toolCall.args === "string"
            ? toolCall.args
            : JSON.stringify(toolCall.args),
        });
      }
    }
  }

  return Array.from(toolCallMap.values());
}

// 使用示例
const model = new ChatOpenAI({ modelName: "gpt-4" }).bindTools([weatherTool]);
const stream = await model.stream("北京天气怎么样？");

const chunks: AIMessageChunk[] = [];
for await (const chunk of stream) {
  chunks.push(chunk);
}

const accumulatedCalls = accumulateToolCalls(chunks);
console.log("累积的工具调用:", accumulatedCalls);

// 解析参数
for (const call of accumulatedCalls) {
  try {
    call.args = JSON.parse(call.args);
  } catch (e) {
    console.warn("参数解析失败:", call.args);
  }
}
```

### 词级别的流式处理

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { AIMessageChunk } from "@langchain/core/messages";

class WordAccumulator {
  private buffer = "";

  addToken(token: string): string | null {
    this.buffer += token;

    // 检查是否形成完整单词
    if (this.buffer.includes(" ") || this.buffer.includes("\n")) {
      const words = this.buffer.split(/(\s+)/);
      const completeWords = words.slice(0, -1).join("");
      this.buffer = words[words.length - 1] || "";

      if (completeWords) {
        return completeWords;
      }
    }

    return null;
  }

  flush(): string {
    const remaining = this.buffer;
    this.buffer = "";
    return remaining;
  }
}

// 使用示例
const model = new ChatOpenAI({ modelName: "gpt-4", temperature: 0 });
const stream = await model.stream("解释 JavaScript 的闭包概念");

const accumulator = new WordAccumulator();

for await (const chunk of stream) {
  const token = chunk.content as string;
  if (token) {
    const word = accumulator.addToken(token);
    if (word) {
      // 输出完整单词
      process.stdout.write(word);
    }
  }
}

// 输出剩余内容
const remaining = accumulator.flush();
if (remaining) {
  process.stdout.write(remaining);
}
```

### 流式输出与前端集成

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatPromptTemplate } from "@langchain/core/prompts";

// 模拟 Server-Sent Events (SSE) 接口
async function* generateSSE(prompt: string) {
  const model = new ChatOpenAI({
    modelName: "gpt-4",
    temperature: 0,
  });

  const stream = await model.stream(prompt);

  let index = 0;
  for await (const chunk of stream) {
    const content = chunk.content as string;
    if (content) {
      // SSE 格式
      yield `data: ${JSON.stringify({
        type: "token",
        content,
        index: index++,
      })}\n\n`;
    }
  }

  // 结束信号
  yield `data: ${JSON.stringify({ type: "done" })}\n\n`;
}

// HTTP 服务器示例
import http from "http";

const server = http.createServer(async (req, res) => {
  if (req.url === "/stream" && req.method === "POST") {
    // 设置 SSE 头
    res.writeHead(200, {
      "Content-Type": "text/event-stream",
      "Cache-Control": "no-cache",
      Connection: "keep-alive",
    });

    // 获取请求体
    let body = "";
    req.on("data", (chunk) => (body += chunk));
    req.on("end", async () => {
      const { prompt } = JSON.parse(body);

      // 流式发送响应
      for await (const sse of generateSSE(prompt)) {
        res.write(sse);
      }

      res.end();
    });
  }
});

server.listen(3000, () => {
  console.log("流式服务器运行在 http://localhost:3000");
});
```

### 流式输出性能优化

```typescript
import { ChatOpenAI } from "@langchain/openai";

// 批量处理优化
class BatchedStreamWriter {
  private buffer: string[] = [];
  private flushInterval: number;
  private onFlush: (batch: string[]) => void;
  private timer: NodeJS.Timeout | null = null;

  constructor(flushInterval: number, onFlush: (batch: string[]) => void) {
    this.flushInterval = flushInterval;
    this.onFlush = onFlush;
  }

  write(token: string) {
    this.buffer.push(token);

    // 如果缓冲区足够大，立即刷新
    if (this.buffer.length >= 10) {
      this.flush();
    } else if (!this.timer) {
      // 设置定时刷新
      this.timer = setTimeout(() => this.flush(), this.flushInterval);
    }
  }

  private flush() {
    if (this.buffer.length > 0) {
      this.onFlush([...this.buffer]);
      this.buffer = [];
    }
    if (this.timer) {
      clearTimeout(this.timer);
      this.timer = null;
    }
  }
}

// 使用示例
const model = new ChatOpenAI({ modelName: "gpt-4", temperature: 0 });
const stream = await model.stream("写一篇长文章");

const writer = new BatchedStreamWriter(100, (batch) => {
  // 每 100ms 或每 10 个 token 刷新一次
  console.log("批量输出:", batch.join(""));
});

for await (const chunk of stream) {
  writer.write(chunk.content as string);
}
```

## 最佳实践

1. **及时显示输出**：使用流式输出提升用户体验
2. **错误处理**：流式过程中可能出现错误，需要适当处理
3. **取消机制**：实现取消功能，允许用户中断流式输出
4. **缓冲策略**：根据应用场景选择合适的缓冲策略
5. **资源清理**：确保流结束后正确清理资源

> **提示**：对于需要实时反馈的应用（如聊天机器人），强烈建议使用流式输出。

> **注意**：流式输出可能会增加网络请求次数，在网络不稳定的情况下需要考虑重连机制。

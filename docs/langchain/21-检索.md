## 概述

检索（Retrieval）是 RAG 系统的核心组件，负责从知识库中找到与用户查询最相关的文档。本章将深入介绍各种检索策略、过滤、排序和重排技术。

## 检索策略

### 相似度检索

相似度检索是最基本的检索方式，基于向量相似度找到最相关的文档。

```typescript
import { OpenAIEmbeddings } from "@langchain/openai";
import { MemoryVectorStore } from "langchain/vectorstores/memory";

// 创建向量存储
const embeddings = new OpenAIEmbeddings();
const vectorStore = await MemoryVectorStore.fromDocuments(documents, embeddings);

// 相似度检索
const query = "机器学习的基本概念";
const results = await vectorStore.similaritySearch(query, 5);

// 带分数的相似度检索
const resultsWithScore = await vectorStore.similaritySearchWithScore(query, 5);

resultsWithScore.forEach(([doc, score]) => {
  console.log(`相似度: ${score.toFixed(4)}`);
  console.log(`内容: ${doc.pageContent}\n`);
});
```

### MMR 检索

MMR（Maximal Marginal Relevance，最大边际相关性）在保证相关性的同时，增加结果的多样性。

```typescript
import { OpenAIEmbeddings } from "@langchain/openai";
import { MemoryVectorStore } from "langchain/vectorstores/memory";

// 创建向量存储
const embeddings = new OpenAIEmbeddings();
const vectorStore = await MemoryVectorStore.fromDocuments(documents, embeddings);

// MMR 检索
const query = "人工智能的应用";
const mmrResults = await vectorStore.maxMarginalRelevanceSearch(query, {
  k: 5,                // 返回结果数量
  fetchK: 20,          // 候选结果数量（必须 >= k）
  lambda: 0.5          // 多样性参数（0 = 最大多样性，1 = 最大相关性）
});

console.log(`MMR 检索找到 ${mmrResults.length} 个多样化结果`);
```

### 阈值检索

阈值检索只返回相似度高于指定阈值的文档，过滤掉低质量结果。

```typescript
import { OpenAIEmbeddings } from "@langchain/openai";
import { MemoryVectorStore } from "langchain/vectorstores/memory";

// 创建向量存储
const embeddings = new OpenAIEmbeddings();
const vectorStore = await MemoryVectorStore.fromDocuments(documents, embeddings);

// 阈值检索
const query = "深度学习框架比较";
const threshold = 0.7;  // 相似度阈值

const allResults = await vectorStore.similaritySearchWithScore(query, 20);

// 过滤低于阈值的结果
const filteredResults = allResults.filter(([_, score]) => score >= threshold);

console.log(`找到 ${filteredResults.length} 个高相关性结果`);

filteredResults.forEach(([doc, score]) => {
  console.log(`相似度: ${score.toFixed(4)}`);
  console.log(`内容: ${doc.pageContent}\n`);
});
```

### 多向量检索

多向量检索使用多种表示方式（如摘要、问题等）来提高检索效果。

```typescript
import { OpenAIEmbeddings } from "@langchain/openai";
import { MemoryVectorStore } from "langchain/vectorstores/memory";
import { ChatOpenAI } from "@langchain/openai";
import { Document } from "@langchain/core/documents";

// 为文档生成多种表示
async function createMultiRepresentations(docs: Document[]): Promise<Document[]> {
  const model = new ChatOpenAI({ modelName: "gpt-4" });
  const multiDocs: Document[] = [];

  for (const doc of docs) {
    // 原始文档
    multiDocs.push(doc);

    // 生成摘要
    const summary = await model.invoke(
      `请用一句话总结以下内容:\n${doc.pageContent}`
    );
    multiDocs.push(new Document({
      pageContent: summary as string,
      metadata: { ...doc.metadata, type: "summary" }
    }));

    // 生成可能的问题
    const questions = await model.invoke(
      `根据以下内容，生成3个可能的问题:\n${doc.pageContent}`
    );
    multiDocs.push(new Document({
      pageContent: questions as string,
      metadata: { ...doc.metadata, type: "questions" }
    }));
  }

  return multiDocs;
}

// 创建多向量存储
const multiDocs = await createMultiRepresentations(documents);
const embeddings = new OpenAIEmbeddings();
const multiVectorStore = await MemoryVectorStore.fromDocuments(multiDocs, embeddings);

// 检索
const results = await multiVectorStore.similaritySearch("什么是神经网络？", 5);
```

## 检索过滤

检索过滤（Retrieval Filtering）在检索后对结果进行筛选，确保结果符合特定条件。

### 元数据过滤

```typescript
import { OpenAIEmbeddings } from "@langchain/openai";
import { MemoryVectorStore } from "langchain/vectorstores/memory";

// 创建带元数据的文档
const documentsWithMetadata = documents.map(doc => new Document({
  pageContent: doc.pageContent,
  metadata: {
    source: doc.source,
    category: doc.category,
    date: doc.date,
    language: doc.language
  }
}));

// 创建向量存储
const embeddings = new OpenAIEmbeddings();
const vectorStore = await MemoryVectorStore.fromDocuments(documentsWithMetadata, embeddings);

// 元数据过滤检索
const query = "机器学习教程";

// 方式1：使用 filter 参数
const results = await vectorStore.similaritySearch(query, 5, {
  filter: {
    category: "tutorial",
    language: "zh"
  }
});

// 方式2：使用检索器的 filter
const retriever = vectorStore.asRetriever({
  k: 5,
  filter: (doc) => {
    return doc.metadata.category === "tutorial" &&
           doc.metadata.language === "zh" &&
           new Date(doc.metadata.date) > new Date("2024-01-01");
  }
});

const filteredResults = await retriever.invoke(query);
```

### 自定义过滤器

```typescript
import { Document } from "@langchain/core/documents";

// 自定义过滤器
class CustomFilter {
  // 按长度过滤
  byLength(minLength: number, maxLength: number) {
    return (doc: Document): boolean => {
      return doc.pageContent.length >= minLength &&
             doc.pageContent.length <= maxLength;
    };
  }

  // 按关键词过滤
  byKeywords(keywords: string[]) {
    return (doc: Document): boolean => {
      const contentLower = doc.pageContent.toLowerCase();
      return keywords.some(keyword => contentLower.includes(keyword.toLowerCase()));
    };
  }

  // 按日期过滤
  byDate(startDate: Date, endDate: Date) {
    return (doc: Document): boolean => {
      const docDate = new Date(doc.metadata.date);
      return docDate >= startDate && docDate <= endDate;
    };
  }

  // 组合过滤器
  combine(...filters: Array<(doc: Document) => boolean>) {
    return (doc: Document): boolean => {
      return filters.every(filter => filter(doc));
    };
  }
}

// 使用自定义过滤器
const filter = new CustomFilter();
const combinedFilter = filter.combine(
  filter.byLength(100, 5000),
  filter.byKeywords(["机器学习", "深度学习"]),
  filter.byDate(new Date("2024-01-01"), new Date())
);

const retriever = vectorStore.asRetriever({
  k: 10,
  filter: combinedFilter
});
```

### 语义过滤

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { OpenAIEmbeddings } from "@langchain/openai";
import { MemoryVectorStore } from "langchain/vectorstores/memory";
import { Document } from "@langchain/core/documents";

// 语义过滤器 - 使用 LLM 判断相关性
class SemanticFilter {
  private model: ChatOpenAI;

  constructor() {
    this.model = new ChatOpenAI({ modelName: "gpt-4" });
  }

  async filterBySemantic(
    docs: Document[],
    query: string,
    threshold: number = 0.7
  ): Promise<Document[]> {
    const filteredDocs: Document[] = [];

    for (const doc of docs) {
      const response = await this.model.invoke(`
请判断以下文档内容与查询的相关性，返回0-1之间的分数。

查询: ${query}
文档: ${doc.pageContent}

只返回数字分数，不要其他内容。
`);

      const score = parseFloat(response as string);

      if (score >= threshold) {
        filteredDocs.push(doc);
      }
    }

    return filteredDocs;
  }
}

// 使用语义过滤
const semanticFilter = new SemanticFilter();
const results = await vectorStore.similaritySearch(query, 20);
const filteredResults = await semanticFilter.filterBySemantic(results, query);
```

## 检索排序

检索排序（Retrieval Ranking）对检索结果进行重新排序，将最相关的结果排在前面。

### 相关性排序

```typescript
import { Document } from "@langchain/core/documents";

// 相关性排序器
class RelevanceRanker {
  // 基于关键词的排序
  rankByKeywords(docs: Document[], query: string): Document[] {
    const queryWords = query.toLowerCase().split(/\s+/);

    return docs.sort((a, b) => {
      const scoreA = this.calculateKeywordScore(a.pageContent, queryWords);
      const scoreB = this.calculateKeywordScore(b.pageContent, queryWords);
      return scoreB - scoreA;
    });
  }

  // 基于 TF-IDF 的排序
  rankByTFIDF(docs: Document[], query: string): Document[] {
    // 计算 TF-IDF
    const tfidfScores = docs.map(doc => ({
      doc,
      score: this.calculateTFIDF(doc.pageContent, query, docs)
    }));

    // 按分数排序
    return tfidfScores
      .sort((a, b) => b.score - a.score)
      .map(item => item.doc);
  }

  private calculateKeywordScore(content: string, queryWords: string[]): number {
    const contentLower = content.toLowerCase();
    let score = 0;

    for (const word of queryWords) {
      const regex = new RegExp(word, 'gi');
      const matches = contentLower.match(regex);
      if (matches) {
        score += matches.length;
      }
    }

    // 归一化
    return score / content.length;
  }

  private calculateTFIDF(
    content: string,
    query: string,
    allDocs: Document[]
  ): number {
    const queryWords = query.toLowerCase().split(/\s+/);
    let score = 0;

    for (const word of queryWords) {
      // TF: 词频
      const tf = (content.toLowerCase().match(new RegExp(word, 'gi')) || []).length;

      // IDF: 逆文档频率
      const docsWithWord = allDocs.filter(doc =>
        doc.pageContent.toLowerCase().includes(word)
      ).length;
      const idf = Math.log(allDocs.length / (docsWithWord + 1));

      score += tf * idf;
    }

    return score;
  }
}

// 使用排序器
const ranker = new RelevanceRanker();
const query = "神经网络训练方法";
const results = await vectorStore.similaritySearch(query, 20);
const rankedResults = ranker.rankByKeywords(results, query);
```

### 学习排序

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { Document } from "@langchain/core/documents";

// LLM 排序器
class LLMRanker {
  private model: ChatOpenAI;

  constructor() {
    this.model = new ChatOpenAI({ modelName: "gpt-4" });
  }

  async rank(docs: Document[], query: string): Promise<Document[]> {
    // 使用 LLM 进行排序
    const prompt = `请根据与查询的相关性对以下文档进行排序。
查询: ${query}

文档列表:
${docs.map((doc, i) => `[${i}] ${doc.pageContent.substring(0, 200)}...`).join('\n')}

请返回排序后的文档索引列表（从最相关到最不相关），用逗号分隔。
例如: 2,0,1,3`;

    const response = await this.model.invoke(prompt);
    const indices = (response as string).split(',').map(Number);

    // 按 LLM 返回的顺序重新排列
    return indices.map(i => docs[i]).filter(doc => doc);
  }

  async rankWithScores(
    docs: Document[],
    query: string
  ): Promise<Array<{ doc: Document; score: number }>> {
    const prompt = `请根据与查询的相关性对以下文档进行评分（0-10分）。
查询: ${query}

文档列表:
${docs.map((doc, i) => `[${i}] ${doc.pageContent.substring(0, 200)}...`).join('\n')}

请返回每个文档的分数，格式为 "索引:分数"，每行一个。
例如:
0:8
1:6
2:9`;

    const response = await this.model.invoke(prompt);
    const lines = (response as string).trim().split('\n');

    const scores = lines.map(line => {
      const [indexStr, scoreStr] = line.split(':');
      return {
        doc: docs[parseInt(indexStr)],
        score: parseFloat(scoreStr)
      };
    });

    return scores.sort((a, b) => b.score - a.score);
  }
}

// 使用 LLM 排序
const llmRanker = new LLMRanker();
const rankedResults = await llmRanker.rank(results, query);
```

## 检索重排

检索重排（Retrieval Reranking）使用专门的重排模型对检索结果进行二次排序，提高排序质量。

### Cross-Encoder 重排

```typescript
import { OpenAIEmbeddings } from "@langchain/openai";
import { MemoryVectorStore } from "langchain/vectorstores/memory";
import { CohereRerank } from "@langchain/cohere";
import { ContextualCompressionRetriever } from "langchain/retrievers/contextual_compression";

// 创建基础检索器
const embeddings = new OpenAIEmbeddings();
const vectorStore = await MemoryVectorStore.fromDocuments(documents, embeddings);
const baseRetriever = vectorStore.asRetriever({ k: 20 });

// 创建 Cohere 重排器
const reranker = new CohereRerank({
  apiKey: process.env.COHERE_API_KEY,
  model: "rerank-multilingual-v2.0",
  topN: 5  // 返回前5个结果
});

// 创建带重排的检索器
const rerankingRetriever = new ContextualCompressionRetriever({
  baseCompressor: reranker,
  baseRetriever
});

// 执行检索和重排
const query = "深度学习的应用场景";
const results = await rerankingRetriever.invoke(query);

console.log(`重排后返回 ${results.length} 个结果`);
```

### LLM 重排

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { Document } from "@langchain/core/documents";

// LLM 重排器
class LLMReranker {
  private model: ChatOpenAI;

  constructor() {
    this.model = new ChatOpenAI({ modelName: "gpt-4" });
  }

  async rerank(
    docs: Document[],
    query: string,
    topK: number = 5
  ): Promise<Document[]> {
    // 对每个文档进行相关性评分
    const scoredDocs = await Promise.all(
      docs.map(async (doc) => {
        const response = await this.model.invoke({
          messages: [
            {
              role: "system",
              content: `请评估以下文档与查询的相关性，返回0-10的分数。
只返回数字，不要其他内容。`
            },
            {
              role: "user",
              content: `查询: ${query}\n文档: ${doc.pageContent}`
            }
          ]
        });

        const score = parseFloat(response as string) || 0;

        return { doc, score };
      })
    );

    // 按分数排序并返回前K个
    return scoredDocs
      .sort((a, b) => b.score - a.score)
      .slice(0, topK)
      .map(item => item.doc);
  }
}

// 使用 LLM 重排
const llmReranker = new LLMReranker();
const initialResults = await vectorStore.similaritySearch(query, 20);
const rerankedResults = await llmReranker.rerank(initialResults, query, 5);
```

### 重排管道

```typescript
import { OpenAIEmbeddings } from "@langchain/openai";
import { MemoryVectorStore } from "langchain/vectorstores/memory";
import { Document } from "@langchain/core/documents";

// 重排管道
class RerankingPipeline {
  private stages: Array<(docs: Document[], query: string) => Promise<Document[]>> = [];

  // 添加重排阶段
  addStage(
    name: string,
    reranker: (docs: Document[], query: string) => Promise<Document[]>
  ) {
    this.stages.push(reranker);
    return this;
  }

  // 执行重排管道
  async execute(
    docs: Document[],
    query: string
  ): Promise<Document[]> {
    let results = docs;

    for (const stage of this.stages) {
      results = await stage(results, query);
    }

    return results;
  }
}

// 创建重排管道
const pipeline = new RerankingPipeline();

// 阶段1：去重
pipeline.addStage("deduplication", async (docs) => {
  const seen = new Set<string>();
  return docs.filter(doc => {
    const key = doc.pageContent;
    if (seen.has(key)) return false;
    seen.add(key);
    return true;
  });
});

// 阶段2：长度过滤
pipeline.addStage("length_filter", async (docs) => {
  return docs.filter(doc => doc.pageContent.length >= 100);
});

// 阶段3：LLM 重排
pipeline.addStage("llm_rerank", async (docs, query) => {
  const reranker = new LLMReranker();
  return reranker.rerank(docs, query, 5);
});

// 使用重排管道
const initialResults = await vectorStore.similaritySearch(query, 50);
const finalResults = await pipeline.execute(initialResults, query);
```

### 混合重排

```typescript
import { OpenAIEmbeddings } from "@langchain/openai";
import { MemoryVectorStore } from "langchain/vectorstores/memory";
import { BM25Retriever } from "langchain/retrievers/bm25";
import { Document } from "@langchain/core/documents";

// 混合重排器
class HybridReranker {
  private vectorWeight: number;
  private bm25Weight: number;
  private llmWeight: number;

  constructor(options: {
    vectorWeight?: number;
    bm25Weight?: number;
    llmWeight?: number;
  } = {}) {
    this.vectorWeight = options.vectorWeight || 0.4;
    this.bm25Weight = options.bm25Weight || 0.3;
    this.llmWeight = options.llmWeight || 0.3;
  }

  async rerank(
    docs: Document[],
    query: string,
    vectorScores: Map<string, number>,
    bm25Scores: Map<string, number>
  ): Promise<Document[]> {
    const llmReranker = new LLMReranker();
    const llmScoredDocs = await llmReranker.rerankWithScores(docs, query);

    // 归一化分数
    const normalizedScores = docs.map(doc => {
      const vectorScore = this.normalizeScore(vectorScores.get(doc.pageContent) || 0);
      const bm25Score = this.normalizeScore(bm25Scores.get(doc.pageContent) || 0);
      const llmScore = this.normalizeScore(
        llmScoredDocs.find(s => s.doc.pageContent === doc.pageContent)?.score || 0
      );

      // 加权组合
      const finalScore =
        this.vectorWeight * vectorScore +
        this.bm25Weight * bm25Score +
        this.llmWeight * llmScore;

      return { doc, score: finalScore };
    });

    // 按最终分数排序
    return normalizedScores
      .sort((a, b) => b.score - a.score)
      .map(item => item.doc);
  }

  private normalizeScore(score: number): number {
    // 简单的归一化到 0-1
    return Math.min(Math.max(score / 10, 0), 1);
  }
}

// 使用混合重排
const hybridReranker = new HybridReranker({
  vectorWeight: 0.5,
  bm25Weight: 0.3,
  llmWeight: 0.2
});
```

> **提示**：重排会增加额外的延迟和成本，建议在对结果质量要求较高的场景中使用。

> **建议**：结合多种重排策略，根据具体需求调整各策略的权重。

## 最佳实践

1. **选择合适的检索策略**：根据数据特点选择向量检索、关键词检索或混合检索
2. **优化过滤条件**：使用元数据过滤缩小检索范围
3. **合理的重排策略**：在性能和质量之间找到平衡
4. **监控检索质量**：定期评估检索效果，持续优化
5. **缓存优化**：对频繁查询进行缓存，提高响应速度

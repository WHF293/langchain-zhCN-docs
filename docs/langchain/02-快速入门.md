# 第2章 快速入门

> 来源: [LangChain 官方文档](https://python.langchain.com/)

本教程将带你从零开始，创建你的第一个 LangChain 应用。

## 安装

### 环境要求

- Node.js >= 18
- npm / yarn / pnpm

### 安装依赖

```bash
# 使用 npm
npm install @langchain/core @langchain/openai @langchain/langgraph

# 使用 yarn
yarn add @langchain/core @langchain/openai @langchain/langgraph

# 使用 pnpm
pnpm add @langchain/core @langchain/openai @langchain/langgraph
```

### 配置环境变量

创建 `.env` 文件：

```bash
# OpenAI API 密钥
OPENAI_API_KEY=sk-your-api-key-here

# 可选：自定义 API 基础地址
# OPENAI_API_BASE=https://api.openai.com/v1
```

> **注意**：请确保你的 API 密钥安全，不要提交到版本控制系统中。

## 创建第一个链

### 基础示例

创建一个简单的翻译链：

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatPromptTemplate } from "@langchain/core/prompts";
import { StringOutputParser } from "@langchain/core/output_parsers";

// 1. 创建模型实例
const model = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0.7,
});

// 2. 创建提示模板
const prompt = ChatPromptTemplate.fromMessages([
  ["system", "你是一个专业的翻译员，请将用户输入翻译成{language}"],
  ["user", "{text}"],
]);

// 3. 创建输出解析器
const parser = new StringOutputParser();

// 4. 组合成链
const chain = prompt.pipe(model).pipe(parser);

// 5. 调用链
const result = await chain.invoke({
  language: "英语",
  text: "你好，世界！今天天气真好。",
});

console.log(result);
// 输出: "Hello, World! The weather is really nice today."
```

### 代码解析

| 步骤 | 组件 | 作用 |
|:---:|:---|:---|
| 1 | ChatOpenAI | 创建 LLM 模型实例 |
| 2 | ChatPromptTemplate | 定义提示模板，支持变量替换 |
| 3 | StringOutputParser | 将模型输出转换为字符串 |
| 4 | .pipe() | 将组件串联成链 |
| 5 | .invoke() | 执行链并获取结果 |

### 流式输出

对于长文本生成，可以使用流式输出：

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatPromptTemplate } from "@langchain/core/prompts";

const model = new ChatOpenAI({ modelName: "gpt-4" });
const prompt = ChatPromptTemplate.fromMessages([
  ["user", "写一个关于{topic}的短故事"],
]);

const chain = prompt.pipe(model);

// 使用 stream 方法获取流式输出
const stream = await chain.stream({ topic: "太空探险" });

for await (const chunk of stream) {
  process.stdout.write(chunk.content as string);
}
```

## 使用工具

让模型调用外部工具来获取信息或执行操作。

### 定义工具

```typescript
import { tool } from "@langchain/core/tools";
import { z } from "zod";

// 定义一个天气查询工具
const weatherTool = tool(
  async ({ city }) => {
    // 这里模拟 API 调用
    const weatherData: Record<string, string> = {
      "北京": "晴天，25°C",
      "上海": "多云，22°C",
      "广州": "阵雨，28°C",
    };

    return weatherData[city] || `未找到 ${city} 的天气信息`;
  },
  {
    name: "get_weather",
    description: "获取指定城市的天气信息",
    schema: z.object({
      city: z.string().describe("城市名称，如：北京、上海"),
    }),
  }
);

// 定义一个计算器工具
const calculatorTool = tool(
  async ({ expression }) => {
    try {
      // 注意：生产环境应使用安全的表达式解析器
      const result = eval(expression);
      return `计算结果：${expression} = ${result}`;
    } catch {
      return `计算错误：无法计算表达式 "${expression}"`;
    }
  },
  {
    name: "calculator",
    description: "计算数学表达式",
    schema: z.object({
      expression: z.string().describe("数学表达式，如：2 + 2"),
    }),
  }
);
```

### 创建使用工具的代理

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { createReactAgent } from "@langchain/langgraph/prebuilt";

// 创建模型
const model = new ChatOpenAI({ modelName: "gpt-4" });

// 创建代理
const agent = createReactAgent({
  llm: model,
  tools: [weatherTool, calculatorTool],
});

// 使用代理
const result = await agent.invoke({
  messages: [
    { role: "user", content: "北京今天天气怎么样？如果气温超过 20 度，告诉我需要穿什么衣服。" }
  ],
});

console.log(result.messages[result.messages.length - 1].content);
```

## 创建 Agent

### 基础 Agent

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { createReactAgent } from "@langchain/langgraph/prebuilt";
import { tool } from "@langchain/core/tools";
import { z } from "zod";
import { MemorySaver } from "@langchain/langgraph";

// 定义工具
const searchTool = tool(
  async ({ query }) => {
    // 模拟搜索
    return `搜索结果：${query} 的最新信息...`;
  },
  {
    name: "search",
    description: "搜索互联网获取信息",
    schema: z.object({
      query: z.string().describe("搜索关键词"),
    }),
  }
);

// 创建模型
const model = new ChatOpenAI({ modelName: "gpt-4" });

// 创建记忆存储
const memory = new MemorySaver();

// 创建 Agent
const agent = createReactAgent({
  llm: model,
  tools: [searchTool],
  checkpointSaver: memory,
});

// 第一轮对话
const result1 = await agent.invoke(
  {
    messages: [
      { role: "user", content: "我叫小明，请记住我的名字" }
    ],
  },
  { configurable: { thread_id: "user-123" } }
);

console.log(result1.messages[result1.messages.length - 1].content);

// 第二轮对话（会记住之前的上下文）
const result2 = await agent.invoke(
  {
    messages: [
      { role: "user", content: "我的名字是什么？" }
    ],
  },
  { configurable: { thread_id: "user-123" } }
);

console.log(result2.messages[result2.messages.length - 1].content);
// 输出会包含"小明"
```

### 带人机交互的 Agent

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { createReactAgent } from "@langchain/langgraph/prebuilt";
import { tool } from "@langchain/core/tools";
import { z } from "zod";
import { MemorySaver, interrupt, Command } from "@langchain/langgraph";

// 定义需要人工确认的工具
const sendEmailTool = tool(
  async ({ to, subject, body }) => {
    // 模拟发送邮件
    return `邮件已发送给 ${to}，主题：${subject}`;
  },
  {
    name: "send_email",
    description: "发送电子邮件",
    schema: z.object({
      to: z.string().describe("收件人邮箱"),
      subject: z.string().describe("邮件主题"),
      body: z.string().describe("邮件内容"),
    }),
  }
);

// 创建带中断的 Agent
const agent = createReactAgent({
  llm: new ChatOpenAI({ modelName: "gpt-4" }),
  tools: [sendEmailTool],
  interruptBefore: ["tools"],  // 在执行工具前中断
});
```

> **提示**：`interruptBefore` 和 `interruptAfter` 可以让你在 Agent 执行的关键节点进行人工审核。

## 完整示例：聊天机器人

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatPromptTemplate, MessagesPlaceholder } from "@langchain/core/prompts";
import { StringOutputParser } from "@langchain/core/output_parsers";
import { RunnableWithMessageHistory } from "@langchain/core/runnables";
import { ChatMessageHistory } from "@langchain/core/stores/history/in_memory";

// 创建模型
const model = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0.7,
});

// 创建提示模板
const prompt = ChatPromptTemplate.fromMessages([
  ["system", "你是一个友好的 AI 助手，名叫小助。请用中文回答问题。"],
  new MessagesPlaceholder("history"),
  ["user", "{input}"],
]);

// 创建链
const chain = prompt.pipe(model).pipe(new StringOutputParser());

// 创建消息历史存储
const messageHistories = new Map<string, ChatMessageHistory>();

// 创建带历史的链
const withMessageHistory = new RunnableWithMessageHistory({
  runnable: chain,
  getMessageHistory: (sessionId) => {
    if (!messageHistories.has(sessionId)) {
      messageHistories.set(sessionId, new ChatMessageHistory());
    }
    return messageHistories.get(sessionId)!;
  },
  inputMessagesKey: "input",
  historyMessagesKey: "history",
});

// 聊天函数
async function chat(sessionId: string, input: string) {
  const result = await withMessageHistory.invoke(
    { input },
    { configurable: { sessionId } }
  );
  return result;
}

// 使用示例
console.log(await chat("session-1", "你好，我叫小明"));
console.log(await chat("session-1", "我的名字是什么？"));
```

## 常见问题

### API 密钥错误

```
Error: OpenAI API key not found
```

**解决方案**：确保已正确设置 `OPENAI_API_KEY` 环境变量。

### 模型不存在

```
Error: The model `gpt-xxx` does not exist
```

**解决方案**：检查模型名称是否正确，确认你的 API 账户有权访问该模型。

### 超时错误

```
Error: Request timed out
```

**解决方案**：检查网络连接，或增加超时时间配置。

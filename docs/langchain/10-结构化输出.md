# 第10章 结构化输出

> 来源: [LangChain 官方文档](https://python.langchain.com/)

---

## 概述

结构化输出 (Structured Output) 是指让语言模型返回符合特定格式的数据，而不是自由形式的文本。这对于构建可靠的应用程序至关重要，因为结构化数据可以直接用于程序逻辑，无需复杂的文本解析。

## 输出模式

### 使用 withStructuredOutput

LangChain 提供了 `withStructuredOutput` 方法，可以直接从模型获取结构化数据：

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { z } from "zod";

// 定义输出模式
const PersonSchema = z.object({
  name: z.string().describe("人名"),
  age: z.number().int().min(0).max(150).describe("年龄"),
  email: z.string().email().describe("电子邮件地址"),
  occupation: z.string().describe("职业"),
  skills: z.array(z.string()).describe("技能列表"),
});

// 创建模型并绑定模式
const model = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0,
});

const structuredModel = model.withStructuredOutput(PersonSchema);

// 调用模型
const result = await structuredModel.invoke(
  "提取信息：张三，28岁，邮箱是 zhangsan@example.com，是一名前端工程师，精通 React 和 TypeScript"
);

console.log(result);
// 输出: {
//   name: "张三",
//   age: 28,
//   email: "zhangsan@example.com",
//   occupation: "前端工程师",
//   skills: ["React", "TypeScript"]
// }
```

### JSON Schema 定义

除了 Zod，也可以使用 JSON Schema 定义输出模式：

```typescript
import { ChatOpenAI } from "@langchain/openai";

const model = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0,
});

// 使用 JSON Schema
const schema = {
  type: "object",
  properties: {
    sentiment: {
      type: "string",
      enum: ["positive", "negative", "neutral"],
      description: "情感分类",
    },
    confidence: {
      type: "number",
      minimum: 0,
      maximum: 1,
      description: "置信度",
    },
    keywords: {
      type: "array",
      items: { type: "string" },
      description: "关键词列表",
    },
    summary: {
      type: "string",
      description: "简短摘要",
    },
  },
  required: ["sentiment", "confidence", "keywords", "summary"],
};

const structuredModel = model.withStructuredOutput(schema);

const result = await structuredModel.invoke(
  "分析这段文本的情感：'这个产品真的很棒，我非常喜欢！'"
);

console.log(result);
```

## 输出验证

```typescript
import { ChatOpenAI } from "@langchain/openai";

const ProductReviewSchema = z.object({
```

### 自定义验证器

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { z } from "zod";

// 自定义验证逻辑
const OrderSchema = z
  .object({
    orderId: z.string().describe("订单号"),
    items: z.array(
      z.object({
        name: z.string(),
        quantity: z.number().int().positive(),
        price: z.number().positive(),
      })
    ),
    totalAmount: z.number().positive(),
    discount: z.number().min(0).max(100).optional(),
  })
  .refine(
    (data) => {
      // 验证总金额是否等于各项金额之和
      const calculatedTotal = data.items.reduce(
        (sum, item) => sum + item.quantity * item.price,
        0
      );
      return Math.abs(calculatedTotal - data.totalAmount) < 0.01;
    },
    {
      message: "总金额必须等于各项金额之和",
      path: ["totalAmount"],
    }
  )
  .refine(
    (data) => {
      // 验证订单号格式
      return /^ORD-\d{8}$/.test(data.orderId);
    },
    {
      message: "订单号格式必须为 ORD-XXXXXXXX",
      path: ["orderId"],
    }
  );

const model = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0,
});

const structuredModel = model.withStructuredOutput(OrderSchema);

const result = await structuredModel.invoke(`
  订单号: ORD-20240115
  商品:
  - iPhone 15, 数量 1, 价格 7999
  - AirPods Pro, 数量 2, 价格 1899
  总金额: 11797
  折扣: 10%
`);

console.log("验证通过:", result);
```

## 输出解析

### 使用 OutputParser

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { StructuredOutputParser } from "@langchain/core/output_parsers";
import { z } from "zod";

// 定义输出模式
const schema = z.object({
  answer: z.string().describe("问题的答案"),
  sources: z.array(z.string()).describe("信息来源"),
  confidence: z.number().min(0).max(1).describe("置信度"),
});

// 创建结构化输出解析器
const parser = StructuredOutputParser.fromZodSchema(schema);

// 获取格式说明
const formatInstructions = parser.getFormatInstructions();
console.log("格式说明:", formatInstructions);

// 创建模型
const model = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0,
});

// 构建提示模板
import { ChatPromptTemplate } from "@langchain/core/prompts";

const prompt = ChatPromptTemplate.fromTemplate(
  `回答以下问题，并按照指定格式输出。

问题: {question}

{format_instructions}`
);

// 创建链
const chain = prompt.pipe(model).pipe(parser);

// 调用链
const result = await chain.invoke({
  question: "什么是 LangChain？",
  format_instructions: formatInstructions,
});

console.log("解析结果:", result);
```

### 自定义输出解析器

```typescript
import { BaseOutputParser } from "@langchain/core/output_parsers";

interface ParsedData {
  command: string;
  args: Record<string, string>;
}

class CommandParser extends BaseOutputParser<ParsedData> {
  lc_namespace = ["langchain", "output_parsers", "custom"];

  getFormatInstructions(): string {
    return `请以以下格式输出命令:
COMMAND: <命令名称>
ARG: <参数名>=<参数值>
ARG: <参数名>=<参数值>
...`;
  }

  async parse(text: string): Promise<ParsedData> {
    const lines = text.trim().split("\n");
    const commandLine = lines.find((line) => line.startsWith("COMMAND:"));

    if (!commandLine) {
      throw new Error("无法解析命令: 缺少 COMMAND 行");
    }

    const command = commandLine.replace("COMMAND:", "").trim();
    const args: Record<string, string> = {};

    for (const line of lines) {
      if (line.startsWith("ARG:")) {
        const argContent = line.replace("ARG:", "").trim();
        const [key, ...valueParts] = argContent.split("=");
        if (key && valueParts.length > 0) {
          args[key.trim()] = valueParts.join("=").trim();
        }
      }
    }

    return { command, args };
  }
}

// 使用自定义解析器
const model = new ChatOpenAI({ modelName: "gpt-4", temperature: 0 });
const parser = new CommandParser();

const prompt = ChatPromptTemplate.fromTemplate(
  `将用户的请求转换为命令格式。

用户请求: {request}

{format_instructions}`
);

const chain = prompt.pipe(model).pipe(parser);

const result = await chain.invoke({
  request: "帮我创建一个名为 test 的文件，内容是 hello world",
  format_instructions: parser.getFormatInstructions(),
});

console.log("命令:", result);
// 输出: { command: "CREATE_FILE", args: { name: "test", content: "hello world" } }
```

## 输出重试

### 自动重试机制

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { withRetry } from "@langchain/core/output_parsers";
import { StructuredOutputParser } from "@langchain/core/output_parsers";
import { z } from "zod";

const schema = z.object({
  result: z.number().describe("计算结果"),
  steps: z.array(z.string()).describe("计算步骤"),
});

const parser = StructuredOutputParser.fromZodSchema(schema);

// 添加重试逻辑
const parserWithRetry = parser.withRetry({
  maxRetries: 3,
});

const model = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0,
});

const prompt = ChatPromptTemplate.fromTemplate(
  `计算以下数学问题，显示步骤:

问题: {question}

{format_instructions}`
);

const chain = prompt.pipe(model).pipe(parserWithRetry);

const result = await chain.invoke({
  question: "计算 (15 + 7) * 3",
  format_instructions: parser.getFormatInstructions(),
});

console.log("计算结果:", result);
```

### 处理解析错误

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { OutputParserException } from "@langchain/core/output_parsers";
import { StructuredOutputParser } from "@langchain/core/output_parsers";
import { z } from "zod";

const schema = z.object({
  title: z.string(),
  content: z.string(),
  tags: z.array(z.string()),
});

const parser = StructuredOutputParser.fromZodSchema(schema);

const model = new ChatOpenAI({ modelName: "gpt-4", temperature: 0 });

async function parseWithFallback(input: string, maxRetries = 3) {
  for (let attempt = 1; attempt <= maxRetries; attempt++) {
    try {
      const result = await parser.parse(input);
      return result;
    } catch (error) {
      if (error instanceof OutputParserException) {
        console.warn(`解析尝试 ${attempt}/${maxRetries} 失败:`, error.message);

        if (attempt === maxRetries) {
          // 最后一次尝试，使用宽松解析
          console.log("使用宽松解析模式...");
          return parseLenient(input);
        }

        // 重试前等待
        await new Promise((resolve) => setTimeout(resolve, 1000 * attempt));
      } else {
        throw error;
      }
    }
  }
}

function parseLenient(input: string) {
  // 尝试从文本中提取 JSON
  const jsonMatch = input.match(/\{[\s\S]*\}/);
  if (jsonMatch) {
    try {
      return JSON.parse(jsonMatch[0]);
    } catch {
      // JSON 解析失败，返回默认值
      return {
        title: "解析失败",
        content: input,
        tags: [],
      };
    }
  }

  return {
    title: "未知",
    content: input,
    tags: [],
  };
}
```

### 验证后重试

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatPromptTemplate } from "@langchain/core/prompts";
import { z } from "zod";

const schema = z.object({
  email: z.string().email("必须是有效的电子邮件地址"),
  phone: z.string().regex(/^\d{11}$/, "手机号必须是 11 位数字"),
  age: z.number().int().min(18).max(100, "年龄必须在 18-100 之间"),
});

const model = new ChatOpenAI({ modelName: "gpt-4", temperature: 0 });
const structuredModel = model.withStructuredOutput(schema);

async function invokeWithValidation(input: string, maxRetries = 3) {
  for (let attempt = 1; attempt <= maxRetries; attempt++) {
    try {
      const result = await structuredModel.invoke(input);
      return result;
    } catch (error: any) {
      console.warn(`尝试 ${attempt} 失败:`, error.message);

      if (attempt === maxRetries) {
        throw new Error(`验证失败，已重试 ${maxRetries} 次`);
      }

      // 修改输入，添加错误信息以便模型修正
      input = `${input}\n\n注意：之前的输出有以下错误，请修正：${error.message}`;
    }
  }
}

// 使用示例
const result = await invokeWithValidation(
  "提取联系信息：我的邮箱是 test@test，手机号是 12345，年龄 15"
);

console.log("验证通过:", result);
```

## 最佳实践

1. **清晰的模式描述**：为每个字段提供详细的 `description`，帮助模型理解预期输出
2. **合理使用默认值**：对于可选字段，设置合理的默认值
3. **验证层级**：结合 Zod 验证和业务逻辑验证
4. **错误处理**：实现适当的错误处理和重试机制
5. **类型安全**：利用 TypeScript 的类型推导确保类型安全

> **提示**：使用 `temperature: 0` 可以获得更一致的结构化输出结果。

> **注意**：复杂模式可能会影响模型的输出质量，建议将复杂任务拆分为多个简单步骤。

## 概述

上下文工程 (Context Engineering) 是指系统地管理和优化发送给语言模型的上下文信息的技术和方法。随着 AI 应用复杂度的增加，如何高效地组织、压缩和传递上下文成为构建高质量应用的关键。上下文工程涉及上下文窗口管理、上下文压缩、上下文隔离和上下文优化等多个方面。

## 上下文窗口

### 理解上下文窗口

上下文窗口 (Context Window) 是语言模型一次能处理的最大 token 数量。不同的模型有不同的上下文窗口大小：

| 模型 | 上下文窗口 | 大约字符数 |
|------|-----------|-----------|
| GPT-3.5-turbo | 4,096 tokens | 约 3,000 字 |
| GPT-4 | 8,192 tokens | 约 6,000 字 |
| GPT-4-turbo | 128,000 tokens | 约 96,000 字 |
| GPT-4o | 128,000 tokens | 约 96,000 字 |
| Claude 3.5 Sonnet | 200,000 tokens | 约 150,000 字 |

### Token 计数

```typescript
import { encoding_for_model } from "tiktoken";

class TokenCounter {
  private encoder: any;

  constructor(model: string = "gpt-4") {
    this.encoder = encoding_for_model(model as any);
  }

  countTokens(text: string): number {
    return this.encoder.encode(text).length;
  }

  countMessagesTokens(messages: { role: string; content: string }[]): number {
    let totalTokens = 0;

    for (const message of messages) {
      // 每条消息有固定开销
      totalTokens += 4; // <|start|>, role, \n, <|end|>

      // 角色 token
      totalTokens += this.countTokens(message.role);

      // 内容 token
      totalTokens += this.countTokens(message.content);
    }

    // 回复开始 token
    totalTokens += 2;

    return totalTokens;
  }

  canFitInWindow(
    messages: { role: string; content: string }[],
    maxTokens: number,
    reservedForOutput: number = 1000
  ): boolean {
    const usedTokens = this.countMessagesTokens(messages);
    return usedTokens + reservedForOutput <= maxTokens;
  }

  trimMessagesToWindow(
    messages: { role: string; content: string }[],
    maxTokens: number,
    reservedForOutput: number = 1000
  ): { role: string; content: string }[] {
    const targetTokens = maxTokens - reservedForOutput;
    const result: { role: string; content: string }[] = [];
    let currentTokens = 0;

    // 从最新的消息开始保留（跳过最后一条，因为通常是要发送的新消息）
    for (let i = messages.length - 1; i >= 0; i--) {
      const message = messages[i];
      const messageTokens = 4 + this.countTokens(message.role) + this.countTokens(message.content);

      if (currentTokens + messageTokens > targetTokens) {
        break;
      }

      result.unshift(message);
      currentTokens += messageTokens;
    }

    return result;
  }

  free() {
    this.encoder.free();
  }
}

// 使用示例
const counter = new TokenCounter("gpt-4");

const text = "这是一段测试文本，用于计算 token 数量。";
console.log(`Token 数: ${counter.countTokens(text)}`);

const messages = [
  { role: "system", content: "你是一个有帮助的助手。" },
  { role: "user", content: "你好！" },
  { role: "assistant", content: "你好！有什么可以帮助你的吗？" },
  { role: "user", content: "请告诉我关于 TypeScript 的知识。" },
];

console.log(`消息 Token 数: ${counter.countMessagesTokens(messages)}`);
console.log(`能否放入 8K 窗口: ${counter.canFitInWindow(messages, 8192)}`);

counter.free();
```

### 动态上下文窗口管理

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { BaseMessage, SystemMessage, HumanMessage, AIMessage } from "@langchain/core/messages";

interface ContextWindowConfig {
  maxTokens: number;
  reservedForOutput: number;
  systemPromptReserve: number;
  minHistoryMessages: number;
}

class ContextWindowManager {
  private config: ContextWindowConfig;
  private tokenCounter: TokenCounter;
  private systemPrompt: string;

  constructor(
    config: ContextWindowConfig,
    systemPrompt: string,
    model: string = "gpt-4"
  ) {
    this.config = config;
    this.tokenCounter = new TokenCounter(model);
    this.systemPrompt = systemPrompt;
  }

  buildContext(
    history: BaseMessage[],
    currentInput: string,
    additionalContext?: string
  ): BaseMessage[] {
    const messages: BaseMessage[] = [];

    // 1. 计算固定开销
    const systemTokens = this.tokenCounter.countTokens(this.systemPrompt);
    const inputTokens = this.tokenCounter.countTokens(currentInput);
    const contextTokens = additionalContext
      ? this.tokenCounter.countTokens(additionalContext)
      : 0;

    const fixedTokens = systemTokens + inputTokens + contextTokens;
    const availableForHistory =
      this.config.maxTokens -
      this.config.reservedForOutput -
      fixedTokens;

    // 2. 添加系统消息
    messages.push(new SystemMessage(this.systemPrompt));

    // 3. 添加额外上下文
    if (additionalContext) {
      messages.push(new SystemMessage(additionalContext));
    }

    // 4. 添加历史消息（从最新的开始）
    const trimmedHistory = this.trimHistory(history, availableForHistory);
    messages.push(...trimmedHistory);

    // 5. 添加当前输入
    messages.push(new HumanMessage(currentInput));

    return messages;
  }

  private trimHistory(history: BaseMessage[], availableTokens: number): BaseMessage[] {
    const result: BaseMessage[] = [];
    let usedTokens = 0;

    // 从最新的消息开始
    for (let i = history.length - 1; i >= 0; i--) {
      const message = history[i];
      const content = typeof message.content === "string" ? message.content : "";
      const messageTokens = this.tokenCounter.countTokens(content) + 4;

      if (usedTokens + messageTokens > availableTokens) {
        break;
      }

      result.unshift(message);
      usedTokens += messageTokens;
    }

    // 确保至少保留最少数量的历史消息
    if (result.length < this.config.minHistoryMessages && history.length > 0) {
      return history.slice(-this.config.minHistoryMessages);
    }

    return result;
  }

  getAvailableTokens(history: BaseMessage[], currentInput: string): number {
    const systemTokens = this.tokenCounter.countTokens(this.systemPrompt);
    const inputTokens = this.tokenCounter.countTokens(currentInput);
    const historyTokens = this.tokenCounter.countMessagesTokens(
      history.map((m) => ({
        role: m._getType(),
        content: typeof m.content === "string" ? m.content : "",
      }))
    );

    return (
      this.config.maxTokens -
      this.config.reservedForOutput -
      systemTokens -
      inputTokens -
      historyTokens
    );
  }

  destroy() {
    this.tokenCounter.free();
  }
}

// 使用示例
const contextManager = new ContextWindowManager(
  {
    maxTokens: 8192,
    reservedForOutput: 1000,
    systemPromptReserve: 500,
    minHistoryMessages: 4,
  },
  "你是一个专业的 AI 助手，擅长回答技术问题。"
);

const history = [
  new HumanMessage("什么是 TypeScript？"),
  new AIMessage("TypeScript 是 JavaScript 的超集，添加了静态类型系统。"),
  new HumanMessage("它有什么优势？"),
  new AIMessage("TypeScript 的主要优势包括类型安全、更好的 IDE 支持和更易于重构。"),
];

const context = contextManager.buildContext(
  history,
  "如何在项目中使用 TypeScript？",
  "项目使用 React 框架。"
);

console.log("构建的上下文消息数:", context.length);
```

## 上下文压缩

### 摘要压缩

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { BaseMessage, HumanMessage, AIMessage, SystemMessage } from "@langchain/core/messages";

class ContextCompressor {
  private model: ChatOpenAI;
  private tokenCounter: TokenCounter;

  constructor(model: string = "gpt-4") {
    this.model = new ChatOpenAI({ modelName: model, temperature: 0 });
    this.tokenCounter = new TokenCounter(model);
  }

  async summarizeConversation(messages: BaseMessage[]): Promise<string> {
    const conversationText = messages
      .map((m) => {
        const role = m._getType() === "human" ? "用户" : "助手";
        const content = typeof m.content === "string" ? m.content : "";
        return `${role}: ${content}`;
      })
      .join("\n");

    const prompt = `请将以下对话总结为简洁的摘要，保留关键信息、用户偏好和重要结论：

${conversationText}

摘要:`;

    const response = await this.model.invoke(prompt);
    return response.content as string;
  }

  async compressContext(
    messages: BaseMessage[],
    maxTokens: number
  ): Promise<BaseMessage[]> {
    const currentTokens = this.tokenCounter.countMessagesTokens(
      messages.map((m) => ({
        role: m._getType(),
        content: typeof m.content === "string" ? m.content : "",
      }))
    );

    if (currentTokens <= maxTokens) {
      return messages;
    }

    // 需要压缩
    // 保留系统消息和最近的消息
    const systemMessages = messages.filter((m) => m._getType() === "system");
    const nonSystemMessages = messages.filter((m) => m._getType() !== "system");

    // 总结较早的消息
    const messagesToSummarize = nonSystemMessages.slice(0, -4); // 保留最近 4 条
    const recentMessages = nonSystemMessages.slice(-4);

    let summary = "";
    if (messagesToSummarize.length > 0) {
      summary = await this.summarizeConversation(messagesToSummarize);
    }

    // 重新构建消息列表
    const compressed: BaseMessage[] = [...systemMessages];

    if (summary) {
      compressed.push(
        new SystemMessage(`之前的对话摘要:\n${summary}`)
      );
    }

    compressed.push(...recentMessages);

    return compressed;
  }

  async extractKeyInformation(messages: BaseMessage[]): Promise<string> {
    const conversationText = messages
      .map((m) => {
        const role = m._getType() === "human" ? "用户" : "助手";
        const content = typeof m.content === "string" ? m.content : "";
        return `${role}: ${content}`;
      })
      .join("\n");

    const prompt = `从以下对话中提取关键信息，以要点列表形式返回：
- 用户需求和目标
- 已讨论的解决方案
- 重要决定和结论
- 待处理的事项

对话:
${conversationText}

关键信息:`;

    const response = await this.model.invoke(prompt);
    return response.content as string;
  }

  destroy() {
    this.tokenCounter.free();
  }
}

// 使用示例
const compressor = new ContextCompressor();

const longConversation = [
  new HumanMessage("我想构建一个聊天应用"),
  new AIMessage("好的，我们可以使用 WebSocket 来实现实时通信。"),
  new HumanMessage("用什么框架比较好？"),
  new AIMessage("推荐使用 Socket.io，它简单易用且兼容性好。"),
  new HumanMessage("后端用 Node.js 可以吗？"),
  new AIMessage("完全可以，Express + Socket.io 是很好的组合。"),
  new HumanMessage("数据库选什么？"),
  new AIMessage("如果需要实时同步，推荐 MongoDB；如果关系复杂，推荐 PostgreSQL。"),
  new HumanMessage("那就用 MongoDB 吧"),
  new AIMessage("好的，技术栈确定为：Node.js + Express + Socket.io + MongoDB"),
];

// 压缩上下文
const compressed = await compressor.compressContext(longConversation, 2000);
console.log("压缩后消息数:", compressed.length);

// 提取关键信息
const keyInfo = await compressor.extractKeyInformation(longConversation);
console.log("关键信息:", keyInfo);

compressor.destroy();
```

### 选择性上下文

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { BaseMessage } from "@langchain/core/messages";

interface ContextRelevance {
  message: BaseMessage;
  relevanceScore: number;
}

class SelectiveContextBuilder {
  private model: ChatOpenAI;

  constructor() {
    this.model = new ChatOpenAI({ modelName: "gpt-4", temperature: 0 });
  }

  async selectRelevantContext(
    currentQuery: string,
    availableContext: BaseMessage[],
    maxItems: number = 10
  ): Promise<BaseMessage[]> {
    // 计算每个上下文项与当前查询的相关性
    const relevanceScores = await this.scoreRelevance(
      currentQuery,
      availableContext
    );

    // 按相关性排序
    relevanceScores.sort((a, b) => b.relevanceScore - a.relevanceScore);

    // 返回最相关的项
    return relevanceScores.slice(0, maxItems).map((r) => r.message);
  }

  private async scoreRelevance(
    query: string,
    messages: BaseMessage[]
  ): Promise<ContextRelevance[]> {
    // 简化版本：基于关键词匹配和简单启发式
    const queryKeywords = this.extractKeywords(query.toLowerCase());

    return messages.map((message) => {
      const content = (
        typeof message.content === "string" ? message.content : ""
      ).toLowerCase();
      const messageKeywords = this.extractKeywords(content);

      // 计算关键词重叠
      const overlap = queryKeywords.filter((k) =>
        messageKeywords.some((mk) => mk.includes(k) || k.includes(mk))
      ).length;

      const relevanceScore =
        queryKeywords.length > 0 ? overlap / queryKeywords.length : 0;

      return { message, relevanceScore };
    });
  }

  private extractKeywords(text: string): string[] {
    const stopWords = new Set([
      "的",
      "了",
      "是",
      "在",
      "和",
      "有",
      "我",
      "你",
      "他",
      "她",
      "它",
      "们",
      "这",
      "那",
      "什么",
      "怎么",
      "如何",
      "为什么",
      "可以",
      "能够",
      "应该",
      "需要",
      "a",
      "the",
      "is",
      "are",
      "was",
      "were",
      "be",
      "been",
      "being",
      "have",
      "has",
      "had",
      "do",
      "does",
      "did",
      "will",
      "would",
      "could",
      "should",
      "may",
      "might",
      "can",
      "shall",
    ]);

    return text
      .split(/[\s,，。！？、；：""''【】《》（）()]+/)
      .filter((word) => word.length > 1 && !stopWords.has(word));
  }

  async buildOptimizedContext(
    query: string,
    systemPrompt: string,
    history: BaseMessage[],
    additionalDocs: BaseMessage[],
    maxTokens: number
  ): Promise<BaseMessage[]> {
    const context: BaseMessage[] = [];

    // 1. 系统提示（必须保留）
    const systemMessage = new (await import("@langchain/core/messages")).SystemMessage(
      systemPrompt
    );
    context.push(systemMessage);

    // 2. 选择相关的历史消息
    const relevantHistory = await this.selectRelevantContext(
      query,
      history,
      6
    );
    context.push(...relevantHistory);

    // 3. 选择相关的文档
    const relevantDocs = await this.selectRelevantContext(
      query,
      additionalDocs,
      3
    );
    context.push(...relevantDocs);

    // 4. 当前查询
    context.push(
      new (await import("@langchain/core/messages")).HumanMessage(query)
    );

    return context;
  }
}

// 使用示例
const selectiveBuilder = new SelectiveContextBuilder();
```

## 上下文隔离

### 用户上下文隔离

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { BaseMessage, HumanMessage, AIMessage, SystemMessage } from "@langchain/core/messages";

interface UserContext {
  userId: string;
  conversationHistory: BaseMessage[];
  userPreferences: Record<string, any>;
  sessionData: Record<string, any>;
  lastActive: number;
}

class ContextIsolationManager {
  private userContexts: Map<string, UserContext> = new Map();
  private globalContext: Record<string, any> = {};
  private maxContexts: number;
  private contextTTL: number;

  constructor(maxContexts: number = 1000, contextTTL: number = 3600000) {
    this.maxContexts = maxContexts;
    this.contextTTL = contextTTL;
  }

  // 获取或创建用户上下文
  getUserContext(userId: string): UserContext {
    if (!this.userContexts.has(userId)) {
      // 检查是否需要清理
      if (this.userContexts.size >= this.maxContexts) {
        this.evictOldContexts();
      }

      this.userContexts.set(userId, {
        userId,
        conversationHistory: [],
        userPreferences: {},
        sessionData: {},
        lastActive: Date.now(),
      });
    }

    const context = this.userContexts.get(userId)!;
    context.lastActive = Date.now();
    return context;
  }

  // 添加对话消息
  addMessage(userId: string, message: BaseMessage) {
    const context = this.getUserContext(userId);
    context.conversationHistory.push(message);
    context.lastActive = Date.now();
  }

  // 更新用户偏好
  updatePreferences(userId: string, preferences: Record<string, any>) {
    const context = this.getUserContext(userId);
    context.userPreferences = { ...context.userPreferences, ...preferences };
  }

  // 更新会话数据
  updateSessionData(userId: string, data: Record<string, any>) {
    const context = this.getUserContext(userId);
    context.sessionData = { ...context.sessionData, ...data };
  }

  // 获取隔离的上下文
  getIsolatedContext(userId: string, systemPrompt: string): BaseMessage[] {
    const context = this.getUserContext(userId);
    const messages: BaseMessage[] = [];

    // 系统提示
    const enrichedPrompt = this.enrichSystemPrompt(systemPrompt, context);
    messages.push(new SystemMessage(enrichedPrompt));

    // 用户对话历史
    messages.push(...context.conversationHistory);

    return messages;
  }

  private enrichSystemPrompt(basePrompt: string, userContext: UserContext): string {
    let prompt = basePrompt;

    // 添加用户偏好
    if (Object.keys(userContext.userPreferences).length > 0) {
      prompt += `\n\n用户偏好:\n${JSON.stringify(userContext.userPreferences, null, 2)}`;
    }

    // 添加会话数据
    if (Object.keys(userContext.sessionData).length > 0) {
      prompt += `\n\n当前会话数据:\n${JSON.stringify(userContext.sessionData, null, 2)}`;
    }

    return prompt;
  }

  // 清理过期上下文
  private evictOldContexts() {
    const now = Date.now();
    const expiredUserIds: string[] = [];

    for (const [userId, context] of this.userContexts) {
      if (now - context.lastActive > this.contextTTL) {
        expiredUserIds.push(userId);
      }
    }

    for (const userId of expiredUserIds) {
      this.userContexts.delete(userId);
    }

    // 如果仍然超过限制，删除最旧的
    if (this.userContexts.size >= this.maxContexts) {
      const sorted = Array.from(this.userContexts.entries()).sort(
        (a, b) => a[1].lastActive - b[1].lastActive
      );

      const toRemove = sorted.slice(0, sorted.length - this.maxContexts + 100);
      for (const [userId] of toRemove) {
        this.userContexts.delete(userId);
      }
    }
  }

  // 清除用户上下文
  clearUserContext(userId: string) {
    this.userContexts.delete(userId);
  }

  // 获取统计信息
  getStats() {
    return {
      activeUsers: this.userContexts.size,
      maxContexts: this.maxContexts,
    };
  }
}

// 使用示例
const isolationManager = new ContextIsolationManager();

// 用户 A
isolationManager.addMessage("user-a", new HumanMessage("我叫张三"));
isolationManager.addMessage("user-a", new AIMessage("你好，张三！"));
isolationManager.updatePreferences("user-a", { language: "zh", theme: "dark" });

// 用户 B
isolationManager.addMessage("user-b", new HumanMessage("我叫李四"));
isolationManager.addMessage("user-b", new AIMessage("你好，李四！"));
isolationManager.updatePreferences("user-b", { language: "en", theme: "light" });

// 获取隔离的上下文
const contextA = isolationManager.getIsolatedContext("user-a", "你是一个助手");
const contextB = isolationManager.getIsolatedContext("user-b", "你是一个助手");

console.log("用户 A 上下文:", contextA.length, "条消息");
console.log("用户 B 上下文:", contextB.length, "条消息");
console.log("统计:", isolationManager.getStats());
```

### 任务上下文隔离

```typescript
import { BaseMessage, SystemMessage } from "@langchain/core/messages";

interface TaskContext {
  taskId: string;
  taskType: string;
  description: string;
  constraints: string[];
  requiredTools: string[];
  context: Record<string, any>;
}

class TaskContextIsolation {
  private taskContexts: Map<string, TaskContext> = new Map();

  createTaskContext(task: TaskContext) {
    this.taskContexts.set(task.taskId, task);
  }

  getTaskContext(taskId: string): TaskContext | undefined {
    return this.taskContexts.get(taskId);
  }

  buildTaskSpecificContext(
    taskId: string,
    baseSystemPrompt: string
  ): SystemMessage | null {
    const task = this.taskContexts.get(taskId);
    if (!task) return null;

    let taskPrompt = baseSystemPrompt;

    taskPrompt += `\n\n## 当前任务`;
    taskPrompt += `\n- 类型: ${task.taskType}`;
    taskPrompt += `\n- 描述: ${task.description}`;

    if (task.constraints.length > 0) {
      taskPrompt += `\n- 约束条件:\n${task.constraints.map((c) => `  - ${c}`).join("\n")}`;
    }

    if (task.requiredTools.length > 0) {
      taskPrompt += `\n- 可用工具: ${task.requiredTools.join(", ")}`;
    }

    if (Object.keys(task.context).length > 0) {
      taskPrompt += `\n- 任务上下文:\n${JSON.stringify(task.context, null, 2)}`;
    }

    return new SystemMessage(taskPrompt);
  }

  completeTask(taskId: string) {
    this.taskContexts.delete(taskId);
  }

  getActiveTasks(): TaskContext[] {
    return Array.from(this.taskContexts.values());
  }
}

    "使用 JSDoc 格式",

  ],
  requiredTools: ["code_parser", "doc_generator"],
```

## 上下文优化

### 上下文优先级排序

```typescript
import { BaseMessage, SystemMessage, HumanMessage, AIMessage } from "@langchain/core/messages";

interface ContextItem {
  content: string;
  type: "system" | "history" | "document" | "instruction";
  priority: number; // 1-10, 10 最高
  tokens: number;
  timestamp?: number;
}

class ContextOptimizer {
  private tokenCounter: TokenCounter;

  constructor(model: string = "gpt-4") {
    this.tokenCounter = new TokenCounter(model);
  }

  optimizeContext(
    items: ContextItem[],
    maxTokens: number
  ): ContextItem[] {
    // 1. 按优先级排序
    const sorted = [...items].sort((a, b) => b.priority - a.priority);

    // 2. 贪心选择
    const selected: ContextItem[] = [];
    let usedTokens = 0;

    for (const item of sorted) {
      if (usedTokens + item.tokens <= maxTokens) {
        selected.push(item);
        usedTokens += item.tokens;
      }
    }

    // 3. 按原始顺序重新排列（保持上下文连贯性）
    const selectedSet = new Set(selected);
    return items.filter((item) => selectedSet.has(item));
  }

  createOptimizedMessages(
    items: ContextItem[],
    maxTokens: number
  ): BaseMessage[] {
    const optimized = this.optimizeContext(items, maxTokens);

    return optimized.map((item) => {
      switch (item.type) {
        case "system":
          return new SystemMessage(item.content);
        case "history":
          // 简化处理，实际需要更复杂的逻辑
          return new HumanMessage(item.content);
        default:
          return new SystemMessage(item.content);
      }
    });
  }

  destroy() {
    this.tokenCounter.free();
  }
}

// 使用示例
const optimizer = new ContextOptimizer();

const contextItems: ContextItem[] = [
  {
    content: "你是一个专业的编程助手",
    type: "system",
    priority: 10,
    tokens: 20,
  },
  {
    content: "用户之前问过关于 React 的问题",
    type: "history",
    priority: 5,
    tokens: 15,
  },
  {
    content: "TypeScript 是 JavaScript 的超集",
    type: "document",
    priority: 7,
    tokens: 12,
  },
  {
    content: "请用中文回答",
    type: "instruction",
    priority: 9,
    tokens: 8,
  },
];

const optimized = optimizer.optimizeContext(contextItems, 100);
console.log("优化后的上下文:", optimized.map((i) => i.content));

optimizer.destroy();
```

### 上下文缓存

```typescript
import { createHash } from "crypto";

interface CachedContext {
  key: string;
  messages: any[];
  timestamp: number;
  hitCount: number;
}

class ContextCache {
  private cache: Map<string, CachedContext> = new Map();
  private maxSize: number;
  private ttl: number;

  constructor(maxSize: number = 100, ttl: number = 300000) {
    this.maxSize = maxSize;
    this.ttl = ttl;
  }

  private generateKey(context: any): string {
    const hash = createHash("md5");
    hash.update(JSON.stringify(context));
    return hash.digest("hex");
  }

  get(context: any): any[] | null {
    const key = this.generateKey(context);
    const cached = this.cache.get(key);

    if (!cached) {
      return null;
    }

    // 检查是否过期
    if (Date.now() - cached.timestamp > this.ttl) {
      this.cache.delete(key);
      return null;
    }

    // 更新命中次数
    cached.hitCount++;
    return cached.messages;
  }

  set(context: any, messages: any[]) {
    const key = this.generateKey(context);

    // 检查缓存大小
    if (this.cache.size >= this.maxSize) {
      this.evict();
    }

    this.cache.set(key, {
      key,
      messages,
      timestamp: Date.now(),
      hitCount: 0,
    });
  }

  private evict() {
    // LRU 策略：删除最旧和最少使用的
    const entries = Array.from(this.cache.entries());

    entries.sort((a, b) => {
      // 综合考虑时间和命中次数
      const scoreA = a[1].hitCount / (Date.now() - a[1].timestamp);
      const scoreB = b[1].hitCount / (Date.now() - b[1].timestamp);
      return scoreA - scoreB;
    });

    // 删除得分最低的 20%
    const toRemove = Math.ceil(entries.length * 0.2);
    for (let i = 0; i < toRemove; i++) {
      this.cache.delete(entries[i][0]);
    }
  }

  clear() {
    this.cache.clear();
  }

  getStats() {
    return {
      size: this.cache.size,
      maxSize: this.maxSize,
      entries: Array.from(this.cache.values()).map((e) => ({
        key: e.key.substring(0, 8) + "...",
        hitCount: e.hitCount,
        age: Date.now() - e.timestamp,
      })),
    };
  }
}

// 使用示例
const contextCache = new ContextCache(50, 600000); // 50 个条目，10 分钟 TTL

// 缓存上下文
const context1 = { userId: "user1", topic: "TypeScript" };
const messages1 = [
  { role: "system", content: "你是一个 TypeScript 专家" },
  { role: "user", content: "什么是泛型？" },
];

contextCache.set(context1, messages1);

// 命中缓存
const cached = contextCache.get(context1);
console.log("缓存命中:", cached !== null);

// 未命中缓存
const context2 = { userId: "user1", topic: "React" };
const cached2 = contextCache.get(context2);
console.log("缓存命中:", cached2 !== null);

console.log("缓存统计:", contextCache.getStats());
```

## 最佳实践

1. **了解模型限制**：清楚所用模型的上下文窗口大小
2. **合理分配上下文**：为系统提示、历史、文档等预留合理的 token 预算
3. **实现压缩策略**：对长对话使用摘要或选择性上下文
4. **隔离用户上下文**：多用户场景下确保上下文隔离
5. **缓存优化**：对重复的上下文构建进行缓存
6. **监控 token 使用**：记录和监控 token 消耗，优化成本
7. **渐进式加载**：按需加载上下文，避免一次性加载过多内容

> **提示**：上下文工程是构建高质量 AI 应用的核心技能。投入时间理解和优化上下文管理，可以显著提升应用的性能和用户体验。

> **建议**：在开发初期就建立上下文管理策略，而不是在应用成熟后再重构。这可以避免后期的技术债务。

> **注意**：不同模型的上下文窗口和 token 计算方式可能不同，在切换模型时需要重新评估上下文策略。

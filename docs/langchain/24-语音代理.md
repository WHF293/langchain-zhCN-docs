## 语音识别

语音识别（Speech Recognition）是将人类语音转换为文本的技术。LangChain 提供了与多种语音识别服务的集成能力。

### 支持的语音识别服务

| 服务 | 特点 | 适用场景 |
|------|------|----------|
| OpenAI Whisper | 高精度，多语言支持 | 通用语音识别 |
| Google Cloud Speech | 实时流式识别 | 实时对话场景 |
| Azure Speech Services | 企业级稳定性 | 生产环境部署 |
| Deepgram | 低延迟，高性价比 | 大规模应用 |

### OpenAI Whisper 集成

```typescript
// 使用 OpenAI Whisper 进行语音识别
import OpenAI from "openai";
import fs from "fs";

const openai = new OpenAI();

async function transcribeAudio(audioFilePath: string): Promise<string> {
  // 读取音频文件
  const audioFile = fs.createReadStream(audioFilePath);

  // 调用 Whisper API
  const transcription = await openai.audio.transcriptions.create({
    file: audioFile,
    model: "whisper-1",
    language: "zh",  // 指定中文
    response_format: "text",
  });

  return transcription;
}

// 使用示例
const text = await transcribeAudio("./audio/recording.mp3");
console.log("识别结果:", text);
```

### 流式语音识别

```typescript
// 实时流式语音识别
import { Recording } from "@anthropic/recording";

async function streamTranscription() {
  const recording = new Recording({
    sampleRate: 16000,
    channels: 1,
  });

  recording.on("data", async (audioChunk) => {
    // 实时发送音频数据进行识别
    const result = await recognizeChunk(audioChunk);
    console.log("实时识别:", result);
  });

  recording.start();
}

// 分块识别函数
async function recognizeChunk(chunk: Buffer): Promise<string> {
  // 将音频块发送到识别服务
  const response = await fetch("/api/transcribe", {
    method: "POST",
    body: chunk,
    headers: {
      "Content-Type": "audio/raw",
    },
  });

  return response.text();
}
```

> **提示**：流式识别适合实时对话场景，可以显著降低用户感知的响应延迟。

## 语音合成

语音合成（Text-to-Speech, TTS）是将文本转换为自然语音的技术。

### 支持的语音合成服务

| 服务 | 声音质量 | 延迟 | 特点 |
|------|----------|------|------|
| OpenAI TTS | 高 | 低 | 多种声音选择 |
| ElevenLabs | 极高 | 中 | 声音克隆 |
| Google Cloud TTS | 高 | 低 | WaveNet 技术 |
| Azure TTS | 高 | 中 | 神经网络语音 |

### OpenAI TTS 集成

```typescript
// 使用 OpenAI 进行语音合成
import OpenAI from "openai";
import fs from "fs";

const openai = new OpenAI();

async function synthesizeSpeech(
  text: string,
  outputPath: string
): Promise<void> {
  // 调用 TTS API
  const response = await openai.audio.speech.create({
    model: "tts-1-hd",  // 高清模型
    voice: "nova",       // 声音选择
    input: text,
    response_format: "mp3",
    speed: 1.0,          // 语速控制
  });

  // 保存音频文件
  const buffer = Buffer.from(await response.arrayBuffer());
  fs.writeFileSync(outputPath, buffer);
}

// 使用示例
await synthesizeSpeech(
  "你好，我是你的AI助手，有什么可以帮助你的吗？",
  "./output/speech.mp3"
);
```

## 语音交互

语音交互（Voice Interaction）是将语音识别和语音合成结合，实现完整的语音对话体验。

### 语音对话流程

```
用户语音 → 语音识别 → 文本处理 → LLM 响应 → 语音合成 → 语音播放
```

### 完整语音助手实现

```typescript
// 语音助手完整实现
import { ChatOpenAI } from "langchain/chat_models/openai";
import { ConversationChain } from "langchain/chains";
import { BufferMemory } from "langchain/memory";

class VoiceAssistant {
  private llm: ChatOpenAI;
  private chain: ConversationChain;
  private memory: BufferMemory;

  constructor() {
    this.llm = new ChatOpenAI({
      modelName: "gpt-4",
      temperature: 0.7,
    });

    this.memory = new BufferMemory();

    this.chain = new ConversationChain({
      llm: this.llm,
      memory: this.memory,
    });
  }

  // 处理语音输入
  async processVoiceInput(audioPath: string): Promise<Buffer> {
    // 1. 语音识别
    const userText = await transcribeAudio(audioPath);
    console.log("用户说:", userText);

    // 2. 获取 AI 响应
    const response = await this.chain.call({
      input: userText,
    });
    console.log("AI 回复:", response.response);

    // 3. 语音合成
    const audioBuffer = await this.textToSpeech(response.response);

    return audioBuffer;
  }

  // 文本转语音
  private async textToSpeech(text: string): Promise<Buffer> {
    const openai = new OpenAI();
    const response = await openai.audio.speech.create({
      model: "tts-1",
      voice: "nova",
      input: text,
    });

    return Buffer.from(await response.arrayBuffer());
  }
}

// 使用示例
const assistant = new VoiceAssistant();
const responseAudio = await assistant.processVoiceInput("./user_input.mp3");
```

### 多轮对话管理

```typescript
// 带上下文的多轮对话
interface ConversationContext {
  sessionId: string;
  history: Message[];
  userPreferences: UserPreferences;
}

interface Message {
  role: "user" | "assistant";
  content: string;
  timestamp: Date;
  audioUrl?: string;
}

class MultiTurnVoiceChat {
  private sessions: Map<string, ConversationContext> = new Map();

  async chat(
    sessionId: string,
    audioInput: Buffer
  ): Promise<{ text: string; audio: Buffer }> {
    // 获取或创建会话
    let context = this.sessions.get(sessionId);
    if (!context) {
      context = {
        sessionId,
        history: [],
        userPreferences: await this.loadPreferences(sessionId),
      };
      this.sessions.set(sessionId, context);
    }

    // 识别语音
    const userText = await transcribeAudio(audioInput);

    // 添加到历史
    context.history.push({
      role: "user",
      content: userText,
      timestamp: new Date(),
    });

    // 生成响应（带上下文）
    const response = await this.generateResponse(context);

    // 合成语音
    const audio = await this.textToSpeech(response);

    return { text: response, audio };
  }

  private async generateResponse(
    context: ConversationContext
  ): Promise<string> {
    // 使用历史上下文生成响应
    const messages = context.history.map((msg) => ({
      role: msg.role,
      content: msg.content,
    }));

    const llm = new ChatOpenAI({ modelName: "gpt-4" });
    const response = await llm.invoke(messages);

    return response.content as string;
  }
}
```

> **注意**：多轮对话需要妥善管理上下文长度，避免超出模型的 token 限制。

## 语音安全

语音安全（Voice Security）是确保语音交互系统安全可靠的重要环节。

### 安全威胁与防护

| 威胁类型 | 说明 | 防护措施 |
|----------|------|----------|
| 语音伪造 | AI 生成的假语音 | 声纹验证，活体检测 |
| 录音重放 | 预录制的语音攻击 | 随机挑战，时间戳验证 |
| 隐私泄露 | 语音数据被窃取 | 加密存储，访问控制 |
| 内容注入 | 恶意语音指令 | 输入验证，权限控制 |

### 声纹验证

```typescript
// 声纹验证实现
interface Voiceprint {
  userId: string;
  embedding: number[];
  createdAt: Date;
}

class VoiceprintAuth {
  private voiceprints: Map<string, Voiceprint> = new Map();

  // 注册声纹
  async register(
    userId: string,
    audioSample: Buffer
  ): Promise<void> {
    // 提取声纹特征
    const embedding = await this.extractVoiceprint(audioSample);

    // 存储声纹
    this.voiceprints.set(userId, {
      userId,
      embedding,
      createdAt: new Date(),
    });
  }

  // 验证声纹
  async verify(
    userId: string,
    audioSample: Buffer
  ): Promise<{ match: boolean; confidence: number }> {
    const stored = this.voiceprints.get(userId);
    if (!stored) {
      throw new Error("用户声纹未注册");
    }

    // 提取当前声纹
    const currentEmbedding = await this.extractVoiceprint(audioSample);

    // 计算相似度
    const similarity = this.cosineSimilarity(
      stored.embedding,
      currentEmbedding
    );

    return {
      match: similarity > 0.85,  // 相似度阈值
      confidence: similarity,
    };
  }

  // 计算余弦相似度
  private cosineSimilarity(a: number[], b: number[]): number {
    const dotProduct = a.reduce((sum, val, i) => sum + val * b[i], 0);
    const normA = Math.sqrt(a.reduce((sum, val) => sum + val * val, 0));
    const normB = Math.sqrt(b.reduce((sum, val) => sum + val * val, 0));
    return dotProduct / (normA * normB);
  }

  // 提取声纹特征（调用外部服务）
  private async extractVoiceprint(audio: Buffer): Promise<number[]> {
    // 调用声纹提取 API
    const response = await fetch("/api/voiceprint/extract", {
      method: "POST",
      body: audio,
    });
    return response.json();
  }
}
```

### 语音内容过滤

```typescript
// 语音内容安全过滤
class VoiceContentFilter {
  private sensitivePatterns: RegExp[] = [
    /密码|password/i,
    /银行卡|credit card/i,
    /身份证|social security/i,
  ];

  // 检查语音内容是否包含敏感信息
  async checkContent(text: string): Promise<{
    safe: boolean;
    issues: string[];
  }> {
    const issues: string[] = [];

    // 检查敏感词
    for (const pattern of this.sensitivePatterns) {
      if (pattern.test(text)) {
        issues.push(`包含敏感信息: ${pattern.source}`);
      }
    }

    // 检查语音长度（防止异常输入）
    if (text.length > 1000) {
      issues.push("语音内容过长");
    }

    return {
      safe: issues.length === 0,
      issues,
    };
  }

  // 脱敏处理
  maskSensitiveInfo(text: string): string {
    let masked = text;
    masked = masked.replace(/\d{16,19}/g, "****");  // 银行卡号
    masked = masked.replace(/\d{17}[\dX]/gi, "****");  // 身份证号
    return masked;
  }
}
```

> **重要**：在处理用户语音数据时，务必遵守相关隐私法规（如 GDPR、个人信息保护法），确保数据安全。

### 加密存储

```typescript
// 语音数据加密存储
import crypto from "crypto";

class SecureVoiceStorage {
  private algorithm = "aes-256-gcm";
  private key: Buffer;

  constructor(encryptionKey: string) {
    this.key = Buffer.from(encryptionKey, "hex");
  }

  // 加密音频数据
  encrypt(audioData: Buffer): {
    encrypted: Buffer;
    iv: Buffer;
    tag: Buffer;
  } {
    const iv = crypto.randomBytes(16);
    const cipher = crypto.createCipheriv(this.algorithm, this.key, iv);

    const encrypted = Buffer.concat([
      cipher.update(audioData),
      cipher.final(),
    ]);

    const tag = cipher.getAuthTag();

    return { encrypted, iv, tag };
  }

  // 解密音频数据
  decrypt(
    encrypted: Buffer,
    iv: Buffer,
    tag: Buffer
  ): Buffer {
    const decipher = crypto.createDecipheriv(
      this.algorithm,
      this.key,
      iv
    );
    decipher.setAuthTag(tag);

    return Buffer.concat([
      decipher.update(encrypted),
      decipher.final(),
    ]);
  }
}
```

> **提示**：加密密钥应使用安全的方式管理，如使用密钥管理服务（KMS）或硬件安全模块（HSM）。

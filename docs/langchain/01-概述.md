# 第1章 LangChain 概述

> 来源: [LangChain 官方文档](https://python.langchain.com/)

---

## LangChain 是什么

LangChain 是一个用于构建大语言模型（LLM）应用的开发框架。它提供了一套工具、组件和接口，帮助开发者将 LLM 与外部数据源、工具和服务连接起来，从而构建强大的 AI 应用。

### 核心价值

| 价值 | 说明 |
|:---|:---|
| **标准化接口** | 统一不同 LLM 提供商的调用方式 |
| **组件化设计** | 模块化的构建块，灵活组合 |
| **链式编排** | 将多个操作串联成复杂的工作流 |
| **工具集成** | 轻松连接外部 API、数据库、文件系统 |
| **记忆管理** | 为对话和任务维护上下文状态 |

### 适用场景

LangChain 适用于以下类型的应用开发：

- **对话式 AI**：智能客服、聊天机器人、虚拟助手
- **文档问答**：基于文档的检索增强生成（RAG）
- **内容生成**：文章写作、代码生成、摘要提取
- **数据分析**：自然语言查询数据库、数据可视化
- **任务自动化**：多步骤工作流、决策树、规划系统

## 核心概念

LangChain 的设计围绕五个核心概念展开：

### 1. 模型（Models）

模型是 LangChain 的基础构建块，提供与 LLM 交互的统一接口。

```typescript
import { ChatOpenAI } from "@langchain/openai";

// 创建模型实例
const model = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0.7,
  maxTokens: 1000,
});

// 调用模型
const response = await model.invoke("你好，请介绍一下自己");
console.log(response.content);
```

### 2. 提示（Prompts）

提示模板用于动态构建发送给模型的输入。

```typescript
import { ChatPromptTemplate } from "@langchain/core/prompts";

// 创建提示模板
const prompt = ChatPromptTemplate.fromMessages([
  ["system", "你是一个专业的{role}"],
  ["user", "{input}"],
]);

// 格式化提示
const messages = await prompt.invoke({
  role: "AI 助手",
  input: "请解释什么是机器学习",
});
```

### 3. 链（Chains）

链将多个组件串联起来，形成完整的处理流程。

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatPromptTemplate } from "@langchain/core/prompts";
import { StringOutputParser } from "@langchain/core/output_parsers";

// 创建组件
const model = new ChatOpenAI({ modelName: "gpt-4" });
const prompt = ChatPromptTemplate.fromMessages([
  ["system", "你是一个专业的翻译员"],
  ["user", "将以下文本翻译成{language}：{text}"],
]);

// 组合成链
const chain = prompt.pipe(model).pipe(new StringOutputParser());

// 执行链
const result = await chain.invoke({
  language: "英语",
  text: "你好，世界！",
});
console.log(result); // "Hello, World!"
```

### 4. 代理（Agents）

代理让模型能够自主决定使用哪些工具以及如何完成任务。

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { createReactAgent } from "@langchain/langgraph/prebuilt";
import { tool } from "@langchain/core/tools";
import { z } from "zod";

// 定义工具
const searchTool = tool(
  async ({ query }) => {
    // 模拟搜索
    return `搜索结果：关于 "${query}" 的信息`;
  },
  {
    name: "search",
    description: "搜索互联网获取信息",
    schema: z.object({
      query: z.string().describe("搜索关键词"),
    }),
  }
);

// 创建代理
const model = new ChatOpenAI({ modelName: "gpt-4" });
const agent = createReactAgent({
  llm: model,
  tools: [searchTool],
});

// 使用代理
const result = await agent.invoke({
  messages: [{ role: "user", content: "今天北京天气怎么样？" }],
});
```

### 5. 记忆（Memory）

记忆组件帮助应用维护对话历史和上下文状态。

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatPromptTemplate } from "@langchain/core/prompts";
import { StringOutputParser } from "@langchain/core/output_parsers";

// 创建带记忆的对话
const model = new ChatOpenAI({ modelName: "gpt-4" });
const prompt = ChatPromptTemplate.fromMessages([
  ["system", "你是一个友好的助手"],
  ["placeholder", "{history}"],  // 历史消息占位符
  ["user", "{input}"],
]);

const chain = prompt.pipe(model).pipe(new StringOutputParser());

// 模拟多轮对话
const history: Array<{ role: string; content: string }> = [];

async function chat(input: string) {
  const result = await chain.invoke({
    history,
    input,
  });

  // 更新历史
  history.push({ role: "user", content: input });
  history.push({ role: "assistant", content: result });

  return result;
}

await chat("我叫小明");
await chat("我的名字是什么？"); // 模型会记得是"小明"
```

## 与 LangGraph 的关系

LangChain 和 LangGraph 是互补的关系：

| 特性 | LangChain | LangGraph |
|:---|:---|:---|
| **定位** | 组件库和基础工具 | 状态机和工作流编排 |
| **适用场景** | 简单的链式调用 | 复杂的有状态应用 |
| **核心抽象** | Chain（链） | Graph（图） |
| **状态管理** | 有限 | 内置持久化状态 |
| **循环支持** | 不支持 | 原生支持 |
| **人机交互** | 基础 | 内置中断和恢复 |

```typescript
// LangChain：简单的线性链
const chain = prompt.pipe(model).pipe(parser);

// LangGraph：复杂的有状态图
import { StateGraph } from "@langchain/langgraph";

const graph = new StateGraph(StateAnnotation)
  .addNode("agent", agentNode)
  .addNode("tools", toolsNode)
  .addEdge("__start__", "agent")
  .addConditionalEdges("agent", shouldContinue)
  .addEdge("tools", "agent")
  .compile();
```

> **建议**：对于简单的 LLM 调用，使用 LangChain 即可；对于需要复杂状态管理、循环、人机交互的应用，建议使用 LangGraph。

## 生态系统

LangChain 拥有丰富的生态系统：

| 组件 | 说明 |
|:---|:---|
| **@langchain/core** | 核心接口和基础类 |
| **@langchain/langgraph** | 状态图编排框架 |
| **@langchain/openai** | OpenAI 集成 |
| **@langchain/community** | 社区集成包 |
| **LangSmith** | 可观测性和调试平台 |
| **LangServe** | 部署为 REST API |

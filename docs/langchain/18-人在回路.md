## 概述

人在回路（Human-in-the-Loop，简称 HITL）是一种设计模式，允许人工在 AI 系统执行过程中进行干预、审批或提供反馈。这种模式在需要高可靠性、合规性或复杂决策的场景中尤为重要。

### 为什么需要人在回路

| 场景 | 原因 | 示例 |
|------|------|------|
| 高风险操作 | 防止错误造成严重后果 | 删除数据、发送邮件 |
| 合规要求 | 法律或政策要求人工审批 | 金融交易、医疗诊断 |
| 质量控制 | 确保输出质量 | 内容生成、代码审查 |
| 学习改进 | 收集人工反馈优化系统 | 标注数据、纠正错误 |

## 人工审批

人工审批（Human Approval）是最常见的 HITL 模式，在执行关键操作前请求人工确认。

### 基本审批流程

```typescript
function createApprovalMiddleware(options: {
  // 需要审批的操作列表
  criticalActions: string[];

  return RunnableLambda.from(
    async (input: any, config?: RunnableConfig) => {

      return input;
    }
  );
```

### 条件审批

```typescript
import { RunnableLambda, RunnableBranch } from "@langchain/core/runnables";

// 创建条件审批逻辑
const conditionalApproval = RunnableBranch.from([
  // 条件1：高风险操作需要审批
  [
    (input) => input.riskLevel === "high",
    RunnableLambda.from(async (input) => {
      const approved = await requestHumanApproval(input);
      if (!approved) throw new Error("高风险操作被拒绝");
      return input;
    })
  ],

  // 条件2：中等风险操作可能需要审批
  [
    (input) => input.riskLevel === "medium" && input.amount > 1000,
    RunnableLambda.from(async (input) => {
      const approved = await requestHumanApproval(input);
      if (!approved) throw new Error("中等风险操作被拒绝");
      return input;
    })
  ],

  // 默认：低风险操作自动通过
  RunnableLambda.from((input) => {
    console.log("低风险操作，自动通过");
    return input;
  })
]);

// 使用
const result = await conditionalApproval.invoke({
  action: "transfer_money",
  amount: 5000,
  riskLevel: "high"
});
```

## 人工干预

人工干预（Human Intervention）允许人工在 AI 执行过程中进行干预，纠正方向或提供额外信息。

### 执行中断

```typescript
import { RunnableLambda } from "@langchain/core/runnables";

// 创建可中断的执行器
const interruptibleExecutor = RunnableLambda.from(
  async (input: any, config?: RunnableConfig) => {
    const executionId = config?.configurable?.executionId;
    let currentStep = 0;
    const steps = input.steps;

    for (const step of steps) {
      currentStep++;

      // 检查是否需要中断
      const shouldInterrupt = await checkInterruption({
        executionId,
        step: currentStep,
        result: step
      });

      if (shouldInterrupt) {
        // 保存当前状态
        const state = {
          executionId,
          currentStep,
          results: steps.slice(0, currentStep),
          status: "interrupted"
        };

        await saveExecutionState(state);

        // 等待人工干预
        const intervention = await waitForIntervention(executionId);

        // 应用干预
        if (intervention.type === "abort") {
          return { status: "aborted", reason: intervention.reason };
        } else if (intervention.type === "modify") {
          // 修改后续步骤
          steps.splice(currentStep, ...intervention.newSteps);
        }
      }

      // 执行当前步骤
      const result = await executeStep(step);
      steps[currentStep - 1] = result;
    }

    return { status: "completed", results: steps };
  }
);

// 检查是否需要中断
async function checkInterruption(context: {
  executionId: string;
  step: number;
  result: any;
}): Promise<boolean> {
  // 根据业务逻辑决定是否中断
  // 例如：高风险步骤、需要确认的步骤等

  const riskySteps = [3, 5, 7]; // 假设这些步骤是高风险的

  if (riskySteps.includes(context.step)) {
    return true;
  }

  return false;
}
```

### 实时反馈

```typescript
import { EventEmitter } from "events";

// 创建反馈收集器
class FeedbackCollector extends EventEmitter {
  private feedbacks: Map<string, any> = new Map();

  // 注册反馈
  registerFeedback(executionId: string, step: number, feedback: any) {
    const key = `${executionId}-${step}`;
    this.feedbacks.set(key, feedback);
    this.emit("feedback", { executionId, step, feedback });
  }

  // 获取反馈
  getFeedback(executionId: string, step: number): any {
    const key = `${executionId}-${step}`;
    return this.feedbacks.get(key);
  }

  // 等待反馈
  async waitForFeedback(
    executionId: string,
    step: number,
    timeout: number = 60000
  ): Promise<any> {
    const key = `${executionId}-${step}`;

    // 检查是否已有反馈
    if (this.feedbacks.has(key)) {
      return this.feedbacks.get(key);
    }

    // 等待反馈
    return new Promise((resolve, reject) => {
      const timer = setTimeout(() => {
        reject(new Error("等待反馈超时"));
      }, timeout);

      this.once("feedback", (data) => {
        if (data.executionId === executionId && data.step === step) {
          clearTimeout(timer);
          resolve(data.feedback);
        }
      });
    });
  }
}

// 使用反馈收集器
const feedbackCollector = new FeedbackCollector();

const chainWithFeedback = RunnableLambda.from(
  async (input: any, config?: RunnableConfig) => {
    const executionId = config?.configurable?.executionId;

    // 执行第一步
    const step1Result = await executeStep1(input);

    // 收集反馈
    const feedback1 = await feedbackCollector.waitForFeedback(
      executionId,
      1
    );

    // 根据反馈调整
    const adjustedInput = applyFeedback(input, feedback1);

    // 执行第二步
    const step2Result = await executeStep2(adjustedInput);

    return { step1: step1Result, step2: step2Result };
  }
);
```

## 人工反馈

人工反馈（Human Feedback）用于收集人工对 AI 输出的评价，以改进系统性能。

### 反馈类型

| 反馈类型 | 说明 | 用途 |
|---------|------|------|
| 点赞/点踩 | 简单的好坏评价 | 快速质量评估 |
| 评分 | 数值评分（如1-5分） | 量化质量评估 |
| 文本反馈 | 详细的文字说明 | 深入问题分析 |
| 标注 | 标记具体问题 | 训练数据收集 |

### 收集反馈

```typescript

const feedbackHandler = RunnableLambda.from(
  async (input: {
    responseId: string;
    feedback: z.infer<typeof feedbackSchema>;
  }) => {
    // 验证反馈

    // 保存反馈
    await saveFeedback({
      responseId: input.responseId,
      ...validatedFeedback,
      timestamp: Date.now()

    // 分析反馈

    // 如果评分低，触发改进流程

      await triggerImprovementProcess({
        responseId: input.responseId,
```

### 反馈学习

```typescript
import { RunnableLambda } from "@langchain/core/runnables";

// 创建反馈学习系统
class FeedbackLearner {
  private feedbackStore: Map<string, any[]> = new Map();

  // 收集反馈
  async collectFeedback(responseId: string, feedback: any) {
    const key = feedback.category || "general";

    if (!this.feedbackStore.has(key)) {
      this.feedbackStore.set(key, []);
    }

    this.feedbackStore.get(key)!.push({
      responseId,
      ...feedback,
      timestamp: Date.now()
    });
  }

  // 分析反馈模式
  async analyzePatterns(): Promise<any> {
    const patterns: any = {};

    for (const [category, feedbacks] of this.feedbackStore) {
      patterns[category] = {
        count: feedbacks.length,
        averageRating: this.calculateAverageRating(feedbacks),
        commonIssues: this.findCommonIssues(feedbacks),
        improvementSuggestions: this.generateSuggestions(feedbacks)
      };
    }

    return patterns;
  }

  // 生成改进建议
  async generateImprovements(): Promise<any> {
    const patterns = await this.analyzePatterns();
    const improvements: any[] = [];

    for (const [category, data] of Object.entries(patterns)) {
      if ((data as any).averageRating < 3.5) {
        improvements.push({
          category,
          priority: "high",
          suggestion: (data as any).improvementSuggestions
        });
      }
    }

    return improvements;
  }

  private calculateAverageRating(feedbacks: any[]): number {
    const ratings = feedbacks
      .map(f => f.rating)
      .filter(r => r !== undefined);

    if (ratings.length === 0) return 0;

    return ratings.reduce((a, b) => a + b, 0) / ratings.length;
  }

  private findCommonIssues(feedbacks: any[]): string[] {
    const issueCount: Map<string, number> = new Map();

    feedbacks.forEach(f => {
      if (f.issues) {
        f.issues.forEach((issue: string) => {
          issueCount.set(issue, (issueCount.get(issue) || 0) + 1);
        });
      }
    });

    return Array.from(issueCount.entries())
      .sort((a, b) => b[1] - a[1])
      .slice(0, 5)
      .map(([issue]) => issue);
  }

  private generateSuggestions(feedbacks: any[]): string[] {
    // 基于反馈生成改进建议
    const suggestions: string[] = [];

    const commonIssues = this.findCommonIssues(feedbacks);

    commonIssues.forEach(issue => {
      suggestions.push(`改进: ${issue}`);
    });

    return suggestions;
  }
}

// 使用反馈学习系统
const feedbackLearner = new FeedbackLearner();

const learningChain = RunnableLambda.from(
  async (input: any) => {
    // 收集反馈
    await feedbackLearner.collectFeedback(input.responseId, input.feedback);

    // 定期分析和改进
    const improvements = await feedbackLearner.generateImprovements();

    return {
      status: "feedback_processed",
      improvements
    };
  }
);
```

## 最佳实践

1. **明确审批流程**：定义清晰的审批规则和流程
2. **异步处理**：使用异步方式处理人工审批，避免阻塞
3. **超时机制**：设置合理的超时时间，避免无限等待
4. **反馈闭环**：将人工反馈用于系统改进
5. **审计日志**：记录所有人工干预操作，便于追溯

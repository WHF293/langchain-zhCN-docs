# 第6章 模型

> 来源: [LangChain 官方文档](https://python.langchain.com/)

---

模型是 LangChain 的核心组件，提供了与大语言模型（LLM）交互的统一接口。

## 模型接口

LangChain 提供两种模型接口：

### ChatModel（聊天模型）

推荐使用的模型接口，支持消息列表输入：

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { HumanMessage, SystemMessage } from "@langchain/core/messages";

// 创建聊天模型
const model = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0.7,
  maxTokens: 1000,
});

// 使用消息列表调用
const response = await model.invoke([
  new SystemMessage("你是一个有帮助的助手"),
  new HumanMessage("你好！"),
]);

console.log(response.content);
```

### LLM（文本补全模型）

传统的文本补全接口：

```typescript
import { OpenAI } from "@langchain/openai";

// 创建 LLM 模型
const llm = new OpenAI({
  modelName: "gpt-3.5-turbo-instruct",
  temperature: 0.7,
  maxTokens: 500,
});

// 直接传入文本
const response = await llm.invoke("写一首关于春天的诗：");
console.log(response);
```

### ChatModel vs LLM

| 特性 | ChatModel | LLM |
|:---|:---|:---|
| **输入格式** | 消息列表 | 字符串 |
| **输出格式** | `AIMessage` 对象 | 字符串 |
| **系统提示** | 原生支持 | 需要手动拼接 |
| **多轮对话** | 原生支持 | 需要手动管理 |
| **推荐程度** | 推荐 | 旧版兼容 |

> **建议**：优先使用 ChatModel，它提供了更好的类型安全和更丰富的功能。

## 模型参数

### 通用参数

| 参数 | 类型 | 说明 | 默认值 |
|:---|:---|:---|:---:|
| `modelName` | string | 模型名称 | - |
| `temperature` | number | 采样温度 (0-2) | 1 |
| `maxTokens` | number | 最大输出 token 数 | - |
| `topP` | number | 核采样参数 | 1 |
| `timeout` | number | 请求超时（毫秒） | - |
| `maxRetries` | number | 最大重试次数 | 2 |
| `verbose` | boolean | 详细日志 | false |

### temperature 参数

```typescript
import { ChatOpenAI } from "@langchain/openai";

// 低温度：更确定性的输出
const precise = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0,  // 最确定性
});

// 中温度：平衡创造性
const balanced = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0.7,  // 平衡
});

// 高温度：更有创造性
const creative = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 1.5,  // 高创造性
});
```

| temperature | 特点 | 适用场景 |
|:---:|:---|:---|
| 0 | 最确定性 | 代码生成、分类、提取 |
| 0.3 | 较确定 | 问答、摘要 |
| 0.7 | 平衡 | 一般对话、写作 |
| 1.0 | 较随机 | 创意写作 |
| 1.5+ | 非常随机 | 头脑风暴 |

## 模型缓存

缓存可以避免重复调用相同的请求，节省成本和时间。

### 内存缓存

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { InMemoryCache } from "@langchain/core/caches";

// 创建缓存
const cache = new InMemoryCache();

// 应用缓存
const model = new ChatOpenAI({
  modelName: "gpt-4",
  cache,
});

// 相同的请求会返回缓存结果
const result1 = await model.invoke("你好");
const result2 = await model.invoke("你好"); // 从缓存返回
```

### Redis 缓存

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { RedisCache } from "@langchain/community/caches/ioredis";
import Redis from "ioredis";

// 创建 Redis 客户端
const client = new Redis({
  host: "localhost",
  port: 6379,
});

// 创建 Redis 缓存
const cache = new RedisCache(client);

const model = new ChatOpenAI({
  modelName: "gpt-4",
  cache,
});
```

## 模型回退

当主模型不可用时，自动切换到备用模型。

### 基础回退

```typescript
import { ChatOpenAI } from "@langchain/openai";

// 主模型
const primaryModel = new ChatOpenAI({
  modelName: "gpt-4",
  maxRetries: 0,  // 不重试，直接回退
});

// 备用模型
const fallbackModel = new ChatOpenAI({
  modelName: "gpt-3.5-turbo",
});

// 配置回退
const modelWithFallback = primaryModel.withFallbacks([fallbackModel]);

// 使用
const result = await modelWithFallback.invoke("你好");
```

### 多级回退

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatAnthropic } from "@langchain/anthropic";

const gpt4 = new ChatOpenAI({ modelName: "gpt-4", maxRetries: 0 });
const gpt35 = new ChatOpenAI({ modelName: "gpt-3.5-turbo", maxRetries: 0 });
const claude = new ChatAnthropic({ model: "claude-3-sonnet" });

// 多级回退链
const model = gpt4.withFallbacks([gpt35, claude]);

// 尝试顺序：gpt-4 → gpt-3.5-turbo → claude-3-sonnet
```

### 带重试的回退

```typescript
import { ChatOpenAI } from "@langchain/openai";

const model = new ChatOpenAI({
  modelName: "gpt-4",
  maxRetries: 2,
  retryDelay: 1000,  // 重试延迟 1 秒
});

// 添加回退
const modelWithFallback = model.withFallbacks([
  new ChatOpenAI({ modelName: "gpt-3.5-turbo" }),
]);
```

## 绑定参数

使用 `bind()` 方法为模型绑定默认参数：

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatPromptTemplate } from "@langchain/core/prompts";

const model = new ChatOpenAI({ modelName: "gpt-4" });

// 绑定工具
const modelWithTools = model.bind({
  tools: [
    {
      type: "function",
      function: {
        name: "get_weather",
        description: "获取天气",
        parameters: {
          type: "object",
          properties: {
            city: { type: "string" },
          },
        },
      },
    },
  ],
});

// 绑定 JSON 模式
const jsonModel = model.bind({
  response_format: { type: "json_object" },
});

// 绑定停止词
const modelWithStop = model.bind({
  stop: ["\n"],
});
```

## 结构化输出

让模型输出符合特定结构的数据：

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { z } from "zod";

// 定义输出结构
const schema = z.object({
  name: z.string().describe("人名"),
  age: z.number().describe("年龄"),
  skills: z.array(z.string()).describe("技能列表"),
});

// 创建带结构化输出的模型
const model = new ChatOpenAI({ modelName: "gpt-4" }).withStructuredOutput(schema);

// 调用
const result = await model.invoke("提取信息：张三，25岁，会 TypeScript 和 Python");

console.log(result);
// { name: "张三", age: 25, skills: ["TypeScript", "Python"] }
```

### 错误处理

```typescript
import { ChatOpenAI } from "@langchain/openai";

const model = new ChatOpenAI({
  modelName: "gpt-4",
  maxRetries: 3,
  timeout: 30000,
});

try {
  const result = await model.invoke("你好");
  console.log(result.content);
} catch (error) {
  if (error.status === 429) {
    console.error("API 限流，请稍后重试");
  } else if (error.status === 401) {
    console.error("API 密钥无效");
  } else {
    console.error("调用失败:", error.message);
  }
}
```

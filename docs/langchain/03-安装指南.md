# 第3章 安装指南

> 来源: [LangChain 官方文档](https://python.langchain.com/)

---

## 系统要求

| 要求 | 最低版本 | 推荐版本 |
|:---|:---:|:---:|
| Node.js | 18.0 | 20.x 或更高 |
| TypeScript | 5.0 | 5.3 或更高 |
| npm | 8.0 | 10.x 或更高 |

## 安装方式

### 核心包安装

```bash
# npm
npm install @langchain/core @langchain/langgraph

# yarn
yarn add @langchain/core @langchain/langgraph

# pnpm
pnpm add @langchain/core @langchain/langgraph
```

### 按需安装集成包

根据你使用的 LLM 提供商，安装相应的集成包：

```bash
# OpenAI
npm install @langchain/openai

# Anthropic (Claude)
npm install @langchain/anthropic

# Google (Gemini)
npm install @langchain/google-genai

# Azure OpenAI
npm install @langchain/azure-openai

# Ollama (本地模型)
npm install @langchain/ollama
```

### 完整安装（包含所有集成）

```bash
npm install langchain
```

> **注意**：`langchain` 包包含所有集成，体积较大。建议按需安装特定的集成包。

## 环境配置

### 环境变量

创建 `.env` 文件配置 API 密钥：

```bash
# OpenAI
OPENAI_API_KEY=sk-your-api-key
OPENAI_ORG_ID=org-your-org-id  # 可选

# Anthropic
ANTHROPIC_API_KEY=sk-ant-your-api-key

# Google
GOOGLE_API_KEY=your-google-api-key

# Azure OpenAI
AZURE_OPENAI_API_KEY=your-azure-key
AZURE_OPENAI_API_INSTANCE_NAME=your-instance
AZURE_OPENAI_API_DEPLOYMENT_NAME=your-deployment
AZURE_OPENAI_API_VERSION=2024-02-01

# LangSmith (可观测性)
LANGCHAIN_TRACING_V2=true
LANGCHAIN_API_KEY=your-langsmith-key
LANGCHAIN_PROJECT=your-project-name
```

### 加载环境变量

```typescript
// 方式1：使用 dotenv
import "dotenv/config";

// 方式2：手动加载
import * as dotenv from "dotenv";
dotenv.config();

// 方式3：在 Node.js 启动时加载
// node -r dotenv/config your-script.ts
```

### TypeScript 配置

`tsconfig.json` 推荐配置：

```json
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "ES2020",
    "moduleResolution": "node",
    "esModuleInterop": true,
    "strict": true,

    "declaration": true,
    "declarationMap": true,
    "sourceMap": true
  },
  "include": ["src/**/*"],
  "exclude": ["node_modules", "dist"]
}
```

## 验证安装

### 基础验证

```typescript
// test-install.ts
import { ChatOpenAI } from "@langchain/openai";
import "dotenv/config";

async function main() {
  try {
    const model = new ChatOpenAI({
      modelName: "gpt-4",
      temperature: 0,
    });

    const result = await model.invoke("说一句话：安装成功");
    console.log("✅ LangChain 安装成功！");
    console.log("响应:", result.content);
  } catch (error) {
    console.error("❌ 安装验证失败:", error);
  }
}

main();
```

运行验证：

```bash
npx tsx test-install.ts
```

### 检查版本

```typescript
import { __version__ } from "@langchain/core";
console.log("LangChain Core 版本:", __version__);
```

## 常见问题

### 安装失败

**问题**：`npm ERR! peer dep` 错误

**解决方案**：
```bash
# 使用 --legacy-peer-deps
npm install --legacy-peer-deps

# 或更新 npm
npm install -g npm@latest
```

### TypeScript 类型错误

**问题**：找不到模块的类型声明

**解决方案**：
```bash
# 安装类型声明
npm install -D @types/node

# 确保 tsconfig.json 中 skipLibCheck: true
```

### 内存不足

**问题**：`JavaScript heap out of memory`

**解决方案**：
```bash
# 增加 Node.js 内存限制
NODE_OPTIONS="--max-old-space-size=4096" npx tsx your-script.ts
```

### 网络问题

**问题**：连接 API 超时

**解决方案**：
```typescript
const model = new ChatOpenAI({
  modelName: "gpt-4",
  timeout: 30000,  // 30 秒超时
  maxRetries: 3,   // 最多重试 3 次
});
```

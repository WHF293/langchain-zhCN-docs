## 📖 学习路径建议

### 🚀 快速上手路径（1-2 天）

```
01-概述 → 03-安装指南 → 02-快速入门
```

适合想要快速体验 LangChain 的开发者，通过动手实践建立直观认识。

### 📚 系统学习路径（1 周）

```
01-概述 → 04-设计哲学 → 05-组件架构 → 06-模型 → 07-消息 → 02-快速入门
```

适合希望深入理解框架设计的开发者，从理论到实践全面掌握。

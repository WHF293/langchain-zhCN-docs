# 第13章 短期记忆

> 来源: [LangChain 官方文档](https://python.langchain.com/)

---

## 概述

短期记忆 (Short-term Memory) 是指在单次对话会话中维护上下文的能力。它使聊天模型能够记住之前的对话内容，从而进行连贯的多轮对话。在 LangChain 中，有多种方式来管理短期记忆。

## 对话历史

### 基本消息列表

```typescript
import { ChatOpenAI } from "@langchain/openai";
import {
  HumanMessage,
  AIMessage,
  SystemMessage,
} from "@langchain/core/messages";

const model = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0,
});

// 手动管理对话历史
const messages: (HumanMessage | AIMessage | SystemMessage)[] = [
  new SystemMessage("你是一个有帮助的助手。"),
];

async function chat(userInput: string): Promise<string> {
  // 添加用户消息
  messages.push(new HumanMessage(userInput));

  // 调用模型
  const response = await model.invoke(messages);

  // 添加 AI 回复
  messages.push(new AIMessage(response.content as string));

  return response.content as string;
}

// 多轮对话
console.log(await chat("你好，我叫张三"));
console.log(await chat("我是一名程序员"));
console.log(await chat("你还记得我的名字吗？"));
// 模型会记得用户叫张三
```

### 按会话 ID 管理历史

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatMessageHistory } from "@langchain/community/stores/message/in_memory";

// 按会话 ID 存储历史
const sessionHistories = new Map<string, ChatMessageHistory>();

function getSessionHistory(sessionId: string): ChatMessageHistory {
  if (!sessionHistories.has(sessionId)) {
    sessionHistories.set(sessionId, new ChatMessageHistory());
  }
  return sessionHistories.get(sessionId)!;
}

// 用户 A 的对话
const historyA = getSessionHistory("user-a");
await historyA.addMessage({ role: "user", content: "我是用户 A" });
await historyA.addMessage({ role: "assistant", content: "你好，用户 A！" });

// 用户 B 的对话
const historyB = getSessionHistory("user-b");
await historyB.addMessage({ role: "user", content: "我是用户 B" });
await historyB.addMessage({ role: "assistant", content: "你好，用户 B！" });

// 各自的对话历史是独立的
console.log("用户 A 消息数:", (await historyA.getMessages()).length); // 2
console.log("用户 B 消息数:", (await historyB.getMessages()).length); // 2
```

## 消息窗口

### 限制消息数量

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatMessageHistory } from "@langchain/community/stores/message/in_memory";
import { BaseMessage } from "@langchain/core/messages";

class WindowChatHistory {
  private history: ChatMessageHistory;
  private maxMessages: number;

  constructor(maxMessages: number = 10) {
    this.history = new ChatMessageHistory();
    this.maxMessages = maxMessages;
  }

  async addMessage(message: BaseMessage) {
    await this.history.addMessage(message);

    // 保持消息数量在限制内
    const messages = await this.history.getMessages();
    if (messages.length > this.maxMessages) {
      // 移除最旧的消息
      const messagesToRemove = messages.slice(0, messages.length - this.maxMessages);
      // 重新创建历史
      this.history = new ChatMessageHistory();
      for (const msg of messages.slice(messagesToRemove.length)) {
        await this.history.addMessage(msg);
      }
    }
  }

  async getMessages(): Promise<BaseMessage[]> {
    return this.history.getMessages();
  }

  async clear() {
    await this.history.clear();
  }
}

// 使用窗口历史
const windowHistory = new WindowChatHistory(5); // 保留最近 5 条消息

const model = new ChatOpenAI({ modelName: "gpt-4" });

async function chatWithWindow(userInput: string): Promise<string> {
  await windowHistory.addMessage({ role: "user", content: userInput });

  const messages = await windowHistory.getMessages();
  const response = await model.invoke(messages);

  await windowHistory.addMessage({ role: "assistant", content: response.content as string });

  return response.content as string;
}

// 即使对话很长，也只保留最近的消息
for (let i = 1; i <= 10; i++) {
  console.log(`轮次 ${i}:`, await chatWithWindow(`消息 ${i}`));
}
```

### Token 窗口限制

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatMessageHistory } from "@langchain/community/stores/message/in_memory";
import { BaseMessage } from "@langchain/core/messages";
import { encoding_for_model } from "tiktoken";

class TokenWindowHistory {
  private messages: BaseMessage[] = [];
  private maxTokens: number;
  private encoder: any;

  constructor(maxTokens: number = 4000) {
    this.maxTokens = maxTokens;
    this.encoder = encoding_for_model("gpt-4");
  }

  addMessage(message: BaseMessage) {
    this.messages.push(message);
    this.trimToTokenLimit();
  }

  private trimToTokenLimit() {
    let totalTokens = 0;
    const messagesToKeep: BaseMessage[] = [];

    // 从最新的消息开始计算
    for (let i = this.messages.length - 1; i >= 0; i--) {
      const message = this.messages[i];
      const content = typeof message.content === "string" ? message.content : "";
      const tokens = this.encoder.encode(content).length;

      if (totalTokens + tokens > this.maxTokens) {
        break;
      }

      totalTokens += tokens;
      messagesToKeep.unshift(message);
    }

    this.messages = messagesToKeep;
  }

  getMessages(): BaseMessage[] {
    return [...this.messages];
  }

  getTokenCount(): number {
    let totalTokens = 0;
    for (const message of this.messages) {
      const content = typeof message.content === "string" ? message.content : "";
      totalTokens += this.encoder.encode(content).length;
    }
    return totalTokens;
  }

  clear() {
    this.messages = [];
  }
}

// 使用 Token 窗口
const tokenHistory = new TokenWindowHistory(2000); // 最多 2000 tokens
```

## 消息摘要

### 自动摘要历史

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatMessageHistory } from "@langchain/community/stores/message/in_memory";
import { BaseMessage, HumanMessage, AIMessage, SystemMessage } from "@langchain/core/messages";

class SummarizingHistory {
  private history: ChatMessageHistory;
  private summary: string = "";
  private model: ChatOpenAI;
  private maxMessagesBeforeSummary: number;

  constructor(maxMessagesBeforeSummary: number = 10) {
    this.history = new ChatMessageHistory();
    this.model = new ChatOpenAI({ modelName: "gpt-4", temperature: 0 });
    this.maxMessagesBeforeSummary = maxMessagesBeforeSummary;
  }

  async addMessage(message: BaseMessage) {
    await this.history.addMessage(message);

    const messages = await this.history.getMessages();
    if (messages.length >= this.maxMessagesBeforeSummary) {
      await this.summarize();
    }
  }

  private async summarize() {
    const messages = await this.history.getMessages();

    // 创建摘要提示
    const summaryPrompt = `请将以下对话总结为简短的摘要，保留关键信息：

${messages.map((m) => `${m._getType()}: ${m.content}`).join("\n")}

摘要:`;

    const summaryResponse = await this.model.invoke(summaryPrompt);
    this.summary = summaryResponse.content as string;

    // 清空历史，只保留摘要
    await this.history.clear();
    await this.history.addMessage(
      new SystemMessage(`之前的对话摘要: ${this.summary}`)
    );
  }

  async getMessages(): Promise<BaseMessage[]> {
    return this.history.getMessages();
  }

  getSummary(): string {
    return this.summary;
  }

  async clear() {
    await this.history.clear();
    this.summary = "";
  }
}

// 使用摘要历史
const summarizingHistory = new SummarizingHistory(6);

const model = new ChatOpenAI({ modelName: "gpt-4" });

async function chatWithSummary(userInput: string): Promise<string> {
  await summarizingHistory.addMessage(new HumanMessage(userInput));

  const messages = await summarizingHistory.getMessages();
  const response = await model.invoke(messages);

  await summarizingHistory.addMessage(new AIMessage(response.content as string));

  return response.content as string;
}

// 长对话会自动摘要
console.log(await chatWithSummary("我正在学习 LangChain"));
console.log(await chatWithSummary("它是一个 AI 应用框架"));
console.log(await chatWithSummary("主要功能包括链、代理、记忆等"));
console.log(await chatWithSummary("你还记得我们聊了什么吗？"));
```

### 滑动窗口与摘要结合

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { BaseMessage, HumanMessage, AIMessage, SystemMessage } from "@langchain/core/messages";

class HybridMemory {
  private recentMessages: BaseMessage[] = [];
  private summary: string = "";
  private model: ChatOpenAI;
  private windowSize: number;

  constructor(windowSize: number = 6) {
    this.model = new ChatOpenAI({ modelName: "gpt-4", temperature: 0 });
    this.windowSize = windowSize;
  }

  async addMessage(message: BaseMessage) {
    this.recentMessages.push(message);

    if (this.recentMessages.length > this.windowSize) {
      // 将旧消息移到摘要
      const oldMessages = this.recentMessages.slice(0, -this.windowSize);
      this.recentMessages = this.recentMessages.slice(-this.windowSize);

      // 更新摘要
      await this.updateSummary(oldMessages);
    }
  }

  private async updateSummary(messages: BaseMessage[]) {
    const messagesText = messages
      .map((m) => `${m._getType()}: ${m.content}`)
      .join("\n");

    const prompt = this.summary
      ? `之前的摘要: ${this.summary}\n\n新的对话:\n${messagesText}\n\n更新后的摘要:`
      : `请总结以下对话:\n${messagesText}\n\n摘要:`;

    const response = await this.model.invoke(prompt);
    this.summary = response.content as string;
  }

  getContext(): BaseMessage[] {
    const context: BaseMessage[] = [];

    if (this.summary) {
      context.push(new SystemMessage(`对话历史摘要: ${this.summary}`));
    }

    context.push(...this.recentMessages);
    return context;
  }

  getSummary(): string {
    return this.summary;
  }

  getRecentMessages(): BaseMessage[] {
    return [...this.recentMessages];
  }

  clear() {
    this.recentMessages = [];
    this.summary = "";
  }
}

// 使用混合记忆
const memory = new HybridMemory(4);
const model = new ChatOpenAI({ modelName: "gpt-4" });

async function chatHybrid(userInput: string): Promise<string> {
  await memory.addMessage(new HumanMessage(userInput));

  const context = memory.getContext();
  const response = await model.invoke(context);

  await memory.addMessage(new AIMessage(response.content as string));

  return response.content as string;
}
```

## 消息压缩

### 基于重要性的压缩

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { BaseMessage, HumanMessage, AIMessage } from "@langchain/core/messages";

interface MessageWithScore {
  message: BaseMessage;
  score: number;
}

class ImportanceCompressor {
  private messages: MessageWithScore[] = [];
  private model: ChatOpenAI;
  private maxMessages: number;

  constructor(maxMessages: number = 10) {
    this.model = new ChatOpenAI({ modelName: "gpt-4", temperature: 0 });
    this.maxMessages = maxMessages;
  }

  async addMessage(message: BaseMessage) {
    // 计算消息重要性分数
    const score = await this.calculateImportance(message);

    this.messages.push({ message, score });

    if (this.messages.length > this.maxMessages) {
      await this.compress();
    }
  }

  private async calculateImportance(message: BaseMessage): Promise<number> {
    // 简单的重要性计算（实际应用中可以更复杂）
    const content = typeof message.content === "string" ? message.content : "";

    // 关键词权重
    const importantKeywords = ["重要", "关键", "记住", "注意", "必须"];
    let score = 5; // 基础分数

    for (const keyword of importantKeywords) {
      if (content.includes(keyword)) {
        score += 2;
      }
    }

    // 问题通常更重要
    if (content.includes("?") || content.includes("？")) {
      score += 1;
    }

    // 较新的消息更重要
    score += this.messages.length * 0.1;

    return Math.min(score, 10);
  }

  private async compress() {
    // 按重要性排序
    this.messages.sort((a, b) => b.score - a.score);

    // 保留最重要的消息
    const importantMessages = this.messages.slice(0, this.maxMessages);

    // 对不重要的消息进行摘要
    const unimportantMessages = this.messages.slice(this.maxMessages);
    if (unimportantMessages.length > 0) {
      const summary = await this.summarizeMessages(
        unimportantMessages.map((m) => m.message)
      );

      // 将摘要作为一条高分消息
      importantMessages.push({
        message: new HumanMessage(`[历史摘要] ${summary}`),
        score: 3,
      });
    }

    this.messages = importantMessages;
  }

  private async summarizeMessages(messages: BaseMessage[]): Promise<string> {
    const text = messages.map((m) => `${m._getType()}: ${m.content}`).join("\n");
    const response = await this.model.invoke(
      `请简短总结以下对话内容:\n${text}\n\n摘要:`
    );
    return response.content as string;
  }

  getMessages(): BaseMessage[] {
    return this.messages
      .sort((a, b) => a.score - b.score)
      .map((m) => m.message);
  }
}

// 使用重要性压缩
const compressor = new ImportanceCompressor(8);
```

### 基于时间衰减的压缩

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { BaseMessage } from "@langchain/core/messages";

interface TimedMessage {
  message: BaseMessage;
  timestamp: number;
  initialRelevance: number;
}

class TimeDecayMemory {
  private messages: TimedMessage[] = [];
  private decayRate: number;
  private maxMessages: number;

  constructor(maxMessages: number = 20, decayRate: number = 0.1) {
    this.maxMessages = maxMessages;
    this.decayRate = decayRate;
  }

  addMessage(message: BaseMessage) {
    this.messages.push({
      message,
      timestamp: Date.now(),
      initialRelevance: 10,
    });

    if (this.messages.length > this.maxMessages) {
      this.pruneOldMessages();
    }
  }

  private pruneOldMessages() {
    const now = Date.now();

    // 计算每条消息的当前相关性
    const messagesWithRelevance = this.messages.map((msg) => {
      const ageInMinutes = (now - msg.timestamp) / (1000 * 60);
      const currentRelevance =
        msg.initialRelevance * Math.exp(-this.decayRate * ageInMinutes);

      return { ...msg, currentRelevance };
    });

    // 按相关性排序，保留最重要的
    messagesWithRelevance.sort((a, b) => b.currentRelevance - a.currentRelevance);
    this.messages = messagesWithRelevance.slice(0, this.maxMessages);
  }

  getMessages(): BaseMessage[] {
    return this.messages.map((m) => m.message);
  }

  getRelevanceScores(): number[] {
    const now = Date.now();
    return this.messages.map((msg) => {
      const ageInMinutes = (now - msg.timestamp) / (1000 * 60);
      return msg.initialRelevance * Math.exp(-this.decayRate * ageInMinutes);
    });
  }
}

// 使用时间衰减记忆
const timeDecayMemory = new TimeDecayMemory(15, 0.05);
```

## 最佳实践

1. **选择合适的窗口大小**：根据模型的上下文窗口和应用场景选择合适的记忆大小
2. **保留系统提示**：确保系统提示始终在消息历史中
3. **处理长对话**：对于长对话，使用摘要或压缩策略
4. **用户隔离**：为不同用户维护独立的对话历史
5. **持久化考虑**：如果需要跨会话记忆，考虑使用长期记忆

> **提示**：对于大多数聊天应用，保留最近 10-20 条消息通常能提供足够的上下文。

> **注意**：过长的对话历史会增加 token 消耗和响应延迟，需要在上下文丰富度和性能之间找到平衡。

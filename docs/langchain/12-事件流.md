> 来源: [LangChain 官方文档](https://python.langchain.com/)

## 概述

事件流 (Event Stream) 是 LangChain 中用于监控和调试链执行过程的机制。通过事件流，开发者可以实时了解模型调用、工具执行、链处理等各个环节的状态和输出。这对于构建可观察的应用程序至关重要。

## 事件类型

### 核心事件类型

| 事件类型 | 说明 | 数据内容 |
|---------|------|---------|
| `on_llm_start` | LLM 开始处理 | 输入消息、配置信息 |
| `on_llm_new_token` | 新 token 生成 | token 内容 |
| `on_llm_end` | LLM 处理完成 | 完整响应、使用统计 |
| `on_llm_error` | LLM 处理出错 | 错误信息 |
| `on_chain_start` | 链开始执行 | 输入数据 |
| `on_chain_end` | 链执行完成 | 输出结果 |
| `on_chain_error` | 链执行出错 | 错误信息 |
| `on_tool_start` | 工具开始执行 | 工具名称、输入参数 |
| `on_tool_end` | 工具执行完成 | 工具输出 |
| `on_tool_error` | 工具执行出错 | 错误信息 |
| `on_retriever_start` | 检索器开始 | 查询内容 |
| `on_retriever_end` | 检索器完成 | 检索结果 |

### 事件数据结构

```typescript
interface BaseEvent {
  event: string;
  name: string;
  run_id: string;
  parent_run_id?: string;
  tags: string[];
  metadata: Record<string, any>;
  data: Record<string, any>;
  start_time: number;
  end_time?: number;
}

interface LLMStartEvent extends BaseEvent {
  event: "on_llm_start";
  data: {
    input: {
      messages: any[];
    };
  };
}

interface LLMNewTokenEvent extends BaseEvent {
  event: "on_llm_new_token";
  data: {
    chunk: {
      content: string;
      additional_kwargs?: Record<string, any>;
    };
  };
}

interface LLMEndEvent extends BaseEvent {
  event: "on_llm_end";
  data: {
    output: {
      content: string;
      usage?: {
        prompt_tokens: number;
        completion_tokens: number;
        total_tokens: number;
      };
    };
  };
}
```

## 事件订阅

### 使用 streamEvents

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatPromptTemplate } from "@langchain/core/prompts";

const model = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0,
});

const prompt = ChatPromptTemplate.fromTemplate("告诉我关于{topic}的知识");
const chain = prompt.pipe(model);

// 订阅事件流
const eventStream = await chain.streamEvents(
  { topic: "人工智能" },
  { version: "v2" }
);

// 处理事件
for await (const event of eventStream) {
  switch (event.event) {
    case "on_chain_start":
      console.log(`链开始: ${event.name}`);
      break;

    case "on_chat_model_start":
      console.log(`模型开始: ${event.name}`);
      break;

    case "on_chat_model_stream":
      process.stdout.write(event.data?.chunk?.content as string || "");
      break;

    case "on_chat_model_end":
      console.log("\n模型完成");
      console.log("Token 使用:", event.data?.output?.usage);
      break;

    case "on_chain_end":
      console.log(`链完成: ${event.name}`);
      break;
  }
}
```

### 使用回调订阅

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { BaseCallbackHandler } from "@langchain/core/callbacks/base";

// 自定义回调处理器
class LoggingCallbackHandler extends BaseCallbackHandler {
  name = "LoggingCallbackHandler";

  async handleLLMStart(llm: any, prompts: string[]) {
    console.log("[LLM 开始]", prompts);
  }

  async handleLLMNewToken(token: string) {
    process.stdout.write(token);
  }

  async handleLLMEnd(output: any) {
    console.log("\n[LLM 结束]", output.llmOutput);
  }

  async handleLLMError(error: Error) {
    console.error("[LLM 错误]", error.message);
  }

  async handleChainStart(chain: any, inputs: any) {
    console.log("[链开始]", chain.id, inputs);
  }

  async handleChainEnd(outputs: any) {
    console.log("[链结束]", outputs);
  }

  async handleChainError(error: Error) {
    console.error("[链错误]", error.message);
  }

  async handleToolStart(tool: any, input: string) {
    console.log("[工具开始]", tool.name, input);
  }

  async handleToolEnd(output: string) {
    console.log("[工具结束]", output);
  }

  async handleToolError(error: Error) {
    console.error("[工具错误]", error.message);
  }
}

// 使用回调
const model = new ChatOpenAI({
  modelName: "gpt-4",
  temperature: 0,
  callbacks: [new LoggingCallbackHandler()],
});

const result = await model.invoke("你好");
```

## 事件过滤

### 按事件类型过滤

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatPromptTemplate } from "@langchain/core/prompts";

  { version: "v2" }
);

// 只处理流式 token 事件
const tokenEvents: string[] = [];

for await (const event of eventStream) {
  if (event.event === "on_chat_model_stream") {
    const content = event.data?.chunk?.content;
    if (content) {
      tokenEvents.push(content as string);
    }
  }
}

console.log("所有 token:", tokenEvents.join(""));
```

## 事件处理

### 事件聚合器

```typescript
import { ChatOpenAI } from "@langchain/openai";

interface AggregatedStats {
  totalTokens: number;
  promptTokens: number;
  completionTokens: number;
  totalDuration: number;
  llmCalls: number;
  toolCalls: number;
  errors: string[];
}

class EventAggregator {
  private stats: AggregatedStats = {
    totalTokens: 0,
    promptTokens: 0,
    completionTokens: 0,
    totalDuration: 0,
    llmCalls: 0,
    toolCalls: 0,
    errors: [],
  };

  private startTime = 0;

  processEvent(event: any) {
    switch (event.event) {
      case "on_chat_model_start":
        this.stats.llmCalls++;
        this.startTime = Date.now();
        break;

      case "on_chat_model_end":
        const usage = event.data?.output?.usage;
        if (usage) {
          this.stats.promptTokens += usage.prompt_tokens || 0;
          this.stats.completionTokens += usage.completion_tokens || 0;
          this.stats.totalTokens += usage.total_tokens || 0;
        }
        this.stats.totalDuration += Date.now() - this.startTime;
        break;

      case "on_tool_start":
        this.stats.toolCalls++;
        break;

      case "on_llm_error":
      case "on_chain_error":
      case "on_tool_error":
        this.stats.errors.push(event.data?.error?.message || "未知错误");
        break;
    }
  }

  getStats(): AggregatedStats {
    return { ...this.stats };
  }

  reset() {
    this.stats = {
      totalTokens: 0,
      promptTokens: 0,
      completionTokens: 0,
      totalDuration: 0,
      llmCalls: 0,
      toolCalls: 0,
      errors: [],
    };
  }
}

// 使用示例
const model = new ChatOpenAI({ modelName: "gpt-4" });
const aggregator = new EventAggregator();

const eventStream = await model.streamEvents(
  "写一篇关于 AI 的短文",
  { version: "v2" }
);

for await (const event of eventStream) {
  aggregator.processEvent(event);
}

console.log("聚合统计:", aggregator.getStats());
```

### 实时事件处理器

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { EventEmitter } from "events";

class RealTimeEventProcessor extends EventEmitter {
  private tokenBuffer: string[] = [];
  private isProcessing = false;

  constructor() {
    super();
    this.setupEventHandlers();
  }

  private setupEventHandlers() {
    this.on("token", (token: string) => {
      this.tokenBuffer.push(token);
      this.emit("token:buffered", token);
    });

    this.on("complete", (fullText: string) => {
      console.log("生成完成:", fullText.length, "字符");
      this.emit("stats", {
        totalTokens: this.tokenBuffer.length,
        text: fullText,
      });
    });

    this.on("error", (error: Error) => {
      console.error("处理错误:", error.message);
    });
  }

  async processStream(stream: AsyncGenerator<any>) {
    this.isProcessing = true;
    this.tokenBuffer = [];

    try {
      for await (const event of stream) {
        if (event.event === "on_chat_model_stream") {
          const content = event.data?.chunk?.content;
          if (content) {
            this.emit("token", content);
          }
        } else if (event.event === "on_chat_model_end") {
          this.emit("complete", this.tokenBuffer.join(""));
        } else if (event.event?.includes("error")) {
          this.emit("error", new Error(event.data?.error?.message || "未知错误"));
        }
      }
    } finally {
      this.isProcessing = false;
    }
  }

  getBuffer(): string[] {
    return [...this.tokenBuffer];
  }

  getFullText(): string {
    return this.tokenBuffer.join("");
  }
}

// 使用示例
const model = new ChatOpenAI({ modelName: "gpt-4" });
const processor = new RealTimeEventProcessor();

// 监听事件
processor.on("token:buffered", (token) => {
  process.stdout.write(token);
});

processor.on("stats", (stats) => {
  console.log("\n统计:", stats);
});

// 处理流
const stream = await model.streamEvents("你好", { version: "v2" });
await processor.processStream(stream);
```

### 事件持久化

```typescript
import { ChatOpenAI } from "@langchain/openai";
import fs from "fs/promises";

interface PersistedEvent {
  timestamp: number;
  event: any;
}

class EventPersister {
  private events: PersistedEvent[] = [];
  private filePath: string;

  constructor(filePath: string) {
    this.filePath = filePath;
  }

  async recordEvent(event: any) {
    this.events.push({
      timestamp: Date.now(),
      event,
    });

    // 定期保存
    if (this.events.length % 100 === 0) {
      await this.save();
    }
  }

  async save() {
    await fs.writeFile(
      this.filePath,
      JSON.stringify(this.events, null, 2)
    );
  }

  async load(): Promise<PersistedEvent[]> {
    try {
      const data = await fs.readFile(this.filePath, "utf-8");
      return JSON.parse(data);
    } catch {
      return [];
    }
  }

  getEvents(): PersistedEvent[] {
    return [...this.events];
  }
}

// 使用示例
const model = new ChatOpenAI({ modelName: "gpt-4" });
const persister = new EventPersister("./events.json");

const eventStream = await model.streamEvents("你好", { version: "v2" });

for await (const event of eventStream) {
  await persister.recordEvent(event);
}

await persister.save();
console.log("事件已保存");

// 稍后加载
const loadedEvents = await persister.load();
console.log("加载的事件数:", loadedEvents.length);
```

## 最佳实践

1. **选择合适的事件**：根据需求选择监听哪些事件，避免处理不必要的事件
2. **异步处理**：事件处理应该是非阻塞的，避免影响主流程
3. **错误处理**：在事件处理器中实现适当的错误处理
4. **资源清理**：确保事件监听器在不再需要时被正确移除
5. **性能考虑**：高频事件（如 token）的处理应尽量轻量

> **提示**：使用 `streamEvents` 方法可以获取最详细的执行信息，适合调试和监控场景。

> **注意**：在生产环境中，注意事件处理的性能影响，避免在事件处理器中执行耗时操作。
